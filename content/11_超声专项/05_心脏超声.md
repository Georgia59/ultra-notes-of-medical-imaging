<a id="us-rheumatic-heart"></a>

## 风湿性心脏病超声

### 二尖瓣狭窄（最常见）

| 超声特征 | 表现 | 机制 |
|---|---|---|
| 瓣叶 | 增厚、回声增强，交界粘连，开放受限 | 风湿性炎症→纤维化、钙化 |
| **M型超声** | 二尖瓣前叶呈**"城墙样"改变**（EF斜率减慢），后叶与前叶**同向运动** | 瓣膜粘连→舒张期左房排空减慢→前叶持续高位 |
| 二维超声 | 舒张期瓣口呈**"鱼嘴状"**（短轴切面），瓣口面积缩小（正常4～6cm²） | 交界粘连致瓣口狭窄 |
| 多普勒 | 二尖瓣口舒张期**高速血流**频谱，跨瓣压差增大 | 狭窄致血流加速 |
| 继发改变 | 左心房增大、右心室增大、肺动脉高压 | 左房压力升高→肺静脉淤血→肺动脉高压→右室增大 |

### 二尖瓣关闭不全

| 特征 | 表现 |
|---|---|
| 二维 | 收缩期瓣叶对合不良/有裂隙 |
| CDFI | 收缩期**左心房内反流信号**（蓝色为主） |
| 左心增大 | 左心房、左心室容量负荷增加→增大 |

### 主动脉瓣狭窄

- 瓣叶增厚、钙化，开放受限
- 左心室向心性肥厚（压力负荷增加）
- 多普勒：主动脉瓣高速血流，跨瓣压差增大

### 主动脉瓣关闭不全

- CDFI：舒张期**左心室流出道内反流信号**
- 左心室增大（容量负荷增加）

<a id="us-coronary"></a>

## 冠心病及其并发症超声

### 心肌梗死超声表现

| 特征 | 表现 |
|---|---|
| **节段性室壁运动异常**（核心） | 梗死区域心肌运动减弱、无运动或矛盾运动 |
| 室壁厚度 | 梗死区变薄 |
| 回声 | 急性期可正常；陈旧期瘢痕→回声增强 |
| 心功能 | EF（射血分数）降低 |

### 并发症超声

| 并发症 | 超声表现 |
|---|---|
| **室壁瘤** | 梗死区室壁局限性向外膨出，收缩期更明显，呈**矛盾运动**（收缩期向外运动） |
| 附壁血栓 | 室壁瘤或梗死区内低回声团块 |
| 室间隔穿孔 | CDFI显示室水平左向右分流 |
| 乳头肌断裂 | 二尖瓣连枷样运动+CDFI重度反流 |

<a id="us-congenital"></a>

## 先天性心脏病超声

### 房间隔缺损（ASD）

| 特征 | 表现 |
|---|---|
| 二维 | 房间隔**回声中断** |
| CDFI | 舒张期**左向右分流**（红色血流信号穿过缺损） |
| 右心改变 | 右心房、右心室增大，肺动脉增宽（右心容量负荷增加） |

### 法洛四联症（TOF）

| 特征 | 表现 |
|---|---|
| 四种畸形 | **肺动脉狭窄**（最决定性）+室间隔缺损+主动脉骑跨+右心室肥厚 |
| CDFI | 室水平双向或**右向左分流**（蓝色血流信号） |
| 主动脉 | 增宽，骑跨于室间隔之上 |

### 考试答题版

风湿性心脏病二尖瓣狭窄：M型"城墙样"改变+后叶同向运动+瓣口"鱼嘴状"+多普勒高速血流。心肌梗死：节段性室壁运动异常（运动减弱/无运动/矛盾运动），EF降低。并发症：室壁瘤（矛盾运动）、附壁血栓。ASD：房间隔回声中断+左向右分流+右心增大。法洛四联症：肺动脉狭窄+室缺+主动脉骑跨+右室肥厚，右向左分流。
