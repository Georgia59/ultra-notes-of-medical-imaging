## 甲状腺疾病超声

### 1. 检查思路

甲状腺超声不是只看“有没有结节”，而是先判断病变是弥漫性还是局灶性。弥漫性病变重点看腺体大小、实质回声和血流；结节性病变重点看组成、回声、形态、边缘和强回声灶。

| 观察项目 | 影像意义 |
| --- | --- |
| 腺体大小 | 弥漫增大提示 Graves 病、桥本早期或其他甲状腺炎；萎缩可见于桥本后期。 |
| 实质回声 | 弥漫减低多提示炎症、淋巴细胞浸润或血流增多；不均匀提示慢性炎症和纤维化。 |
| CDFI 血流 | 血流明显增多提示功能亢进或炎症活动，但血流丰富不能单独判断恶性。 |
| 结节形态 | “高大于宽”、边缘不规则、微钙化和极低回声提示恶性风险增高。 |
| 颈部淋巴结 | 囊变、微钙化、皮髓质分界消失或周边血流提示转移风险。 |

### 2. Graves病（弥漫性毒性甲状腺肿）

Graves 病本质是促甲状腺激素受体抗体刺激甲状腺滤泡细胞，使腺体弥漫增生、血流增加和激素合成增强。因此超声核心是“弥漫增大 + 低回声 + 血流极丰富”。

| 特征 | 超声表现 |
|---|---|
| 大小 | 甲状腺**弥漫性对称性增大** |
| 回声 | 实质回声**弥漫性减低**，不均匀 |
| CDFI | **"火海征"**——甲状腺内弥漫丰富血流信号，呈搏动性闪烁 |

**机制：** 滤泡上皮增生和间质血管扩张使腺体增大、血流增加；淋巴细胞浸润和组织结构改变使回声减低、不均匀。

### 3. 桥本甲状腺炎

桥本甲状腺炎本质是自身免疫性慢性淋巴细胞性甲状腺炎。早期炎症和淋巴细胞浸润可使腺体增大，后期滤泡破坏和纤维化可使腺体萎缩。

| 特征 | 超声表现 |
|---|---|
| 大小 | 甲状腺弥漫性增大（后期可萎缩） |
| 回声 | 实质回声**弥漫性不均匀减低**，呈网格状/结节状高回声带分隔 |
| CDFI | 血流信号增多但不如Graves病丰富 |

**机制：** 淋巴细胞浸润和滤泡破坏导致低回声；纤维化和残存小叶结构形成网格状或条索状高回声分隔。

### 4. 甲状腺癌（乳头状癌最常见）

甲状腺癌超声判断的核心不是“血流多不多”，而是结节生长方式和内部结构是否提示浸润性生长。乳头状癌常有微钙化、低回声、边缘不规则和纵横比大于 1。

| 特征 | 超声表现 |
|---|---|
| 结节形态 | **低回声**或极低回声，边缘不规则/毛刺状，纵横比＞1（"**高大于宽**"） |
| 钙化 | **微钙化**（≤1mm砂粒样钙化——特征性） |
| CDFI | 血流可增多、减少或不明显，分布可紊乱，但血流丰富并非可靠的单独恶性标准 |
| 淋巴结 | 颈部淋巴结转移（皮髓质分界不清、微钙化） |

| 恶性征象 | 机制 |
| --- | --- |
| 极低回声 | 肿瘤细胞密集、正常滤泡胶质少，声学界面改变。 |
| 高大于宽 | 肿瘤倾向穿越正常组织平面纵向浸润，而不是沿组织间隙横向生长。 |
| 边缘不规则/毛刺 | 肿瘤向周围甲状腺实质浸润，界面不光整。 |
| 微钙化 | 乳头状癌内砂粒体钙化，形成细小强回声点，常无明显声影。 |
| 转移淋巴结微钙化/囊变 | 转移灶保留乳头状癌钙化或发生坏死囊变。 |

### 5. ACR TI-RADS分级与处理

> TI-RADS存在不同版本。以下明确采用 **ACR TI-RADS**，根据组成、回声、形态、边缘和强回声灶积分；是否穿刺必须同时考虑分级和最大径，不能仅凭“4级或5级”直接决定FNA。

| 分级 | 积分/风险描述 | ACR处理阈值 |
|---|---|---|
| TR1 | 0分，良性 | 不需FNA或随访 |
| TR2 | 2分，不可疑 | 不需FNA或随访 |
| TR3 | 3分，轻度可疑 | ≥2.5cm建议FNA；≥1.5cm随访 |
| TR4 | 4～6分，中度可疑 | ≥1.5cm建议FNA；≥1.0cm随访 |
| TR5 | ≥7分，高度可疑 | ≥1.0cm建议FNA；≥0.5cm随访 |

### 6. Graves病 vs 桥本 vs 甲状腺癌

| 项目 | Graves病 | 桥本甲状腺炎 | 甲状腺癌 |
|---|---|---|---|
| 累及范围 | 弥漫 | 弥漫 | 局灶结节 |
| 回声 | 弥漫减低 | 弥漫不均减低，网格状 | 低/极低回声结节 |
| 血流 | **"火海征"** | 增多（不如Graves） | 结节内丰富紊乱 |
| 特征 | 对称增大 | TPOAb/TgAb升高 | 微钙化+高大于宽 |

### 【考试答题版】

甲状腺超声先判断弥漫性病变还是结节性病变。Graves 病表现为甲状腺弥漫对称性增大、实质回声弥漫减低、CDFI 血流极丰富，呈“火海征”，机制是滤泡增生和血流显著增加。

桥本甲状腺炎表现为甲状腺弥漫不均匀减低回声，可见网格状或条索状高回声分隔，血流可增多但通常不如 Graves 病丰富，机制为慢性淋巴细胞浸润、滤泡破坏和纤维化。

甲状腺癌尤其乳头状癌常表现为低或极低回声结节、边缘不规则、纵横比大于 1、微钙化和可疑颈部淋巴结。ACR TI-RADS 需按积分和结节最大径共同决定 FNA 或随访，不能只看分级。

### 【一句话记忆】

**Graves 看火海，桥本看网格，乳头癌看低回声、微钙化和高大于宽；TI-RADS 要分级加大小一起看。**

### 【易错点】

- 血流丰富不是甲状腺癌的可靠单独恶性标准。
- 微钙化比粗大钙化更提示乳头状癌，但需结合形态、边缘和回声。
- TI-RADS 4 或 5 级不等于一定穿刺，仍要看结节大小和临床背景。
- Graves 和桥本都可弥漫低回声、血流增多，需结合甲功和抗体。

## 周围血管超声

### 1. 周围血管超声总原则

血管超声同时看形态和血流。二维灰阶显示血管壁、内膜-中膜厚度、斑块和血栓；彩色多普勒显示血流充盈和方向；频谱多普勒通过流速、频谱增宽和波形变化判断狭窄程度。

### 2. 动脉粥样硬化

动脉粥样硬化的本质是脂质沉积、炎症和纤维帽形成，使内膜增厚、斑块突入管腔，并造成局部狭窄或闭塞。狭窄处管腔变窄，血流速度增快，狭窄后可出现湍流和频谱增宽。

| 表现 | 超声特征 |
|---|---|
| 内膜-中膜厚度（IMT） | 增厚（＞1.0mm） |
| 斑块 | 局部隆起突入管腔，可为软斑块（低回声）、硬斑块（强回声伴声影）、混合斑块 |
| 狭窄 | CDFI显示局部血流束变细、流速增高、频谱增宽 |
| 闭塞 | CDFI管腔内无血流信号 |

| 征象 | 机制 |
| --- | --- |
| 软斑块低回声 | 脂质、坏死核心和出血成分较多，反射界面少。 |
| 硬斑块强回声伴声影 | 钙化明显，声波反射和衰减强。 |
| 狭窄处流速增高 | 同一血流量通过更窄管腔，速度升高。 |
| 频谱增宽 | 狭窄处和狭窄后湍流增多，速度分布不均。 |

### 3. 下肢深静脉血栓（DVT）

DVT 是深静脉内血栓形成，常与血流淤滞、血管内皮损伤和高凝状态有关。超声最重要的诊断动作是探头加压：正常静脉可被压闭，血栓充填后管腔不能压闭。

| 表现 | 超声特征 |
|---|---|
| 管腔 | 增宽，腔内**低/等回声**充填（急性：低回声/无回声；慢性：中高回声） |
| 压缩性 | 探头加压管腔**不能压闭**（最重要诊断征象） |
| CDFI | 血栓段**血流信号消失**或充盈缺损 |
| 远端挤压 | 正常时血流加速→DVT时无反应 |

| 征象 | 机制 |
| --- | --- |
| 急性血栓低回声 | 新鲜血栓含水多、纤维组织少，回声低。 |
| 慢性血栓回声增高 | 血栓机化、纤维化和钙化使内部反射增加。 |
| 管腔不能压闭 | 血栓占据管腔，使静脉失去正常可压缩性。 |
| 血流充盈缺损 | 血栓阻挡血流，彩色多普勒不能完全充盈管腔。 |
| 远端挤压无反应 | 挤压远端后血流不能顺利通过血栓段。 |

### 4. 鉴别诊断

| 鉴别对象 | 相似点 | 区分点 |
| --- | --- | --- |
| 软斑块 vs 血栓 | 均可低回声并造成管腔狭窄或充盈缺损。 | 软斑块附着于动脉壁且常伴 IMT 增厚；静脉血栓位于静脉腔内，关键是不可压闭。 |
| 慢性 DVT vs 急性 DVT | 均可管腔内见血栓回声。 | 急性血栓低回声、管腔扩张；慢性血栓回声较高、管壁增厚、管腔可变窄并有侧支。 |
| Baker 囊肿破裂 vs DVT | 小腿肿痛相似。 | Baker 囊肿位于腘窝或小腿肌间隙，静脉仍可压闭且腔内无血栓。 |

### 【考试答题版】

Graves病：甲状腺弥漫增大+弥漫回声减低+"火海征"。桥本：弥漫不均质减低+网格状+TPOAb/TgAb升高。甲状腺癌：低回声结节+微钙化+高大于宽；ACR TI-RADS需按积分和结节大小共同决定FNA或随访。动脉粥样硬化：IMT增厚+斑块+狭窄处流速增高。DVT：管腔增宽+探头不能压闭+CDFI无血流信号。

周围血管超声要同时评价形态和血流。动脉粥样硬化表现为 IMT 增厚、斑块突入管腔，狭窄处彩色血流束变细、流速增高、频谱增宽；闭塞时管腔内无血流信号。下肢 DVT 表现为静脉管腔内低或等回声血栓，探头加压不能压闭，CDFI 血流信号消失或充盈缺损，远端挤压无血流加速。

### 【一句话记忆】

**动脉狭窄看斑块、流速高和频谱宽；DVT 最关键看静脉不能压闭。**

### 【易错点】

- DVT 的核心征象是不可压缩，不是单纯“看见低回声”。
- 急性血栓可很低回声，容易漏诊，要结合压缩试验和彩色血流。
- CDFI 无血流不一定等于闭塞，低流速和仪器参数也会影响显示。
- 动脉狭窄程度不能只凭斑块大小判断，需结合流速和频谱改变。
