## 甲状腺疾病超声

### Graves病（弥漫性毒性甲状腺肿）

| 特征 | 超声表现 |
|---|---|
| 大小 | 甲状腺**弥漫性对称性增大** |
| 回声 | 实质回声**弥漫性减低**，不均匀 |
| CDFI | **"火海征"**——甲状腺内弥漫丰富血流信号，呈搏动性闪烁 |

### 桥本甲状腺炎

| 特征 | 超声表现 |
|---|---|
| 大小 | 甲状腺弥漫性增大（后期可萎缩） |
| 回声 | 实质回声**弥漫性不均匀减低**，呈网格状/结节状高回声带分隔 |
| CDFI | 血流信号增多但不如Graves病丰富 |

### 甲状腺癌（乳头状癌最常见）

| 特征 | 超声表现 |
|---|---|
| 结节形态 | **低回声**或极低回声，边缘不规则/毛刺状，纵横比＞1（"**高大于宽**"） |
| 钙化 | **微钙化**（≤1mm砂粒样钙化——特征性） |
| CDFI | 血流可增多、减少或不明显，分布可紊乱，但血流丰富并非可靠的单独恶性标准 |
| 淋巴结 | 颈部淋巴结转移（皮髓质分界不清、微钙化） |

### ACR TI-RADS分级与处理

> TI-RADS存在不同版本。以下明确采用 **ACR TI-RADS**，根据组成、回声、形态、边缘和强回声灶积分；是否穿刺必须同时考虑分级和最大径，不能仅凭“4级或5级”直接决定FNA。

| 分级 | 积分/风险描述 | ACR处理阈值 |
|---|---|---|
| TR1 | 0分，良性 | 不需FNA或随访 |
| TR2 | 2分，不可疑 | 不需FNA或随访 |
| TR3 | 3分，轻度可疑 | ≥2.5cm建议FNA；≥1.5cm随访 |
| TR4 | 4～6分，中度可疑 | ≥1.5cm建议FNA；≥1.0cm随访 |
| TR5 | ≥7分，高度可疑 | ≥1.0cm建议FNA；≥0.5cm随访 |

### Graves病 vs 桥本 vs 甲状腺癌

| 项目 | Graves病 | 桥本甲状腺炎 | 甲状腺癌 |
|---|---|---|---|
| 累及范围 | 弥漫 | 弥漫 | 局灶结节 |
| 回声 | 弥漫减低 | 弥漫不均减低，网格状 | 低/极低回声结节 |
| 血流 | **"火海征"** | 增多（不如Graves） | 结节内丰富紊乱 |
| 特征 | 对称增大 | TPOAb/TgAb升高 | 微钙化+高大于宽 |

## 周围血管超声

### 动脉粥样硬化

| 表现 | 超声特征 |
|---|---|
| 内膜-中膜厚度（IMT） | 增厚（＞1.0mm） |
| 斑块 | 局部隆起突入管腔，可为软斑块（低回声）、硬斑块（强回声伴声影）、混合斑块 |
| 狭窄 | CDFI显示局部血流束变细、流速增高、频谱增宽 |
| 闭塞 | CDFI管腔内无血流信号 |

### 下肢深静脉血栓（DVT）

| 表现 | 超声特征 |
|---|---|
| 管腔 | 增宽，腔内**低/等回声**充填（急性：低回声/无回声；慢性：中高回声） |
| 压缩性 | 探头加压管腔**不能压闭**（最重要诊断征象） |
| CDFI | 血栓段**血流信号消失**或充盈缺损 |
| 远端挤压 | 正常时血流加速→DVT时无反应 |

### 考试答题版

Graves病：甲状腺弥漫增大+弥漫回声减低+"火海征"。桥本：弥漫不均质减低+网格状+TPOAb/TgAb升高。甲状腺癌：低回声结节+微钙化+高大于宽；ACR TI-RADS需按积分和结节大小共同决定FNA或随访。动脉粥样硬化：IMT增厚+斑块+狭窄处流速增高。DVT：管腔增宽+探头不能压闭+CDFI无血流信号。
