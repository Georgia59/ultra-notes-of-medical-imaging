## 高频机制速查

- **良性慢生长：**多造成骨质受压、变薄和光滑重塑。
- **恶性浸润：**多造成不规则、虫蚀状骨破坏并跨间隙扩展。
- **液体：**超声无回声伴后方增强，CT低密度，MRI多T1低T2高。
- **钙化/结石/气体：**超声强回声伴声影，CT高密度；MRI通常低信号。
- **高细胞肿瘤：**DWI受限、ADC降低，如淋巴瘤和部分鳞癌。
- **丰富血供：**强烈强化、流空或多普勒血流丰富，如鼻咽纤维血管瘤和颈动脉体瘤。
