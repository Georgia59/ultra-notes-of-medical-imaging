# 眼眶炎性假瘤与海绵状血管瘤

> 教材依据：《医学影像学》第9版第三章“眼眶特发性炎症”“海绵状血管瘤”。本页用于比较最常见的炎性浸润与良性低流量脉管性病变。

<a id="orbital-inflammatory-pseudotumor"></a>
## 一、眼眶炎性假瘤

### 1. 疾病本质

眼眶炎性假瘤现多称眼眶特发性炎症，是排除感染、甲状腺相关眼病和肿瘤后诊断的非特异性炎症，可累及泪腺、眼外肌、肌锥脂肪、巩膜及视神经周围组织。病理从急性水肿和炎细胞浸润逐渐转为纤维化。

### 2. 底层病理逻辑

| 病理因素 | 影像推导 |
| --- | --- |
| 炎细胞浸润和水肿 | 软组织增厚，T2信号可升高并强化。 |
| 炎症跨越正常解剖间隙 | 边界常不清，可同时累及多个眼眶结构。 |
| 眼外肌全程受累 | 肌腹和肌腱附着处均增粗，区别于典型Graves眼病。 |
| 慢性纤维化 | T2信号降低、强化可持续，运动受限明显。 |
| 血管通透性增加 | 增强后中度或明显强化。 |

### 3. 临床表现及机制

| 表现 | 机制 |
| --- | --- |
| 眼痛、眼红、眼睑肿胀 | 急性炎症和组织张力升高。 |
| 眼球突出 | 眶内炎性组织体积增大。 |
| 复视、眼球运动受限 | 眼外肌和肌腱受累。 |
| 视力下降 | 眶尖或视神经周围炎症压迫视神经。 |

### 4. CT 表现及机制

- 眶内不规则等密度软组织影，边界不清，可累及泪腺、眼外肌、肌锥和眶尖。
- 眼外肌增粗常包括肌腱，反映炎症沿肌肉全程扩展。
- 增强后较明显强化，源于炎症充血和通透性增加。
- 通常无侵袭性骨破坏；慢性压迫可有轻微骨重塑。

### 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 等或稍低信号 | 炎性组织替代正常脂肪。 |
| T2WI | 急性期偏高，慢性纤维型偏低 | 水肿延长T2，致密纤维缩短T2。 |
| FLAIR | 软组织高/等信号 | 非液体组织不被抑制。 |
| DWI/ADC | 可轻度受限，但通常不如淋巴瘤明显 | 炎细胞增加但细胞密度低于淋巴瘤。 |
| 增强T1WI | 中度至明显强化 | 充血和毛细血管通透性增加。 |

### 6. 超声表现及机制

病变多呈低回声不均匀软组织，眼外肌增粗；彩色多普勒可见血流增加。低回声与水肿和炎细胞浸润有关。

### 7. 关键征象机制

- **肌腹和肌腱同时增粗：**炎症不受肌腱附着界限限制。
- **跨间隙浸润：**炎性反应可沿筋膜、脂肪和血管周围扩散。
- **T2低信号慢性型：**胶原纤维增加、自由水减少。

### 8. 急性与慢性对比

| 项目 | 急性炎症型 | 慢性纤维型 |
| --- | --- | --- |
| 临床 | 疼痛红肿明显 | 疼痛可轻，运动受限持续 |
| T2 | 较高 | 较低 |
| 边界 | 模糊 | 浸润性增厚 |
| 激素反应 | 常较好 | 纤维化后较差 |

### 9. 鉴别诊断

| 疾病 | 区分点 | 底层逻辑 |
| --- | --- | --- |
| Graves眼病 | 双侧多见，肌腹增粗而肌腱相对保留，以下直肌/内直肌常见 | 自身免疫主要累及肌腹间质。 |
| 淋巴瘤 | 疼痛较少，边界相对清，DWI受限更明显、ADC更低 | 淋巴瘤细胞高度致密。 |
| 眶蜂窝织炎 | 有鼻窦感染和全身感染证据，可见脓肿 | 细菌感染而非特发性免疫炎症。 |

### 10. 考试答题版

眼眶炎性假瘤为非特异性炎症，可累及眼外肌、泪腺、眶脂肪及眶尖。CT表现为边界不清的不规则等密度软组织影，眼外肌增粗常累及肌腹及肌腱，增强明显。MRI呈T1等或低信号，急性水肿期T2较高，慢性纤维化时T2可降低，增强后明显强化。结合眼痛、急性起病及跨间隙受累有助诊断。

### 11. 一句话记忆

**疼痛急、边界糊、跨间隙，眼外肌连肌腱一起粗，是炎性假瘤。**

## 二、眼眶海绵状血管瘤

### 1. 疾病本质

眼眶海绵状血管瘤实质为边界清楚、包膜完整的低流量静脉性脉管畸形，多见于成人，常位于肌锥内。

### 2. 底层病理逻辑

- 扩张血管腔内血流缓慢，病变边界清楚并呈渐进性充填。
- 病变膨胀性生长，推移而非浸润眼球、视神经和眼外肌。
- 血液含水丰富，T2高；缓慢血流可见内部渐进强化。

### 3. 临床表现及机制

缓慢无痛性轴性眼球突出最常见，源于肌锥内肿块向前推眼球；压迫视神经时出现视力下降。

### 4. CT 表现及机制

- 肌锥内圆形或椭圆形、边界光滑的等密度肿块。
- 动态增强早期可不均匀点片强化，延迟期逐渐均匀。
- 机制：对比剂由血管丰富区向扩张、低流速血窦逐步充填。

### 5. MRI 表现及机制

| 序列 | 表现 | 机制 |
| --- | --- | --- |
| T1WI | 等或稍低 | 血液和间质含水较多。 |
| T2WI | 明显高信号，可有低信号纤维隔 | 静脉血窦含水丰富。 |
| DWI/ADC | 通常不明显受限 | 血管腔和细胞外空间较大。 |
| 动态增强 | 由局灶到弥漫的渐进性强化 | 低流量血窦逐渐被对比剂充填。 |

### 6. 超声表现及机制

多为边界清楚的低至中等回声实性肿块，可压缩性有限；彩色多普勒因低流量可能仅见少量血流。

### 7. 关键征象机制

- **肌锥内、边界光滑：**病变包膜完整、膨胀生长。
- **渐进性强化：**低流量扩张血窦逐步充填。
- **结构受压移位而不被侵犯：**良性膨胀性生长。

### 8. 典型与非典型

典型病变均匀、边界清、渐进强化；合并血栓、出血或纤维化时信号和强化可不均。

### 9. 鉴别诊断

神经鞘瘤可同为肌锥内肿块，但较大时囊变、强化不均，通常缺乏典型渐进性充填；淋巴瘤常塑形包绕结构且弥散受限。

### 10. 考试答题版

眼眶海绵状血管瘤多位于肌锥内，表现为圆形或椭圆形、边界清楚光滑的肿块，推移周围结构。CT多呈等密度，MRI T1等或稍低、T2明显高信号。动态增强具有由局灶性强化逐渐向全瘤均匀强化发展的渐进性充填特点，反映低流量扩张血窦。

### 11. 一句话记忆

**肌锥内圆而光，T2亮，动态增强慢慢填满。**
