## 原发腹膜后肿瘤

### 1. 疾病本质

原发腹膜后肿瘤指原发于腹膜后间隙内脂肪、肌肉、纤维、淋巴、神经等组织的肿瘤（不包括腹膜后各器官发生的肿瘤）。恶性约占**85%**，以间叶组织来源的**肉瘤**最常见（脂肪肉瘤、平滑肌肉瘤、纤维肉瘤等）。良性少见（脂肪瘤、神经鞘瘤、畸胎瘤等）。临床表现缺乏特异性，肿瘤较大压迫邻近器官时才出现症状。

### 2. 底层病理逻辑

- **病理因素：** 不同组织来源→不同密度/信号特征。腹膜后间隙解剖空间大→肿瘤常生长至很大才出现症状。良恶性肿瘤的生长方式不同→良性边界清晰，恶性浸润周围结构。
- **对影像的影响：** 脂肪肉瘤含脂肪密度。平滑肌肉瘤广泛坏死。肿瘤使腹膜后脏器受压移位→可据此判断肿瘤起源间隙。

### 3. CT表现

| 特征 | 良性 | 恶性 |
|---|---|---|
| 大小 | 一般较小 | **常巨大** |
| 质地 | 均匀 | **不均质**（坏死、囊变） |
| 边界 | 清楚 | 不清，浸润周围结构 |
| 增强 | 均匀强化 | **不均匀强化** |

| 肿瘤类型 | CT特征 |
|---|---|
| 脂肪肉瘤 | 含**脂肪性低密度灶**+软组织分隔 |
| 平滑肌肉瘤 | **广泛坏死**低密度区 |
| 神经源性肿瘤 | 脊柱两旁，界清 |

### 4. 腹膜后脏器受压移位规律（判断肿瘤起源）

| 肿瘤间隙 | 脏器移位方向 |
|---|---|
| 右侧肾旁前间隙 | 升结肠、十二指肠降段→向前移位 |
| 左侧肾旁前间隙 | 胰体尾→推向右前方（病变在胰后方）或右后方（病变在胰前方） |
| 肾周间隙 | 肾脏受压推移、肾轴旋转 |

## 腹膜后淋巴瘤

### 1. 疾病本质

淋巴瘤可首先或单独累及腹膜后淋巴结，也可作为全身性疾病的一部分。肿瘤沿淋巴链和组织间隙生长，常包绕主动脉、下腔静脉及其分支，而早期不明显破坏血管。

### 2. 影像学表现及机制

| 检查 | 典型表现 | 机制 |
|---|---|---|
| CT平扫 | 腹主动脉旁、下腔静脉旁多发结节或融合成均匀软组织肿块 | 多站淋巴结肿大融合 |
| CT增强 | 多呈轻至中度较均匀强化；治疗前明显坏死相对少 | 淋巴瘤细胞均匀、间质和血供较一致 |
| 血管关系 | 包绕血管但早期管腔仍较通畅 | 沿间隙塑形生长而非首先侵蚀血管 |
| MRI | T1WI低信号、T2WI中高信号，DWI明显高信号、ADC降低 | 高细胞密度限制水分子弥散 |
| 全身征象 | 可伴脾大及其他区域淋巴结增大 | 全身淋巴系统受累 |

### 3. 鉴别诊断

| 鉴别疾病 | 区分点 |
|---|---|
| 淋巴结转移瘤 | 常有明确原发癌，结节大小和强化不一，部分可坏死 |
| 腹膜后纤维化 | 多为主动脉前方板样组织，包绕输尿管并致内移和积水，晚期T2低信号 |
| 原发腹膜后肉瘤 | 多为单个巨大不均质肿块，推挤而非沿淋巴链包绕 |

### 4. 考试答题版

腹膜后淋巴瘤表现为腹主动脉及下腔静脉旁多发或融合的较均匀软组织肿块，轻至中度较均匀强化，可包绕大血管但早期不明显造成狭窄；MRI常见DWI明显受限。合并脾大及其他区域淋巴结肿大时更支持淋巴瘤。

### 5. 一句话记忆

**腹膜后均匀融合肿块包绕血管、DWI明显受限，又有多站淋巴结，优先考虑淋巴瘤。**

## 腹膜后转移瘤

### 1. 疾病本质

腹内脏器、睾丸及其他恶性肿瘤可经淋巴道、血行或沿筋膜间隙直接蔓延至腹膜后。常表现为腹膜后转移性淋巴结，也可形成浸润性软组织肿块。

### 2. 影像学表现及机制

- 腹主动脉旁、髂血管旁或原发肿瘤引流区出现多发淋巴结，大小、密度和强化可不一致。
- 鳞癌、生殖细胞肿瘤等转移结节可发生中央坏死，增强后呈周边强化。
- 病变可侵犯或压迫血管、输尿管，并造成肾积水。
- PET/CT出现异常代谢有助于寻找原发灶和全身分期，但炎症也可高代谢。

### 3. 鉴别诊断

| 鉴别疾病 | 区分点 |
|---|---|
| 腹膜后淋巴瘤 | 淋巴瘤常更均匀、融合并包绕血管，坏死相对少 |
| 腹膜后纤维化 | 纤维化呈板样包绕并以输尿管内移为特征，无明确原发肿瘤时需谨慎 |
| 原发腹膜后肉瘤 | 单个巨大肿块更常见，可含脂肪或广泛坏死等组织特征 |

### 4. 考试答题版

腹膜后转移瘤多由腹内脏器、睾丸或其他恶性肿瘤经淋巴道、血行或直接蔓延形成。CT和MRI常见腹主动脉旁或髂血管旁多发淋巴结，密度和强化可不均，部分出现中央坏死，并可侵犯血管、输尿管导致肾积水。诊断需结合原发肿瘤史和全身其他转移证据。

### 5. 一句话记忆

**有原发癌史＋引流区多发不均匀腹膜后淋巴结，先考虑转移；均匀融合包绕血管则更像淋巴瘤。**

## 腹膜后纤维化

### 1. 疾病本质

腹膜后纤维化是以腹膜后组织慢性非特异性炎症伴纤维组织增生为特点的少见疾病。分特发性（约占2/3）和继发性（药物、肿瘤、感染、创伤、手术、放疗等）。病理演变：早期（炎性渗出浸润）→中期（成纤维细胞增生+胶原形成）→后期（大量致密纤维化组织）。

**输尿管是最早并最常受侵的器官**→纤维组织包绕→上尿路积水和继发肾功能不全（最主要临床表现）。

### 2. 影像学表现

| 特征 | CT表现 | 机制 |
|---|---|---|
| 位置 | 肾门以下、第4/5腰椎前方，沿腹主动脉长轴走行，可达髂总动脉周围 | 病变沿大血管分布 |
| 形态 | 不规则软组织影，两侧对称或非对称，包绕腹主动脉、下腔静脉和输尿管 | 纤维组织增生包绕而非推挤 |
| 增强（早期） | **动脉期明显强化** | 炎症活动→血管丰富 |
| 增强（中晚期） | **轻度延迟强化**（纤维化越成熟强化越轻） | 纤维成分增多→血供减少 |
| MRI T2WI | 早期（活动期）→高/稍高信号；中晚期→**信号逐渐减低** | T2WI信号反映病变活动程度 |

### 3. 鉴别诊断

| 鉴别疾病 | 区分点 | 底层逻辑 |
|---|---|---|
| 腹膜后淋巴瘤 | 病变范围更广泛；T2WI常呈高信号 | 淋巴瘤细胞含水量高 |
| 淋巴结转移瘤 | 范围更广泛；T2WI高信号 | 转移瘤细胞含水量高 |
| 其他间叶肿瘤 | 对腹膜后组织主要为**推挤**而非包绕 | 肿瘤占位效应为主，纤维化为包绕性 |

腹膜后纤维化特征：**包绕而非侵犯**血管和输尿管（与恶性肿瘤的侵犯不同）。

### 4. 考试答题版

原发腹膜后肿瘤恶性约占85%，以间叶性肉瘤最常见。CT：恶性者常巨大、不均质、不均匀强化、边界不清；脂肪肉瘤含脂肪密度，平滑肌肉瘤广泛坏死。腹膜后纤维化：沿腹主动脉走行的不规则软组织影，**包绕**（非推挤）血管和输尿管，输尿管最常受累→上尿路积水。增强特点：活动期明显强化，晚期轻度延迟强化。T2WI信号反映活动程度（早期高→晚期低）。与淋巴瘤鉴别：淋巴瘤范围更广且T2WI高信号。

### 5. 一句话记忆

**巨大不均质后腹膜肿块 = 肉瘤（脂肪肉瘤含脂肪）；沿腹主动脉分布的包绕性软组织影+输尿管积水 = 腹膜后纤维化。**
