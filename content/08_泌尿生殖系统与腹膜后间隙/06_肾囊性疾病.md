## 单纯性肾囊肿

### 1. 疾病本质

单纯性肾囊肿是肾脏最常见的囊性病变，55岁以上者约50%可发现。为薄壁充液囊腔，临床多无症状。

### 2. 影像学表现

| 检查 | 表现 | 机制 |
|---|---|---|
| 超声 | 圆形/椭圆形**无回声区**，边界清晰，囊壁薄而光滑，**后方回声增强** | 液体无回声+透声好 |
| CT平扫 | 圆形/类圆形均匀**水样低密度**（0～20HU），边缘锐利 | 囊内为纯浆液 |
| CT增强 | **无强化**，境界更清楚 | 囊壁和囊液均无血供 |
| MRI | T1WI低信号、T2WI**明显高信号** | 液性成分特征 |

### 3. 复杂性囊肿与Bosniak分级

当囊肿合并出血、感染或钙化而成为复杂性囊肿时，需按Bosniak分级（2019版）评估：

| 分级 | 性质 | 关键特征 |
|---|---|---|
| **Ⅰ级** | 良性单纯性 | 壁薄（≤2mm）光滑，均匀液体，无分隔/钙化 |
| **Ⅱ级** | 良性（不需随诊） | 壁薄光滑，少（1～3个）且薄的分隔，或CT平扫均匀高密度（≥70HU） |
| **ⅡF级** | 不确定（需随诊） | 壁略增厚（3mm）+强化，或≥4个光滑薄分隔 |
| **Ⅲ级** | 中等概率恶性 | ≥1个强化厚（≥4mm）壁或分隔，或壁/分隔不规则增厚 |
| **Ⅳ级** | 恶性 | ≥1个强化结节（≥4mm） |

### 4. 一句话记忆

**肾内水样密度无强化囊性灶 + 壁菲薄光滑 = 单纯性肾囊肿（Bosniak Ⅰ级）；壁增厚/分隔/结节强化→升级。**

## 成人型多囊肾

### 1. 疾病本质

成人型多囊肾为**常染色体显性遗传病**，常合并多囊肝和多囊胰。囊肿发生始于胎儿期，累及所有肾单位，出生后囊肿大小和数量缓慢增大增多。中年后表现为腹部肿块、血尿和肾性高血压，晚期因肾脏结构被广泛破坏而出现**尿毒症**。

### 2. 影像学表现

| 检查 | 表现 | 机制 |
|---|---|---|
| 超声 | 双肾体积增大、形态失常，肾实质被无数大小不等**无回声区**替代，各无回声区**互不相通**，其间见纤细带状高回声 | 双侧弥漫多发囊肿 |
| CT | 双肾体积增大、边缘分叶状，布满多发大小不等囊肿，密度和信号特征同单纯性囊肿（水样密度，无强化），部分囊肿内可见出血呈稍高密度 | 囊肿广泛替代正常肾实质 |
| 伴随表现 | 可能发现同时存在的**多囊肝**和/或**多囊胰** | 多器官受累 |

### 3. 与单纯性多发肾囊肿的鉴别

| 特征 | 成人型多囊肾 | 多发单纯性肾囊肿 |
|---|---|---|
| 囊肿数量 | 无数，布满全肾 | 数个至数十个 |
| 正常肾实质 | 极少甚至难以识别 | 大部分保留 |
| 遗传性 | 常染色体显性遗传 | 散发 |
| 伴随 | 多囊肝/多囊胰 | 无 |

### 4. 考试答题版

单纯性肾囊肿：肾内水样密度/信号无强化囊性灶，壁薄光滑（Bosniak Ⅰ级）。Bosniak分级用于评估囊性病变恶性风险（2019版）。成人型多囊肾：常染色体显性遗传，双肾布满无数大小不等囊肿，正常肾实质极少，常合并多囊肝和多囊胰，晚期发生尿毒症。

### 5. 一句话记忆

**单纯囊肿 = 壁薄无强化水样密度；多囊肾 = 双肾无数囊肿+家族遗传+多囊肝/胰+晚期尿毒症。**
