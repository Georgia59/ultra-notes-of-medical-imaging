## 四、高频征象速查

| 征象 | 疾病 | 含义 |
|---|---|---|
| 鹿角状结石 | 肾结石 | 结石填充肾盂肾盏形成鹿角状 |
| 虫蚀状肾盏 | 肾结核（早期） | 肾小盏边缘不规则破坏 |
| 串珠状/笔杆状输尿管 | 输尿管结核 | 多发狭窄与扩张交替（串珠）或全程僵直（笔杆） |
| 肾自截 | 肾结核（晚期） | 全肾广泛钙化+肾功能完全丧失 |
| 充盈缺损（肾盂内） | 肾盂癌 | 肿瘤占据对比剂空间 |
| 充盈缺损（膀胱内） | 膀胱癌 | 乳头状肿瘤突入膀胱腔 |
| 瘤栓 | RCC | 肾静脉/下腔静脉增粗+腔内充盈缺损 |
| 反相位信号降低 | 肾上腺腺瘤 | 含丰富脂质→同反相位信号差异 |
| T2WI周围带低信号灶 | 前列腺癌 | 肿瘤取代正常高信号的周围带 |
| 联合带连续性中断 | 子宫内膜癌（肌层侵犯） | 肿瘤破坏子宫内膜与肌层界面 |
| 网膜饼 | 卵巢癌腹膜转移 | 大网膜被肿瘤浸润明显增厚 |
| 中心点征 | Caroli病 | 囊状扩张胆管包绕门静脉分支 |

## 五、本章总体诊断逻辑

### 泌尿系统

```
腰痛/血尿 → 超声→CT
  ├── 肾影内强回声+声影 → 肾结石
  ├── 输尿管走行区高密度+上方积水 → 输尿管结石
  ├── 肾实质富血供明显强化肿块+廓清快 → RCC
  ├── 肾盂内充盈缺损+轻度强化 → 肾盂癌
  ├── 含脂肪密度肿块 → AML
  ├── 肾盏虫蚀状+输尿管串珠状+钙化 → 肾结核
  └── 水样密度无强化 → 肾囊肿（Bosniak分级评估恶性风险）
```

### 肾上腺

```
肾上腺肿块 → CT/MRI
  ├── 双侧弥漫增大+形态正常 → 皮质增生
  ├── 低密度＜10HU+反相位信号降低 → 腺瘤
  ├── 大而不均+T2WI显著高信号+阵发性高血压 → 嗜铬细胞瘤
  └── 双侧+原发瘤史 → 转移瘤
```

### 前列腺

```
老年男性+排尿困难+PSA升高 → MRI
  ├── 移行带增大+增生结节+假包膜 → BPH
  └── 周围带T2WI低信号+DWI受限 → 前列腺癌
```

### 女性生殖系统

```
阴道出血 → MRI
  ├── 宫颈T2WI稍高信号取代低信号基质 → 宫颈癌
  └── 内膜增厚+联合带中断 → 子宫内膜癌

盆腔肿块 → 超声/MRI
  ├── T2WI明显低信号 → 子宫肌瘤
  ├── 囊实性+实性强化+腹膜转移 → 卵巢癌
  ├── 含脂肪+钙化 → 畸胎瘤
  └── 壁薄水样信号无强化 → 卵巢囊肿
```
