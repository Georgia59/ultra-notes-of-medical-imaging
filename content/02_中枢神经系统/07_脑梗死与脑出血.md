# 脑梗死与脑出血

> 教材依据：《医学影像学》第9版第二章“脑出血”“脑梗死”。急性卒中首先要回答：有无出血、是否大血管闭塞、是否存在可挽救的缺血半暗带。

<a id="cerebral-infarction"></a>
## 一、脑梗死

### 1. 疾病本质

脑梗死是脑血管闭塞导致供血区脑组织缺血坏死。核心病理链为：**灌注下降 → 能量耗竭 → 钠钾泵衰竭 → 细胞毒性水肿 → 不可逆坏死**。核心梗死周围低灌注但尚存活的组织称缺血半暗带。

### 2. 底层病理逻辑

- **细胞毒性水肿：**水进入细胞，细胞外间隙缩小，DWI高、ADC低，是超急性期最敏感征象。
- **总含水量增加：**数小时后CT衰减降低，T2/FLAIR升高。
- **血脑屏障破坏：**亚急性期可出现脑回样强化。
- **坏死液化：**慢性期形成脑软化灶，接近脑脊液密度/信号并伴局部脑萎缩。
- **血流再通后渗漏：**可发生出血性转化。

### 3. 临床表现及机制

| 临床表现 | 机制 |
| --- | --- |
| 突发偏瘫、偏身感觉障碍 | 运动或感觉传导通路供血中断。 |
| 失语 | 优势半球语言区缺血。 |
| 偏盲 | 视放射或枕叶缺血。 |
| 共济失调、眩晕 | 小脑或脑干后循环缺血。 |
| 症状突然达到高峰或阶梯加重 | 栓塞常突发；血栓形成可进展。 |

### 4. CT 表现及机制

#### CT 平扫

- 超急性期可正常；早期征象包括动脉致密征、岛带消失、豆状核轮廓模糊和脑沟变浅。
- **动脉致密征：**血栓内红细胞和纤维蛋白浓缩，X线衰减增高。
- 数小时至数日出现符合血管供血区的低密度、肿胀和占位效应。
- 慢性期低密度脑软化并伴脑沟、脑室扩大。

#### CT 增强/CTA/CTP

- CTA显示责任动脉狭窄或闭塞。
- CTP核心梗死区CBF/CBV下降；半暗带CBF下降、MTT/Tmax延长而CBV相对保存。
- 亚急性期可脑回样强化，反映屏障破坏和再灌注。

### 5. MRI 表现及机制

| 序列 | 表现 | 机制 |
| --- | --- | --- |
| T1WI | 急性期低信号 | 含水增加使T1延长。 |
| T2WI/FLAIR | 数小时后高信号；FLAIR早期可阴性 | 水肿使T2延长；超早期总含水变化尚不足。 |
| DWI/ADC | 数分钟出现DWI高、ADC低 | 细胞毒性水肿限制弥散。 |
| SWI/T2* | 可见责任动脉磁敏感血栓征或出血转化 | 血栓/血液产物造成磁敏感效应。 |
| PWI | 低灌注范围大于DWI核心时形成不匹配 | 代表尚未梗死的缺血半暗带。 |
| 增强T1WI | 亚急性脑回样强化 | 屏障破坏和再灌注。 |

### 6. 超声表现及机制

颈动脉超声可显示斑块和狭窄；经颅多普勒可检测颅内血流速度、方向和侧支循环。狭窄处血流速度升高源于管腔变窄后的流速代偿。

### 7. 关键征象机制

- **DWI高、ADC低：**细胞毒性水肿，是急性脑梗死核心征象。
- **血管供血区分布：**闭塞血管决定缺血组织范围。
- **DWI-PWI不匹配：**低灌注区大于不可逆核心，提示半暗带。
- **脑回样强化：**亚急性屏障破坏和再灌注。

### 8. 时期对比

| 时期 | CT | MRI/DWI | 增强 |
| --- | --- | --- | --- |
| 超急性 | 可正常或仅早期征 | DWI高、ADC低最敏感 | 通常无 |
| 急性 | 供血区低密度、肿胀 | T2/FLAIR逐渐高，DWI持续高 | 可无 |
| 亚急性 | 低密度和占位达高峰后减轻 | ADC逐渐假性正常化 | 脑回样强化 |
| 慢性 | 脑软化、萎缩 | 类脑脊液信号、胶质增生 | 一般不强化 |

### 9. 鉴别诊断

| 疾病 | 区分点 | 底层逻辑 |
| --- | --- | --- |
| 胶质瘤 | 不严格符合血管区，持续生长，灌注/Cho可升高 | 肿瘤浸润不受单一动脉区限制。 |
| 脑炎 | 有感染症状和特定分布，如颞叶 | 炎症传播模式不同于血管闭塞。 |
| 低血糖脑病 | 常双侧对称且结合低血糖史 | 系统代谢障碍而非局灶血管闭塞。 |

### 10. 考试答题版

脑梗死为脑血管闭塞引起的缺血性坏死。CT超急性期可无明显异常，早期可见责任动脉致密征、岛带消失、豆状核模糊和脑沟变浅，随后出现符合血管供血区的低密度及肿胀。MRI DWI可在发病数分钟出现高信号、ADC降低，是急性脑梗死最敏感征象；T2WI/FLAIR随后出现高信号。CTA/MRA显示血管闭塞，灌注成像可评估核心梗死和缺血半暗带。

### 11. 一句话记忆

**急性梗死先看DWI：高DWI、低ADC、沿血管区；灌注大于弥散就是半暗带。**

## 二、脑出血

### 1. 疾病本质

脑出血是脑实质内血管破裂形成血肿。高血压性脑出血常见于基底节、丘脑、脑桥和小脑；年轻患者或非典型部位应考虑血管畸形、动脉瘤、肿瘤、凝血障碍等。

### 2. 底层病理逻辑

- **新鲜凝血块浓缩：**CT高密度。
- **血肿周围炎症和屏障破坏：**形成血管源性水肿和占位效应。
- **血红蛋白逐步降解：**MRI信号按超急性、急性、亚急性和慢性阶段变化。
- **含铁血黄素沉积：**慢性期SWI/T2*低信号环长期存在。

### 3. 临床表现及机制

| 表现 | 机制 |
| --- | --- |
| 突发头痛、呕吐、意识障碍 | 血肿迅速形成并升高颅内压。 |
| 偏瘫、偏身感觉障碍 | 基底节/丘脑血肿破坏内囊及感觉通路。 |
| 瞳孔改变、脑疝 | 血肿及水肿造成中线移位和脑干受压。 |
| 症状常在活动中发生 | 血压骤升使病变小动脉破裂。 |

### 4. CT 表现及机制

#### CT 平扫

- 急性期类圆形或不规则高密度血肿，周围低密度水肿。
- 可破入脑室或蛛网膜下腔，形成脑室内/沟池高密度。
- 血肿和水肿可压迫脑室、中线移位。
- 随时间血肿从周边向中心密度降低，最终形成低密度软化灶。

#### CT 增强/CTA

- 急性单纯血肿一般不需增强。
- CTA“点征”为血肿内对比剂外渗，提示活动性出血和血肿扩大风险。
- 非典型出血应行CTA/MRA寻找血管畸形或动脉瘤。

### 5. MRI 表现及机制

| 时期 | T1WI | T2WI | 主要物质与机制 |
| --- | --- | --- | --- |
| 超急性 | 等/低 | 高 | 氧合血红蛋白，水含量高 |
| 急性 | 等/低 | 低 | 细胞内脱氧血红蛋白造成T2缩短 |
| 早期亚急性 | 高 | 低 | 细胞内高铁血红蛋白缩短T1 |
| 晚期亚急性 | 高 | 高 | 红细胞破裂，细胞外高铁血红蛋白 |
| 慢性 | 中心液化，周边低 | 周边低 | 含铁血黄素/铁蛋白造成磁敏感低信号 |

SWI/T2*对微出血和陈旧出血最敏感。

### 6. 超声表现及机制

婴幼儿经囟超声中急性血肿多呈高回声，因血块内部反射界面增多；液化后回声降低。

### 7. 关键征象机制

- **CT高密度血肿：**新鲜凝血块蛋白和细胞浓缩。
- **周围低密度环：**血肿周围血管源性水肿。
- **SWI微出血：**含铁血液产物造成磁敏感效应。
- **CTA点征：**对比剂正在外渗，提示持续出血。

### 8. 病因位置对比

| 类型 | 常见位置 | 提示 |
| --- | --- | --- |
| 高血压性 | 基底节、丘脑、脑桥、小脑 | 深部穿支小动脉病变 |
| 脑淀粉样血管病 | 老年人脑叶皮质下 | 多发脑叶微出血 |
| 血管畸形 | 年轻人、任何部位 | 邻近流空血管/异常血管团 |
| 肿瘤性出血 | 肿块内、周围强化 | 血肿吸收后仍有占位或结节强化 |

### 9. 鉴别诊断

| 疾病 | 区分点 | 底层逻辑 |
| --- | --- | --- |
| 钙化 | CT均高密度，但钙化稳定、无水肿；双能CT/SWI相位可辅助 | 钙盐和血液均高衰减但生物演变不同。 |
| 出血性肿瘤 | 周围结节强化、持续占位、灌注增高 | 肿瘤组织在血肿之外仍有活性血供。 |
| 出血性梗死 | 符合血管供血区，DWI有梗死背景 | 再灌注导致坏死组织内点片出血。 |

### 10. 考试答题版

急性脑出血CT表现为脑实质内类圆形或不规则高密度血肿，周围可见低密度水肿，伴脑室受压、中线移位，血液可破入脑室或蛛网膜下腔。高血压性出血常位于基底节、丘脑、脑桥和小脑。MRI信号随血红蛋白演变而变化，亚急性期因高铁血红蛋白常呈T1高信号，慢性期血肿周围含铁血黄素在T2*/SWI上呈低信号环。CTA点征提示活动性出血和血肿扩大风险。

### 11. 一句话记忆

**急性出血CT一眼高密度；MRI记住亚急性T1高，陈旧期SWI留黑环。**
