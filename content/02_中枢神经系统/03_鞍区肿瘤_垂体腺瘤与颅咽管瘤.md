# 鞍区肿瘤：垂体腺瘤与颅咽管瘤

> 教材依据：《医学影像学》第9版第二章“垂体瘤”“颅咽管瘤”；课程 PPT 第321-341张。

<a id="pituitary-adenoma"></a>
## 一、垂体腺瘤

### 1. 疾病本质

垂体腺瘤起源于腺垂体细胞，30-60岁常见。按激素分泌分为功能性和非功能性；直径小于10mm为微腺瘤，大于10mm为大腺瘤。肿瘤自鞍内生长，可向鞍上、蝶窦及海绵窦扩展。

### 2. 底层病理逻辑

- **腺垂体来源：**病变中心在鞍内，早期改变为垂体增大、上缘膨隆、垂体柄偏移。
- **微腺瘤血供与正常垂体不同：**正常垂体在增强早期迅速强化，微腺瘤相对延迟，因此动态增强早期呈低强化结节。
- **大腺瘤受鞍膈限制：**向鞍上生长时在鞍膈处受压，形成“8字形”或“雪人征”。
- **肿瘤增大后缺血：**可发生出血、坏死、囊变，形成混杂密度/信号和不均匀强化。
- **慢性压迫骨质：**蝶鞍扩大、鞍底下陷或骨质吸收。

### 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 泌乳、闭经、肢端肥大、库欣表现 | 功能性腺瘤分泌相应激素。 |
| 双颞侧偏盲 | 鞍上生长压迫视交叉中央交叉纤维。 |
| 垂体功能低下 | 大腺瘤压迫正常垂体组织和垂体柄。 |
| 眼球运动障碍 | 向海绵窦侵犯累及第Ⅲ、Ⅳ、Ⅵ脑神经。 |
| 突发头痛、视力下降 | 垂体卒中造成瘤内急性出血或梗死。 |

### 4. CT 表现及机制

#### CT 平扫

- 大腺瘤多为鞍内及鞍上等或稍高密度肿块；出血呈高密度，囊变坏死呈低密度。
- 蝶鞍扩大、鞍底下陷或破坏，源于长期慢性压迫。
- 微腺瘤平扫常不易显示，因为体积小且与正常垂体密度接近。

#### CT 增强

- 大腺瘤可均匀、不均匀或环形强化，取决于活肿瘤、坏死和囊变比例。
- 微腺瘤动态增强早期常低于正常垂体，延迟期差异可缩小。

### 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 多为稍低信号；出血可高信号 | 肿瘤含水增加；亚急性血液中高铁血红蛋白缩短T1。 |
| T2WI | 等或高信号，囊变更高 | 自由水增加。 |
| FLAIR | 实性部分等/高信号，囊液依成分而异 | 蛋白含量影响液体抑制程度。 |
| DWI/ADC | 一般无特异性；急性梗死区可受限 | 缺血后细胞毒性水肿限制弥散。 |
| 动态增强T1WI | 微腺瘤早期相对低强化；大腺瘤明显强化 | 正常垂体先强化，微腺瘤血流和对比剂到达较慢。 |

### 6. 超声表现及机制

颅内垂体腺瘤不以超声诊断。

### 7. 关键征象机制

- **动态增强早期低强化结节：**正常垂体先强化，微腺瘤强化延迟；是微腺瘤检出的关键。
- **雪人征：**鞍内肿瘤向上穿过鞍膈，腰部受鞍膈束缚。
- **海绵窦侵犯和颈内动脉包绕：**肿瘤突破鞍旁边界沿低阻力间隙生长。

### 8. 分型对比

| 项目 | 微腺瘤 | 大腺瘤 |
| --- | --- | --- |
| 大小 | <10mm | >10mm |
| 临床 | 多因激素异常早期发现 | 多有视野缺损、垂体低下和压迫症状 |
| CT | 常不显示 | 鞍内/鞍上肿块和骨改变明显 |
| MRI | 小异常信号，动态早期低强化 | 雪人状肿块，可坏死、出血 |
| 关键 | 动态薄层增强 | 中心在鞍内、蝶鞍扩大 |

### 9. 鉴别诊断

| 疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 鞍结节脑膜瘤 | 鞍上实性强化肿块 | 脑膜瘤中心在鞍上、宽基底、均匀显著强化，可有脑膜尾；垂体瘤中心在鞍内并扩大蝶鞍 | 起源分别为硬膜与腺垂体。 |
| 颅咽管瘤 | 鞍上/鞍内肿块 | 颅咽管瘤多囊实性、钙化常见；垂体瘤钙化少 | 胚胎残余上皮易产生角蛋白、胆固醇和钙化。 |
| 动脉瘤 | 鞍旁肿块、可强化 | CTA/MRA见与血管相连及流空 | 本质为血管腔扩张。 |

### 10. 考试答题版

垂体腺瘤起源于腺垂体，小于10mm为微腺瘤，大于10mm为大腺瘤。大腺瘤表现为鞍内及鞍上肿块，蝶鞍扩大，向鞍上生长可呈雪人征，CT为等或稍高密度，MRI呈T1稍低、T2等或高信号，增强明显，可因坏死、囊变或出血而不均匀。微腺瘤以薄层动态增强MRI最佳，增强早期在明显强化的正常垂体内呈相对低强化结节，并可见垂体上缘膨隆、垂体柄偏移和鞍底下陷。

### 11. 一句话记忆

**微腺瘤看动态早期低强化，大腺瘤看鞍内中心、蝶鞍扩大和雪人征。**

## 二、颅咽管瘤

### 1. 疾病本质

颅咽管瘤起源于胚胎颅咽管残余上皮，是鞍上区常见良性肿瘤，儿童和老年人可见双峰发病。虽属良性，但邻近视交叉、下丘脑和垂体，局部危害显著。多为囊性或囊实性，囊液可富含胆固醇、角蛋白和蛋白质，囊壁常钙化。

### 2. 底层病理逻辑

- **胚胎残余上皮位于鞍上：**病变中心通常在鞍上，向鞍内或第三脑室延伸。
- **胆固醇和高蛋白囊液：**可使T1WI呈高信号；若以水为主则T1低、T2高。
- **角蛋白和钙化：**CT呈高密度钙化；MRI多为T1/T2低信号。
- **囊壁和实性成分有血供：**增强后强化；囊液无血供不强化。
- **压迫下丘脑-垂体和视路：**产生内分泌、发育及视觉症状。

### 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 生长发育迟缓、垂体功能低下 | 压迫垂体和垂体柄。 |
| 尿崩症 | 损害下丘脑-神经垂体抗利尿激素通路。 |
| 肥胖、嗜睡、体温或行为异常 | 下丘脑受压或侵犯。 |
| 视力下降、双颞侧偏盲 | 压迫视交叉。 |
| 头痛、呕吐、脑积水 | 向第三脑室生长并阻塞脑脊液循环。 |

### 4. CT 表现及机制

#### CT 平扫

- 鞍上为主的囊性、实性或囊实性肿块。
- 点片状、弧线状或蛋壳样钙化常见，儿童更典型；钙盐使X线衰减显著增高。
- 囊液密度取决于蛋白和胆固醇浓度，高蛋白时可高于脑脊液。

#### CT 增强

- 囊壁和实性部分呈环形或团块状强化，因为有血供；囊腔不强化。

### 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 囊液可高、等或低信号 | 胆固醇/蛋白浓度高时T1缩短；清亮液体以水为主时T1低。 |
| T2WI | 多为高信号，角蛋白或钙化可低信号 | 水延长T2；致密角蛋白和钙化缺乏可移动质子。 |
| FLAIR | 高蛋白囊液常不能完全被抑制 | 囊液并非单纯脑脊液。 |
| DWI/ADC | 角蛋白丰富区域可受限 | 黏稠内容物限制水分子运动。 |
| 增强T1WI | 囊壁及实性部分强化 | 有血供的上皮和间质强化，囊液无血供。 |

### 6. 超声表现及机制

常规超声不用于颅咽管瘤诊断。

### 7. 关键征象机制

- **鞍上囊实性肿块伴钙化：**胚胎残余上皮发生囊变、角化和钙盐沉积。
- **T1高信号囊液：**胆固醇或高蛋白缩短T1。
- **囊壁环形强化：**囊壁有血供，囊液无血供。

### 8. 分型对比

| 项目 | 造釉细胞型 | 乳头型 |
| --- | --- | --- |
| 人群 | 儿童多见 | 成人多见 |
| 病理 | 角化、胆固醇裂隙和钙化明显 | 实性乳头状上皮为主 |
| CT | 囊实性、钙化常见 | 实性较多、钙化少 |
| MRI | 囊液信号复杂 | 实性成分较突出 |
| 鉴别重点 | 与Rathke囊肿、畸胎瘤鉴别 | 与垂体大腺瘤、脑膜瘤鉴别 |

### 9. 鉴别诊断

| 疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 垂体大腺瘤 | 鞍内鞍上肿块 | 垂体瘤鞍内中心、蝶鞍扩大，钙化少；颅咽管瘤鞍上中心、囊实性和钙化多 | 起源部位和组织成分不同。 |
| Rathke裂囊肿 | 鞍区囊性病变 | 通常无钙化、壁薄且强化少，可见囊内结节 | 单纯上皮囊肿，增殖和血供较少。 |
| 鞍上脑膜瘤 | 实性强化 | 脑膜瘤均匀显著强化并有宽基底、脑膜尾；颅咽管瘤囊变和钙化多 | 脑膜瘤血供丰富而囊变少。 |

### 10. 考试答题版

颅咽管瘤起源于胚胎颅咽管残余上皮，多位于鞍上，可向鞍内及第三脑室生长。CT 多表现为鞍上囊性或囊实性肿块，囊壁或实性部分常有点片状、弧线状或蛋壳样钙化，增强后囊壁及实性部分强化而囊腔不强化。MRI 信号因囊液成分而异：以水为主时T1低、T2高，富含蛋白或胆固醇时T1可高，角蛋白或钙化可呈T1、T2低信号。结合儿童或老年患者、鞍上位置、囊实性和钙化可提示诊断。

### 11. 一句话记忆

**颅咽管瘤：鞍上、囊实、爱钙化，胆固醇高时T1也高。**
