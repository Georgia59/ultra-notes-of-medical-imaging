# 颅脑外伤

> 教材依据：《医学影像学》第9版第二章“脑外伤”；学习大纲重点为脑挫裂伤、硬膜外血肿、硬膜下血肿和蛛网膜下腔出血。急性颅脑外伤首选 CT。

## 1. 疾病本质

颅脑外伤由外力导致颅骨、脑实质、脑膜和血管损伤。影像判断的核心是：**血液位于哪个解剖间隙、损伤是否累及脑实质、是否造成占位效应和脑疝。**

## 2. 底层病理逻辑

| 病理因素 | 影像推导 |
| --- | --- |
| 新鲜凝血块蛋白和细胞浓缩 | 急性血肿 CT 呈高密度。 |
| 血肿液化和血红蛋白降解 | 随时间CT密度逐渐降低；MRI信号按出血时期规律变化。 |
| 血肿受硬膜附着限制 | 硬膜外血肿多呈双凸透镜形，通常不跨颅缝。 |
| 血液沿硬膜下间隙铺展 | 硬膜下血肿呈新月形，可跨颅缝但受大脑镰、小脑幕限制。 |
| 脑实质撞击和血管破裂 | 脑挫裂伤形成出血、坏死和水肿混杂灶，常位于额颞叶底面。 |
| 蛛网膜下腔积血 | 血液沿脑沟、脑池分布，CT显示沟池高密度。 |
| 轴索剪切损伤 | 灰白质交界、胼胝体和脑干出现微出血及弥散异常，CT可阴性。 |

## 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 意识障碍 | 弥漫脑损伤、脑干损伤或颅内压升高。 |
| 头痛、呕吐 | 脑膜刺激和颅内压增高。 |
| “中间清醒期” | 硬膜外血肿早期短暂恢复，随后动脉性血肿扩大压迫脑组织。 |
| 进行性昏迷、瞳孔不等大 | 血肿扩大形成颞叶钩回疝，压迫动眼神经和脑干。 |
| 局灶功能缺损 | 局部脑挫裂伤或血肿压迫功能区。 |

## 4. CT 表现及机制

### CT 平扫

#### 脑挫裂伤

- 额叶、颞叶前下部常见斑片状低密度水肿内混有点片状高密度出血。
- 机制：脑组织撞击颅底骨嵴，形成毛细血管破裂、组织坏死和细胞毒性/血管源性水肿。
- 病灶可在数小时内“进展”，故早期阴性或较轻时需复查CT。

#### 硬膜外血肿

- 颅骨内板下双凸透镜形高密度影，边界清楚，通常不跨颅缝。
- 常伴邻近颅骨骨折。
- 机制：多为脑膜中动脉破裂，血液剥离硬膜，但硬膜在颅缝处附着牢固。

#### 硬膜下血肿

- 颅骨内板下新月形高密度影，可跨颅缝并广泛覆盖半球。
- 机制：桥静脉撕裂后，血液在潜在硬膜下间隙铺展。
- 亚急性期可与脑实质等密度，慢性期呈低密度；此时需观察脑沟消失、中线移位等间接征象。

#### 外伤性蛛网膜下腔出血

- 脑沟、脑裂和脑池内条片状高密度。
- 机制：软脑膜或皮质表面小血管破裂，血液进入蛛网膜下腔并随脑脊液分布。

#### 弥漫性轴索损伤

- CT可正常，或仅见灰白质交界、胼胝体及脑干小出血灶。
- 机制：旋转加减速产生剪切力，损伤轴索和小血管。

### CT 增强

急性外伤一般不需常规增强。增强可能掩盖少量出血；血管损伤疑虑时可行CTA。

## 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI/T2WI | 血肿信号随时期变化；水肿T1低、T2高 | 血红蛋白氧合状态、铁价态及红细胞完整性改变T1/T2。 |
| FLAIR | 脑挫裂伤水肿和蛛网膜下腔异常高信号 | 含蛋白/血液的脑脊液不能被完全抑制。 |
| DWI/ADC | 轴索损伤及急性细胞毒性水肿可受限 | 细胞膜泵衰竭使水进入细胞，细胞外空间缩小。 |
| SWI/T2* | 对微出血最敏感 | 脱氧血红蛋白和含铁血黄素产生磁敏感效应。 |
| 增强T1WI | 急性期通常非核心检查 | 强化并不能替代平扫对出血的判断。 |

## 6. 超声表现及机制

成人不常用。婴幼儿经囟超声可显示脑室内出血和部分实质损伤；急性血块可呈高回声，随后液化回声降低。

## 7. 关键征象机制

- **双凸透镜形血肿：** 硬膜被血液剥离但受颅缝固定，提示硬膜外血肿。
- **新月形血肿：** 血液沿硬膜下潜在间隙广泛铺展，提示硬膜下血肿。
- **脑沟脑池高密度：** 血液进入蛛网膜下腔并随脑脊液分布。
- **额颞叶底部混杂灶：** 脑组织撞击不规则颅底形成对冲性挫裂伤。
- **SWI多发微出血：** 剪切损伤累及小血管，支持弥漫性轴索损伤。

## 8. 急性与慢性血肿对比

| 项目 | 急性 | 亚急性 | 慢性 |
| --- | --- | --- | --- |
| CT | 高密度 | 等密度阶段可漏诊 | 低密度，可分隔或再出血 |
| MRI | 信号复杂，T2*敏感 | 高铁血红蛋白使T1高 | 含铁血黄素低信号包膜、液化 |
| 机制 | 新鲜凝血块浓缩 | 血红蛋白氧化及红细胞破坏 | 蛋白降解、液化和含铁沉积 |

## 9. 鉴别诊断

| 鉴别 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 硬膜外 vs 硬膜下血肿 | 均为颅骨内板下高密度 | 双凸且不跨颅缝 vs 新月形且可跨颅缝 | 解剖间隙和硬膜附着不同。 |
| 亚急性硬膜下血肿 vs 正常脑实质 | 可等密度 | 脑沟受压、中线移位、增强后膜强化 | 血肿密度变化但占位仍存在。 |
| 蛛网膜下腔出血 vs 高密度血管 | 均位于沟池 | 出血范围异常且常不对称，结合外伤和复查 | 血液充填脑脊液间隙。 |

## 10. 考试答题版

急性颅脑外伤首选CT平扫。脑挫裂伤多位于额、颞叶前下部，表现为低密度水肿内混有点片状高密度出血，可伴占位效应。硬膜外血肿表现为颅骨内板下双凸透镜形高密度影，通常不跨颅缝，常伴骨折；硬膜下血肿呈新月形，可跨颅缝并广泛覆盖脑表面。蛛网膜下腔出血表现为脑沟、脑裂及脑池内高密度。MRI尤其SWI对弥漫性轴索损伤的微出血更敏感，DWI可显示急性轴索和细胞毒性损伤。

## 11. 一句话记忆

**硬外双凸不跨缝，硬下新月可跨缝；沟池见血是蛛网膜下，额颞混杂是脑挫裂。**
