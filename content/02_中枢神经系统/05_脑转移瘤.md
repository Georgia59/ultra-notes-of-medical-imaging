# 脑转移瘤

> 教材依据：《医学影像学》第9版第二章“脑转移瘤”；课程 PPT 第355-360张。

## 1. 疾病本质

脑转移瘤是颅外恶性肿瘤经血行播散至脑内形成的继发肿瘤，多见于中老年人。常见原发瘤包括肺癌、乳腺癌、肾癌和黑色素瘤等。转移瘤多停留于脑内小动脉末端，故常位于皮髓质交界区，也可累及小脑和脑膜。

## 2. 底层病理逻辑

| 病理因素 | 影像推导 |
| --- | --- |
| 血行播散 | 常多发，分布于灰白质交界和血流丰富区域。 |
| 膨胀性生长、边界相对清楚 | 瘤体常类圆形，较胶质瘤少沿白质弥漫浸润。 |
| 血脑屏障显著破坏 | 多明显结节状或环形强化。 |
| 周围血管源性水肿 | 白质低密度/T2-FLAIR高信号，常明显大于瘤体。 |
| 生长快、中心缺血 | 较大病灶常坏死，形成环形强化。 |
| 某些原发瘤易出血 | 黑色素瘤、绒癌、肾癌、甲状腺癌等转移可出现高密度或磁敏感出血。 |

## 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 头痛、呕吐、视盘水肿 | 多发肿瘤和广泛水肿使颅内压升高。 |
| 癫痫 | 皮质附近转移灶刺激脑皮质。 |
| 偏瘫、失语、共济失调 | 转移灶位于相应功能区、小脑或压迫传导束。 |
| 症状进展较快 | 转移瘤增殖快且水肿明显。 |
| 全身肿瘤相关表现 | 由原发肿瘤及其他器官转移所致。 |

## 4. CT 表现及机制

### CT 平扫

- 常为多发类圆形等或低密度结节，位于灰白质交界区。
- 瘤周水肿明显，可出现“小瘤大水肿”；水肿主要位于白质，因为血管源性水肿沿细胞外间隙扩散。
- 坏死呈低密度，出血呈高密度。

### CT 增强

- 呈明显均匀结节状、块状或环形强化。
- 小病灶增强后比平扫更易检出；强化反映转移灶新生血管和血脑屏障破坏。

## 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 多为低或等信号；黑色素或亚急性出血可高信号 | 黑色素顺磁性及高铁血红蛋白缩短T1。 |
| T2WI | 多为高信号；黑色素或出血可低信号 | 含水增多使T2升高，顺磁物质使T2缩短。 |
| FLAIR | 瘤周大片高信号水肿 | 血管源性水肿不能被抑制。 |
| DWI/ADC | 实性区可不同程度受限，坏死中心多不明显受限 | 取决于细胞密度；坏死液通常弥散较自由。 |
| 增强T1WI | 多发结节状或环形明显强化 | 屏障破坏和新生血管。 |
| SWI/T2* | 出血性转移瘤呈低信号 | 血液降解产物造成磁敏感效应。 |

## 6. 超声表现及机制

常规超声不用于成人脑转移瘤诊断。术中超声可见实性或囊实性肿块及周围水肿。

## 7. 关键征象机制

- **灰白质交界分布：**血管在此变细、血流减慢，循环肿瘤栓易停留。
- **多发结节或环形强化：**血行播散形成多个病灶，较大病灶中心坏死。
- **小瘤大水肿：**转移灶释放血管通透性因子并破坏屏障，液体沿白质广泛扩散。
- **黑色素瘤转移T1高、T2低：**黑色素具有顺磁性，缩短T1和T2。

## 8. 分型对比

| 项目 | 实性转移灶 | 坏死囊性转移灶 | 出血性转移灶 |
| --- | --- | --- | --- |
| CT | 等或稍高密度结节 | 中心低密度 | 高密度 |
| MRI | T1低、T2高 | 中心长T1长T2 | 信号随出血时期变化 |
| 增强 | 均匀或结节状 | 环形 | 周边/实性部分强化 |
| 机制 | 活肿瘤为主 | 生长快于血供 | 脆弱新生血管破裂 |

## 9. 鉴别诊断

| 疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 胶质母细胞瘤 | 坏死、环形强化、水肿 | 胶质瘤多单发、沿白质浸润和跨胼胝体；转移瘤常多发、灰白质交界、小瘤大水肿 | 浸润传播路径不同。 |
| 脑脓肿 | 环形强化和水肿 | 脓腔DWI明显受限，脓肿壁常薄而光滑；转移坏死中心多不受限 | 脓液黏稠且细胞蛋白丰富。 |
| 淋巴瘤 | 多发、明显强化 | 淋巴瘤常位于深部/室周，实性均匀强化且ADC低，免疫正常者坏死少 | 淋巴瘤细胞密度高。 |

## 10. 考试答题版

脑转移瘤多由肺癌、乳腺癌等经血行播散而来，常多发并位于灰白质交界区。CT平扫多为等或低密度结节，周围血管源性水肿明显，可见小瘤大水肿；坏死为低密度，出血为高密度。增强后呈明显结节状、块状或环形强化。MRI多呈T1低、T2及FLAIR高信号，增强后多发明显强化；黑色素或出血性转移灶可呈T1高信号，SWI可显示出血。结合多发、灰白质交界、显著水肿及原发肿瘤史可提示诊断。

## 11. 一句话记忆

**多发、灰白质交界、小瘤大水肿、结节或环形强化，先想脑转移。**
