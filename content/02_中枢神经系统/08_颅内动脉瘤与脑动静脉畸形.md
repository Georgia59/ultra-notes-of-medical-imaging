# 颅内动脉瘤与脑动静脉畸形

<a id="intracranial-aneurysm"></a>
## 一、颅内动脉瘤

### 1. 疾病本质

颅内动脉瘤是脑动脉壁局限性异常扩张，好发于脑底动脉环分叉处。破裂后最常引起自发性蛛网膜下腔出血。

### 2. 底层病理逻辑

- 动脉壁中膜和弹力层薄弱，在血流冲击处逐渐膨出。
- 腔内高速血流使CTA/MRA/DSA显示囊状突出；MRI常见流空。
- 部分瘤腔血流缓慢形成附壁血栓，产生混杂密度/信号和不完全强化。
- 破裂后血液进入蛛网膜下腔，沿脑池、脑沟分布。

### 3. 临床表现及机制

| 表现 | 机制 |
| --- | --- |
| 雷击样剧烈头痛 | 血液突然进入蛛网膜下腔，强烈刺激脑膜。 |
| 颈项强直 | 脑膜刺激。 |
| 动眼神经麻痹 | 后交通动脉瘤压迫第Ⅲ脑神经。 |
| 意识障碍 | 颅内压骤升、再出血或脑血管痉挛。 |

### 4. CT 表现及机制

- 未破裂动脉瘤可呈圆形等/高密度结节，钙化或血栓使密度不均。
- 破裂后脑池、脑裂和脑沟高密度；出血最集中的脑池可提示动脉瘤位置。
- CTA显示与动脉相连的囊状、梭形突出及瘤颈。
- 增强仅使仍有血流的瘤腔强化，血栓不强化。

### 5. MRI 表现及机制

| 序列 | 表现 | 机制 |
| --- | --- | --- |
| T1/T2 | 高速血流呈流空，血栓呈分层混杂信号 | 流动质子离开成像层面；不同时期血栓信号不同。 |
| TOF-MRA | 动脉瘤高信号 | 流入增强效应。 |
| 增强MRA | 显示瘤腔和血管关系 | 对比剂缩短血液T1。 |
| SWI | 血栓、微出血低信号 | 磁敏感效应。 |

### 6. 超声

经颅多普勒主要用于蛛网膜下腔出血后监测血管痉挛；流速升高提示管腔痉挛变窄。

### 7. 关键征象机制

- **脑池高密度：**蛛网膜下腔积血。
- **与动脉连续的囊状突起：**动脉壁局限扩张。
- **部分血栓化“靶样/分层”表现：**中心流动血液、周围不同时期血栓。

### 8. 分型对比

| 类型 | 形态 | 常见机制 | 影像重点 |
| --- | --- | --- | --- |
| 囊状 | 分叉处囊形突出 | 局部壁薄弱 | 瘤颈和分支关系 |
| 梭形 | 一段血管环形扩张 | 动脉粥样硬化/夹层 | 累及血管长度 |
| 夹层性 | 壁内血肿、真假腔 | 内膜撕裂 | 壁内血肿和管腔改变 |

### 9. 鉴别诊断

需与血管襻、脑膜瘤等强化结节鉴别。多平面CTA/MRA见病灶与动脉腔连续是关键；DSA仍是精细显示瘤颈和分支关系的重要方法。

### 10. 考试答题版

颅内动脉瘤好发于脑底动脉环分叉处，破裂后常引起蛛网膜下腔出血。CT平扫表现为脑池、脑裂和脑沟内高密度积血；CTA显示与动脉相连的囊状或梭形扩张，可显示瘤颈、分支和附壁血栓。MRI可见高速血流流空，MRA显示瘤腔，部分血栓化动脉瘤呈分层混杂信号。DSA可精确显示动脉瘤形态及载瘤动脉。

### 11. 一句话记忆

**雷击头痛加脑池积血，CTA找与动脉相连的瘤颈。**

## 二、脑动静脉畸形

### 1. 疾病本质

脑动静脉畸形（AVM）是先天性异常血管团，供血动脉不经正常毛细血管床直接与引流静脉相通，形成高流量分流。

### 2. 底层病理逻辑

- 缺少毛细血管阻力，血流高速进入静脉，使供血动脉和引流静脉扩张迂曲。
- 血管团内高速流动在MRI产生多发流空。
- 长期高流量和盗血可造成周围脑缺血、胶质增生和萎缩。
- 畸形血管壁薄弱，易破裂出血；反复微出血可沉积含铁血黄素。

### 3. 临床表现及机制

| 表现 | 机制 |
| --- | --- |
| 脑出血或蛛网膜下腔出血 | 畸形血管或伴发动脉瘤破裂。 |
| 癫痫 | 血管团、缺血和胶质增生刺激皮质。 |
| 头痛 | 血流动力学异常或小量出血。 |
| 神经功能缺损 | 盗血、出血或压迫。 |

### 4. CT 表现及机制

- 平扫可见边界不清的等/稍高密度迂曲血管团，可有钙化和出血。
- 增强/CTA显示迂曲强化血管团、粗大供血动脉和早显引流静脉。
- 周围可见低密度缺血、软化或血肿。

### 5. MRI 表现及机制

| 序列 | 表现 | 机制 |
| --- | --- | --- |
| T1/T2 | 蜂窝状、蚯蚓状流空血管 | 高速血流导致流空。 |
| FLAIR | 周围胶质增生高信号 | 慢性缺血和组织反应。 |
| SWI | 静脉、微出血和含铁沉积明显 | 脱氧血及血液产物磁敏感。 |
| MRA | 显示血管团和供引流血管 | 流入增强或对比增强。 |
| DWI | 急性出血/缺血并发症可异常 | 反映继发组织损伤。 |

### 6. 超声

经颅多普勒可见供血动脉流速增高、阻力下降，源于低阻力动静脉分流。

### 7. 关键征象机制

- **流空血管团：**高速血液在回波采集前离开层面。
- **粗大供血动脉和早显静脉：**血流绕过毛细血管床直接分流。
- **盗血现象：**低阻力AVM分走邻近正常脑组织血流。

### 8. 检查方法对比

| 方法 | 优势 |
| --- | --- |
| CT | 快速检出出血、钙化 |
| CTA | 无创显示血管构筑 |
| MRI/MRA | 显示血管团、周围脑组织及陈旧出血 |
| DSA | 时间和空间分辨率最高，显示供血、瘤巢、引流，是治疗规划关键 |

### 9. 鉴别诊断

海绵状血管畸形通常无粗大供血动脉和早显静脉，MRI呈“爆米花样”混杂信号及含铁低信号环；AVM则以高流量流空血管团为核心。

### 10. 考试答题版

脑动静脉畸形是供血动脉不经毛细血管床直接与引流静脉相通形成的高流量血管团。CT可见迂曲血管影、钙化或继发出血，增强及CTA显示明显强化的血管团、粗大供血动脉和引流静脉。MRI表现为多发迂曲、蜂窝状流空血管影，SWI可显示异常静脉及陈旧出血。DSA可动态显示供血动脉、畸形血管团和早显引流静脉，是评价血管构筑的重要方法。

### 11. 一句话记忆

**AVM就是没有毛细血管的高速短路：供血动脉粗、瘤巢乱、引流静脉早显。**
