## 高频机制速查

- **急性梗死：** 细胞毒性水肿 → DWI高、ADC低。
- **脓肿：** 黏稠脓液 → 脓腔DWI高、ADC低。
- **高细胞肿瘤：** 细胞外间隙缩小 → ADC降低。
- **血脑屏障破坏：** 对比剂外漏 → 强化。
- **肿瘤坏死：** 中心无血供不强化，周边活肿瘤强化 → 不规则环形/花环状强化。
- **新鲜出血：** 凝血块高衰减 → CT高密度；MRI随血红蛋白状态演变。
