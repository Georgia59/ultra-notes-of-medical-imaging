# 脑膜瘤

> 教材依据：《医学影像学》第9版第二章“脑膜瘤”；课程 PPT 第298-320张。

## 1. 疾病本质

脑膜瘤起源于蛛网膜粒帽细胞，多为脑外、硬膜附着性肿瘤，好发于中老年女性。常见于大脑凸面、矢状窦旁、蝶骨嵴、嗅沟、大脑镰、小脑幕和桥小脑角。多数为 WHO 1级，生长缓慢、包膜相对完整；少数为非典型或间变型，可侵犯脑实质和颅骨。

## 2. 底层病理逻辑

| 病理因素 | 影像推导 |
| --- | --- |
| 脑外、宽基底附着硬膜 | 肿块与硬膜广泛相连，推压脑实质，可见脑脊液间隙、白质塌陷等脑外征象。 |
| 细胞和纤维成分较多 | CT 多呈等或稍高密度；MRI 信号常接近脑皮质，纤维越多，T2信号越低。 |
| 血供丰富，多由脑膜动脉供血 | CT/MRI 增强多均匀显著强化，PWI 的 rCBV 可明显增高。 |
| 砂粒体和钙化 | CT 呈点片状高密度；MRI 常呈 T1/T2低信号。 |
| 长期压迫颅骨 | 可引起邻近骨质增生、硬化；高级别肿瘤也可造成骨质破坏。 |
| 压迫静脉或静脉窦 | 静脉回流受阻可产生与肿瘤大小不成比例的血管源性水肿。 |
| 高级别肿瘤生长快、浸润 | 边界可不规则，坏死囊变和不均匀强化增多，脑实质侵犯更常见。 |

## 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 头痛、呕吐、视盘水肿 | 肿瘤及水肿造成占位效应和颅内压升高。 |
| 癫痫 | 大脑凸面或近皮质肿瘤长期压迫、刺激脑皮质。 |
| 局灶神经功能障碍 | 肿瘤压迫相邻运动、感觉、语言或脑神经结构。 |
| 症状进展缓慢 | 多数脑膜瘤生长缓慢，脑组织可长期代偿。 |

## 4. CT 表现及机制

### CT 平扫

- **类圆形或分叶状、边界清楚的等或稍高密度脑外肿块：**细胞、纤维和血管含量较高，X 线衰减高于含水较多的脑实质。
- **宽基底贴附硬膜：**反映蛛网膜粒帽细胞来源。
- **点片状钙化：**砂粒体钙盐沉积，CT 最敏感。
- **邻近颅骨增生或破坏：**缓慢刺激成骨多形成增生，高级别侵犯可形成破坏。
- **水肿和占位效应：**水肿多为静脉受压和肿瘤分泌因子导致的血管源性水肿。

### CT 增强

- **多数均匀显著强化：**血供丰富且肿瘤毛细血管内缺乏正常血脑屏障。
- **高级别可不均匀强化：**坏死、囊变或内部组织结构不均。
- **脑膜尾征：**邻近硬脑膜线状增厚强化，多由肿瘤浸润、静脉淤血或反应性增生共同形成，并非脑膜瘤绝对特异征。

## 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 等或稍低信号 | 细胞和水含量接近脑皮质。 |
| T2WI | 等或稍高信号；纤维型可偏低 | 自由水增多使 T2升高；胶原纤维限制质子运动并缩短 T2。 |
| FLAIR | 多为等或高信号，瘤周水肿高信号 | 肿瘤组织及血管源性水肿不能被脑脊液抑制脉冲完全抑制。 |
| DWI/ADC | 可轻度受限，高级别常更明显 | 细胞密度增加使细胞外间隙缩小。 |
| 增强 T1WI | 均匀显著强化，常见脑膜尾征 | 丰富血供及缺乏血脑屏障使钆对比剂快速进入组织间隙。 |
| SWI/T2* | 钙化或出血呈低信号 | 局部磁敏感差导致去相位。 |
| PWI | rCBV 明显增高 | 肿瘤血管丰富。 |
| MRS | Cho 升高，NAA 降低或缺如，可见 Ala 峰 | 细胞膜更新活跃且肿瘤内无正常神经元；丙氨酸峰有提示意义。 |

## 6. 超声表现及机制

常规超声不是颅内脑膜瘤主要检查。术中超声可见边界较清楚的实性肿块，回声取决于纤维、钙化和血管成分；钙化呈强回声，可伴声影；彩色多普勒可显示丰富血流。

## 7. 关键征象机制

- **宽基底贴附硬膜**
  - 机制：肿瘤起源于蛛网膜粒帽细胞并沿硬膜生长。
  - 提示意义：支持脑外、脑膜来源。
- **脑膜尾征**
  - 机制：邻近硬膜受肿瘤浸润、反应性增生和静脉淤血后增厚强化。
  - 提示意义：支持脑膜来源，但转移瘤、淋巴瘤等也可出现。
- **均匀显著强化**
  - 机制：肿瘤毛细血管丰富且无完整血脑屏障。
  - 提示意义：典型 WHO 1级脑膜瘤常见。
- **邻近骨质增生**
  - 机制：慢性压迫和肿瘤刺激成骨反应。
  - 提示意义：有助于与多数脑内肿瘤鉴别。

## 8. 分型/分级对比

| 项目 | 典型 WHO 1级 | 高级别脑膜瘤 |
| --- | --- | --- |
| 病理 | 生长慢、包膜较完整 | 细胞增殖快、可浸润脑实质 |
| 边界 | 清楚、规则 | 不规则，脑瘤界面消失 |
| 坏死囊变 | 少见 | 较常见 |
| 水肿 | 轻或无，但静脉受压时可重 | 常较明显 |
| 强化 | 均匀显著 | 不均匀、斑片状 |
| 骨改变 | 增生、硬化常见 | 破坏相对更多 |

## 9. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 胶质瘤 | 可为表浅肿块并伴水肿 | 胶质瘤位于脑内、边界浸润、强化常不均；无典型宽基底和骨增生 | 起源分别为脑实质与脑膜。 |
| 硬膜转移瘤 | 可有硬膜尾征和明显强化 | 常有原发癌史、多发、骨质破坏更多 | 转移瘤侵袭性生长，反应性成骨不如典型脑膜瘤。 |
| 神经鞘瘤 | 均为脑外肿瘤并明显强化 | 神经鞘瘤沿脑神经和神经孔生长，易囊变；脑膜瘤宽基底贴硬膜 | 起源结构不同。 |
| 淋巴瘤 | 可呈等高密度及明显强化 | 多位于脑内深部、DWI受限明显；骨增生少见 | 淋巴瘤细胞密度更高。 |

## 10. 考试答题版

脑膜瘤起源于蛛网膜粒帽细胞，多见于中老年女性，常位于大脑凸面、矢状窦旁、蝶骨嵴等处。CT 平扫多表现为类圆形或分叶状等或稍高密度脑外肿块，边界清楚，以宽基底与硬脑膜相连，可见钙化、瘤周水肿及邻近颅骨增生或破坏；增强后多均匀显著强化。MRI 上 T1WI 多呈等或稍低信号，T2WI 呈等或稍高信号，增强后明显均匀强化，邻近脑膜可见脑膜尾征。诊断要点为好发部位、脑外征象、宽基底、显著强化和骨质改变。

## 11. 一句话记忆

**脑外宽基贴硬膜，CT等高密度、增强均匀亮，脑膜尾加骨增生。**
