# 弥漫性胶质瘤

> **资料依据**
>

> - 学习大纲：胶质瘤属于颅内肿瘤重点内容及见习病例。
>
> **口径说明：**课程 PPT 沿用“星形细胞瘤Ⅰ-Ⅳ级”的传统教学表述；第 9 版教材依据 WHO 第 5 版，强调分子分型，成人型弥漫性胶质瘤包括星形细胞瘤（IDH 突变型）、少突胶质瘤（IDH 突变且 1p/19q 联合缺失）和胶质母细胞瘤（IDH 野生型）。考试若按教材作答，优先采用教材口径；比较影像时仍可使用“低级别与高级别”这一实用框架。

## 1. 疾病本质

弥漫性胶质瘤是起源于神经上皮组织的原发性中枢神经系统肿瘤，成人多发生于大脑，病变主要位于白质并常累及灰质。其本质不是单纯形成一个边界清楚的肿块，而是肿瘤细胞沿脑白质纤维束弥漫浸润正常脑组织，因此难以完整切除、容易复发。

- 成人型主要包括星形细胞瘤、少突胶质瘤和胶质母细胞瘤。
- 成人星形细胞瘤和少突胶质瘤均需有 **IDH 突变**；少突胶质瘤还需有 **1p/19q 联合缺失**。
- 胶质母细胞瘤为 **IDH 野生型、WHO 4级**。
- 儿童型中常见弥漫性中线胶质瘤，多位于脑干、丘脑或脊髓等中线结构。

## 2. 底层病理逻辑

| 病理因素 | 对影像的影响 |
| --- | --- |
| 弥漫浸润性生长 | 肿瘤细胞沿白质纤维束向周围扩展，因此形态不规则、边界不清；可沿胼胝体跨越中线。影像边界相对清楚也不代表病理上无浸润。 |
| 含水量增加、微囊样改变 | 水使 X 线衰减减低，CT 多呈低密度；同时延长 T1、T2弛豫，MRI 多表现为 T1WI 低信号、T2WI 高信号。 |
| 细胞密度增高 | 细胞外间隙缩小，水分子运动受限，因此 DWI 信号升高、ADC 值降低；通常恶性度越高越明显。 |
| 肿瘤新生血管增多 | 局部血容量和血流量增加，PWI 显示 rCBV、rCBF 增高；提示活性肿瘤组织及较高恶性度。 |
| 血管结构幼稚、血脑屏障破坏 | 对比剂由血管内进入细胞外间隙，产生强化；屏障破坏越明显，通常强化越显著，DCE 的 Ktrans 也越高。 |
| 坏死、囊变 | 中心液化区无有效血供，CT 呈低密度，MRI 呈长 T1、长 T2信号，增强后不强化；周边活肿瘤强化，形成不规则环形或花环状强化。 |
| 出血 | 急性血液在 CT 上衰减高，表现为高密度；MRI 信号随血红蛋白状态变化，亚急性期高铁血红蛋白常使 T1WI 呈高信号，SWI/T2*可见磁敏感低信号。 |
| 钙化 | CT 上因 X 线衰减高而呈高密度；MRI 多呈低信号并可在 SWI 上出现磁敏感效应。少突胶质瘤相对多见。 |
| 瘤周血管源性水肿 | 血脑屏障破坏后液体进入细胞外间隙，CT 呈低密度，T2WI/FLAIR 呈高信号；高级别肿瘤通常更明显。 |
| 占位效应 | 肿瘤及水肿共同增加局部体积，造成脑沟、脑池和脑室受压，中线结构移位，严重时可形成脑疝。 |
| 细胞膜合成和神经元破坏 | MRS 显示 Cho 增高，反映细胞膜合成与更新活跃；NAA 降低，反映正常神经元减少或受损；坏死和无氧代谢可产生 Lip、Lac 峰。 |

**总推导：**

> 浸润和含水增加解释“边界不清、CT 低密度、T2 高信号”；细胞密度增加解释“ADC 降低”；新生血管与屏障破坏解释“灌注增高和强化”；坏死、出血、囊变解释“混杂信号与花环状强化”。

## 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 癫痫 | 肿瘤累及或刺激大脑皮质，改变局部神经元兴奋性；生长缓慢的低级别胶质瘤可长期以癫痫为首发症状。 |
| 头痛 | 肿瘤和瘤周水肿增加颅内容积，牵拉痛敏感结构并引起颅内压升高。 |
| 呕吐 | 颅内压增高刺激延髓呕吐中枢，典型时可呈喷射性。 |
| 视盘水肿 | 颅内压升高经视神经蛛网膜下腔传导，影响视神经轴浆运输和静脉回流。 |
| 偏瘫、感觉障碍、失语等局灶症状 | 肿瘤浸润、压迫或破坏运动、感觉、语言等功能区及相应白质纤维束。 |
| 认知或精神改变 | 额叶、颞叶或边缘系统受累，或弥漫性脑功能受到肿瘤及水肿影响。 |
| 症状进行性加重 | 肿瘤持续浸润性生长，且高级别肿瘤可快速增殖、坏死、出血并加重水肿。 |

## 4. CT 表现及机制

### CT 平扫

- **部位和形态：**病变主要位于白质，常累及灰质，可呈斑片状、实性、囊性或囊实性肿块。  
  **机制：**胶质细胞广泛存在于脑实质，肿瘤常沿白质纤维束浸润，而不是仅向周围膨胀。

- **密度：**多呈等、低或混杂密度。低级别病变常以低密度为主，高级别病变更易呈混杂密度。  
  **机制：**肿瘤含水量增加时 X 线衰减降低；高级别肿瘤同时含活肿瘤、坏死、囊变和出血，因此密度不均。

- **边界：**常不清楚、形态不规则。  
  **机制：**肿瘤细胞沿神经纤维和细胞间隙弥漫浸润，病变与正常脑组织之间缺乏真正包膜。

- **瘤周水肿：**低级别病变常较轻，高级别病变常明显。  
  **机制：**高级别肿瘤新生血管幼稚，血脑屏障破坏明显，血浆液体渗入细胞外间隙形成血管源性水肿。

- **占位效应：**肿瘤较大或级别较高时明显，可见脑室受压、中线移位。  
  **机制：**肿瘤体积、瘤周水肿和静脉回流障碍共同增加局部容积。

- **坏死和囊变：**表现为肿瘤内低密度区，高级别更常见。  
  **机制：**肿瘤生长速度超过血供能力，引起缺血坏死和液化；囊液含水量高，CT 值低。

- **出血：**表现为病灶内高密度，高级别更常见。  
  **机制：**肿瘤新生血管结构异常、壁薄且易破裂。

- **钙化：**表现为高密度，少突胶质瘤相对多见。  
  **机制：**钙盐沉积对 X 线的衰减明显高于脑组织。

### CT 增强

- **低级别肿瘤：**多数不强化或仅轻度强化。  
  **机制：**肿瘤血管相对成熟，血脑屏障保存较好，对比剂外漏有限。需要注意：不强化不等于没有肿瘤，也不绝对等于低级别。

- **高级别肿瘤：**多呈明显斑片状、结节状、不规则环状或花环状强化。  
  **机制：**活肿瘤区新生血管丰富且血脑屏障破坏；坏死区无血供而不强化，因此形成不均匀或环形强化。

- **囊壁厚薄不均并见壁结节：**提示高级别囊变胶质瘤。  
  **机制：**周边活肿瘤组织分布不均，局部肿瘤增殖形成壁结节；不同于成熟脑脓肿相对光滑、均匀的脓肿壁。

- **跨胼胝体蝶形强化：**可见于高级别胶质瘤，尤其胶质母细胞瘤。  
  **机制：**肿瘤沿胼胝体白质纤维束向对侧半球浸润。

## 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 肿瘤实质多呈稍低或混杂信号；囊变、坏死呈低信号；亚急性出血可呈高信号。 | 肿瘤和坏死区含水增多使 T1延长；亚急性出血中的高铁血红蛋白具有 T1缩短效应。 |
| T2WI | 多呈均匀或不均匀高信号，高级别常因坏死、出血形成混杂信号。 | 肿瘤含水量和细胞外水增加使 T2延长；坏死液化呈高信号，血液产物可造成复杂信号。 |
| FLAIR | 肿瘤实质和瘤周水肿多呈高信号；IDH 突变型星形细胞瘤可出现 T2-FLAIR 失配征。 | FLAIR 抑制自由脑脊液信号，但肿瘤内结合状态不同的水和水肿仍呈高信号；大量微囊样改变、细胞密度较低可使部分区域在 FLAIR 上相对低信号。 |
| DWI/ADC | 高级别肿瘤的实性高细胞区常 DWI 较高、ADC 较低；肿瘤坏死区通常不呈明显弥散受限。 | 细胞密度越高，细胞外间隙越窄，水分子弥散越受限；坏死液体黏稠度通常低于脓液，水分子运动相对自由。 |
| 增强 T1WI | 低级别多不强化或轻度强化；高级别多明显、不均匀、结节状或花环状强化。 | 强化取决于血供、血管通透性和血脑屏障破坏，而不是仅由肿瘤大小决定。 |
| SWI/T2* | 出血、含铁血黄素或钙化区可见低信号和磁敏感效应。 | 血液降解产物、铁和钙改变局部磁场均匀性，导致快速去相位。钙化与出血必要时结合 CT 判断。 |
| DTI | 白质纤维束可受压移位、浸润或破坏中断，高级别更易破坏。 | 胶质瘤沿白质纤维束生长；低级别可主要推移，高级别浸润破坏更明显。 |
| PWI | 活性肿瘤区 rCBV、rCBF 增高；高级别通常更明显。 | 新生血管增加使单位组织内血容量和血流量增加。 |
| DCE | Ktrans 增高，高级别通常更明显。 | 新生血管通透性增加、血脑屏障破坏，使对比剂更易进入血管外细胞外间隙。 |
| MRS | Cho 升高，NAA 和 Cr 降低，NAA/Cho 倒置、Cho/Cr 升高；可见 Lip、Lac 峰。 | Cho 反映细胞膜合成和更新；NAA 反映神经元完整性；Lip、Lac 分别提示坏死和无氧代谢。 |

## 6. 超声表现及机制

常规经颅超声受成人颅骨阻挡，不是成人弥漫性胶质瘤的常规诊断方法。婴幼儿可经未闭合囟门检查，术中超声也可用于病变定位。

- 肿瘤可呈低回声、高回声或混合回声，取决于细胞密度、坏死、出血和钙化等成分。
- 坏死、囊变区回声较低或近无回声，因内部反射界面少；液性成分可伴后方回声增强。
- 出血或钙化可形成强回声；明显钙化可伴后方声影，因声波被强烈反射和衰减。
- 彩色多普勒可显示肿瘤血流，高级别肿瘤通常血管较丰富，但经颅检查受声窗限制。

**临床意义：**胶质瘤定性和分级主要依赖 MRI，多参数 MRI 的价值远高于常规超声。

## 7. 关键征象机制

### 7.1 不规则花环状强化

- **征象：**肿瘤周边呈厚薄不均、结节状的不规则环形或花环状强化，中心不强化。
- **机制：**中心为缺血坏死，无有效血供；周边为血供丰富、屏障破坏的活肿瘤。活肿瘤分布不均，因此强化环不规则并可见壁结节。
- **提示意义：**强烈提示高级别胶质瘤，但脑转移瘤、脓肿等也可环形强化，不能单凭此征确诊。

### 7.2 跨胼胝体“蝴蝶征”

- **征象：**病变跨越胼胝体累及双侧大脑半球，形成蝶形外观。
- **机制：**胼胝体由横行白质纤维构成，为胶质瘤细胞向对侧浸润提供连续通道。
- **提示意义：**常见于胶质母细胞瘤，也可见于原发性中枢神经系统淋巴瘤，应结合坏死、强化和 DWI 判断。

### 7.3 T2-FLAIR 失配征

- **征象：**病灶在 T2WI 上几乎均匀高信号，但在 FLAIR 上大部分相对低信号，仅周缘呈高信号。
- **机制：**肿瘤内部大量微囊样改变且细胞密度较低，使其水分状态更接近可被 FLAIR 部分抑制的自由水。
- **提示意义：**对 IDH 突变型星形细胞瘤有提示价值，但敏感性有限，最终分型仍依赖分子病理。

### 7.4 ADC 降低

- **征象：**肿瘤实性部分 DWI 高信号、ADC 低值。
- **机制：**高细胞密度使细胞外间隙缩小，限制水分子弥散。
- **提示意义：**ADC 越低通常提示恶性度越高，但需避开坏死、出血区域并结合其他序列。

### 7.5 rCBV、Ktrans 增高

- **征象：**PWI 显示 rCBV/rCBF 增高，DCE 显示 Ktrans 增加。
- **机制：**rCBV 主要反映肿瘤新生血管数量；Ktrans 主要反映血管通透性和血脑屏障破坏。
- **提示意义：**有助于寻找活性最高的肿瘤区、评估级别并指导活检，但炎症和治疗后改变也可能升高。

### 7.6 Cho 升高、NAA 降低

- **征象：**MRS 上 Cho 峰升高、NAA 峰降低，Cho/Cr 升高，严重坏死时出现 Lip、Lac 峰。
- **机制：**肿瘤细胞快速增殖导致细胞膜更新活跃，Cho 增高；神经元被浸润和破坏，NAA 降低；坏死和缺氧产生 Lip、Lac。
- **提示意义：**支持肿瘤及较高活性，但不能脱离常规 MRI 单独诊断。

## 8. 分型/分级对比

### 低级别与高级别胶质瘤

| 比较项目 | 低级别胶质瘤 | 高级别胶质瘤 |
| --- | --- | --- |
| 病理差异 | 生长较慢，细胞异型和增殖程度较低，血管相对成熟，坏死少。 | 生长迅速，细胞密度和异型性高，新生血管丰富，可有微血管增生、坏死和出血。 |
| 临床差异 | 病程较长，癫痫较常见，局灶症状可缓慢进展。 | 病程较短、进展快，颅内压增高和神经功能缺损较明显。 |
| CT | 多为较均匀低密度，坏死、出血少，水肿及占位效应相对轻。 | 多为混杂密度，常见坏死、囊变或出血，水肿及占位效应明显。 |
| MRI | T1低、T2高信号相对均匀；可见 T2-FLAIR 失配征。 | 信号不均，常有坏死、出血；可跨胼胝体生长。 |
| DWI/ADC | 弥散受限不明显，ADC 相对较高。 | 实性高细胞区弥散受限明显，ADC 较低。 |
| 增强 | 多不强化或轻度强化。 | 多明显不均匀、结节状、环形或花环状强化。 |
| PWI/DCE | rCBV、rCBF 和 Ktrans 相对较低。 | rCBV、rCBF 和 Ktrans 常明显增高。 |
| MRS | Cho 可升高、NAA 降低，但变化相对较轻。 | Cho/Cr 升高更明显，NAA 明显降低，坏死时可见 Lip、Lac 峰。 |
| 鉴别要点 | “较均匀、坏死少、水肿轻、强化少、ADC 较高”。 | “混杂、坏死出血、水肿重、花环强化、ADC 低、灌注高”。 |

> **重要限制：**影像分级是概率判断，不是病理分级的替代。部分低级别肿瘤可以强化，部分高级别肿瘤也可能强化不明显。

### 星形细胞瘤、少突胶质瘤与胶质母细胞瘤

| 类型 | 核心病理/分子特点 | 常见影像倾向 | 关键提示 |
| --- | --- | --- | --- |
| 星形细胞瘤，IDH 突变型 | IDH 突变，弥漫浸润生长 | T2/FLAIR 高信号，可出现 T2-FLAIR 失配征；强化随级别升高而增加 | 失配征有提示价值，确诊依赖分子病理 |
| 少突胶质瘤 | IDH 突变并有 1p/19q 联合缺失 | 成人幕上、额叶常见，钙化相对多，常有癫痫 | CT 对钙化最敏感；“额叶肿瘤伴条带、点片或团块状钙化”具有提示性 |
| 胶质母细胞瘤，IDH 野生型 | WHO 4级，高度恶性，微血管增生和坏死明显 | 混杂信号、显著水肿、不规则花环状强化，可跨胼胝体 | “坏死 + 花环强化 + 蝴蝶征 + ADC低 + rCBV高” |

## 9. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 脑转移瘤 | 可有坏死、环形强化和明显水肿。 | 常多发，位于灰白质交界区；水肿范围可明显大于瘤体，病灶相对边界清楚。 | 血行转移瘤易停留在血管口径变化的灰白质交界处，多呈膨胀性生长；胶质瘤更常沿白质纤维束浸润。 |
| 脑脓肿 | 可呈环形强化并伴水肿和占位效应。 | 成熟脓肿壁常较薄、光滑、均匀；脓腔 DWI 明显高、ADC 低，而胶质瘤坏死中心通常不受限。 | 脓液黏稠、含大量炎细胞和蛋白，强烈限制水分子运动；肿瘤坏死液相对稀薄。 |
| 原发性中枢神经系统淋巴瘤 | 可位于深部或跨胼胝体，DWI 受限。 | 免疫正常者多呈实性、较均匀明显强化，坏死和出血相对少，ADC 常明显降低。 | 淋巴瘤细胞排列极其致密且血管外间隙小，因此弥散受限明显；胶质母细胞瘤血管和坏死分布更不均。 |
| 脑梗死 | 亚急性梗死可有肿胀、T2高信号和强化。 | 病变通常符合动脉供血区；急性期 DWI 受限早于强化，随后按时间演变。 | 梗死由血管闭塞导致细胞毒性水肿，具有血管分布和时间规律；胶质瘤跨越血管供血边界并持续生长。 |
| 脑炎 | 可呈斑片状 T2/FLAIR 高信号、边界不清和轻度强化。 | 常有发热、脑脊液异常及特定分布，如单纯疱疹病毒性脑炎偏好颞叶、岛叶。 | 炎症以水肿和细胞浸润为主，结合感染证据和分布判断；肿瘤更常出现持续占位、代谢及灌注异常。 |
| 瘤样脱髓鞘病变 | 可形成较大肿块和环形强化。 | 常见“开环征”，强化环开口朝向皮质；灌注通常低于高级别胶质瘤，随访或激素治疗后可缩小。 | 活跃脱髓鞘主要发生在病灶推进缘，血管增生不如恶性肿瘤，因此强化和灌注模式不同。 |
| 脑膜瘤 | 表浅胶质瘤可与脑膜瘤相邻。 | 脑膜瘤为脑外肿瘤，常有宽基底、脑脊液间隙、脑膜尾征和均匀显著强化。 | 脑膜瘤起源于蛛网膜粒帽细胞并由硬脑膜供血；胶质瘤起源于脑内并浸润白质。 |

## 10. 考试答题版

弥漫性胶质瘤是神经上皮组织起源的原发性中枢神经系统肿瘤，成人多发生于大脑，主要位于白质并常累及灰质，具有弥漫浸润性生长、边界不清、难以完整切除及易复发等特点。临床主要表现为局灶性神经功能障碍、癫痫及肿瘤占位效应所致的颅内压增高症状。

CT 平扫表现为脑内不规则等、低或混杂密度病灶，边界不清，可伴瘤周水肿和占位效应；高级别肿瘤易发生坏死、囊变和出血，少突胶质瘤较易钙化。增强扫描低级别肿瘤多不强化或轻度强化，高级别肿瘤多呈明显斑片状、结节状、不规则环形或花环状强化，囊壁常厚薄不均并可见壁结节。

MRI 上病变多呈 T1WI 稍低或混杂信号、T2WI/FLAIR 高信号；坏死囊变呈长 T1、长 T2信号，亚急性出血可呈 T1高信号。高级别肿瘤实性部分细胞密度高，DWI 信号较高、ADC 值较低；PWI 可见 rCBV、rCBF 增高，DCE 可见 Ktrans 增加。MRS 表现为 Cho 升高、NAA 和 Cr 降低，可出现 Lip、Lac 峰。胶质母细胞瘤可见花环状强化，并可沿胼胝体向对侧生长形成“蝴蝶征”。

鉴别诊断应结合病变部位、数目、强化环形态及多参数 MRI。脑脓肿脓腔常明显弥散受限；转移瘤常多发并位于灰白质交界区；淋巴瘤多呈实性均匀强化和明显弥散受限；脑梗死通常符合血管供血区。

## 11. 一句话记忆

**低级别：慢、较均匀、水肿轻、强化少、ADC较高；高级别：浸润快、坏死出血、水肿重、ADC低、灌注高、花环强化，可跨胼胝体成蝴蝶。**
