# 听神经瘤

> 教材依据：《医学影像学》第9版第二章“听神经瘤”；课程 PPT 第342-354张。规范名称为前庭神经鞘瘤。

## 1. 疾病本质

听神经瘤多起源于第Ⅷ脑神经前庭支的 Schwann 细胞，是桥小脑角区最常见的良性脑外肿瘤。早期位于内听道，增大后沿内听道向桥小脑角池生长，常有完整包膜，可发生囊变、坏死和出血。

## 2. 底层病理逻辑

| 病理因素 | 影像推导 |
| --- | --- |
| 沿前庭神经生长 | 肿瘤以内听道为起点，形成内听道内和桥小脑角池内两部分。 |
| 慢性扩张内听道 | 内听道呈漏斗状扩大，骨缘相对光滑。 |
| 实性神经鞘组织 | CT多为等密度，MRI多T1低/等、T2高，增强明显。 |
| Antoni A/B区比例不同及退变 | 大肿瘤内部信号和强化可不均，易囊变。 |
| 脑外膨胀性生长 | 推压脑干、小脑和第四脑室，晚期可形成梗阻性脑积水。 |

## 3. 临床表现及机制

| 临床表现 | 产生机制 |
| --- | --- |
| 单侧进行性听力下降、耳鸣 | 前庭蜗神经耳蜗成分受压。 |
| 眩晕、平衡障碍 | 前庭神经受累。 |
| 面部麻木、角膜反射减弱 | 较大肿瘤压迫三叉神经。 |
| 面瘫 | 面神经受压。 |
| 共济失调 | 小脑及小脑脚受压。 |
| 颅内压增高 | 第四脑室受压导致梗阻性脑积水。 |

## 4. CT 表现及机制

### CT 平扫

- 桥小脑角区等、低或混杂密度肿块；囊变低密度，出血高密度。
- 内听道扩大、骨质吸收，常呈漏斗状，源于缓慢膨胀压迫而非侵袭性破坏。
- 肿瘤较大时压迫脑干、小脑及第四脑室。

### CT 增强

- 小肿瘤多均匀明显强化，大肿瘤因囊变坏死可不均匀强化。
- 内听道内小肿瘤需薄层增强检查，但MRI更敏感。

## 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 等或稍低信号 | 肿瘤含水量略高于脑实质。 |
| T2WI | 高信号，可不均匀 | 黏液样基质、囊变和含水增加。 |
| FLAIR | 多为高信号 | 非单纯脑脊液组织不被完全抑制。 |
| DWI/ADC | 多不明显受限，ADC相对较高 | 神经鞘瘤细胞排列不如淋巴瘤致密，且可有疏松Antoni B区。 |
| 增强T1WI | 实性部分明显强化 | 血供和血管通透性较高；囊变区不强化。 |
| 高分辨T2/CISS | 显示内听道内小结节及脑神经关系 | 脑脊液高信号衬托低信号神经和肿瘤。 |

## 6. 超声表现及机制

常规超声不用于颅内听神经瘤诊断。

## 7. 关键征象机制

- **内听道扩大：** 肿瘤在骨性管道内缓慢生长，产生光滑压迫性骨吸收。
- **“冰淇淋征”：** 内听道内成分似圆锥，桥小脑角池成分似冰淇淋球，反映沿神经生长路径。
- **明显强化：** 实性 Schwann 细胞肿瘤血供较丰富。
- **小肿瘤MRI优于CT：** MRI软组织对比高且高分辨T2可直接显示脑神经与脑脊液界面。

## 8. 分型/分期对比

| 项目 | 内听道内小肿瘤 | 桥小脑角区大肿瘤 |
| --- | --- | --- |
| 症状 | 听力下降、耳鸣为主 | 多脑神经、小脑和颅压症状 |
| CT | 可漏诊 | 肿块和内听道扩大明显 |
| MRI | 内听道内强化小结节 | 囊变、不均匀强化和明显占位 |
| 骨改变 | 轻度扩大 | 漏斗状明显扩大 |

## 9. 鉴别诊断

| 疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 桥小脑角脑膜瘤 | 脑外肿块、明显强化 | 脑膜瘤宽基贴岩骨、可钙化和骨增生；听神经瘤以内听道为中心并扩大内听道 | 起源为脑膜与神经鞘。 |
| 表皮样囊肿 | 位于桥小脑角、包绕结构 | 类脑脊液信号但DWI显著受限，通常不强化 | 角蛋白碎屑限制弥散且无血供。 |
| 三叉神经鞘瘤 | 神经鞘瘤信号相似 | 沿三叉神经和Meckel腔生长，不以内听道为中心 | 起源脑神经不同。 |

## 10. 考试答题版

听神经瘤多起源于第Ⅷ脑神经前庭支，是桥小脑角区最常见的良性脑外肿瘤。CT表现为以内听道为中心的桥小脑角区等或低密度肿块，内听道呈漏斗状扩大，增强后明显强化，大肿瘤可因囊变坏死而强化不均。MRI上肿瘤T1WI多呈等或稍低信号，T2WI呈高信号，增强后实性部分明显强化；高分辨T2和薄层增强MRI可检出内听道内小肿瘤。较大肿瘤可压迫脑干、小脑和第四脑室并引起脑积水。

## 11. 一句话记忆

**桥小脑角肿块以内听道为根，内听道漏斗状扩大，增强明显就是听神经瘤。**
