# 静脉血栓栓塞与下腔静脉滤器

> **口径说明：** 本页重点不是替代临床抗凝指南，而是从影像和介入角度理解 VTE、DVT、PE、下腔静脉滤器和导管接触溶栓。考试答题以“为什么需要介入、介入解决什么风险、有什么并发症”为主线。

## 1. 总论：VTE 的病理风险链

静脉血栓栓塞症（VTE）包括深静脉血栓形成（DVT）和肺栓塞（PE）。DVT 多发生于下肢深静脉，血栓脱落后经髂静脉、下腔静脉、右心进入肺动脉，即形成 PE。

### 1.1 底层病理逻辑

| 病理因素 | 影像和介入意义 |
| --- | --- |
| 血流淤滞 | 长期卧床、术后、肿瘤或妊娠等状态使静脉血流变慢，血栓更易形成。 |
| 内皮损伤 | 手术、创伤、导管和炎症损伤血管内皮，启动凝血。 |
| 高凝状态 | 肿瘤、妊娠、遗传性血栓倾向等使血液更易凝固。 |
| 血栓脱落 | 近端 DVT 尤其髂股静脉血栓脱落风险高，可导致 PE。 |
| 抗凝受限 | 抗凝是基础治疗，但活动性出血或近期颅内出血等情况使抗凝不能安全使用。 |

**总推导：** DVT 的危险不只在下肢肿胀，而在“血栓脱落到肺动脉”。下腔静脉滤器的逻辑是机械拦截血栓，导管溶栓的逻辑是局部清除血栓负荷。

### 1.2 影像评估

| 检查 | 主要价值 |
| --- | --- |
| 下肢静脉超声 | DVT 首选筛查，核心征象为静脉不能压闭、腔内血栓回声、CDFI 充盈缺损或无血流。 |
| CTV/MRV | 显示髂静脉、下腔静脉或盆腔深部血栓，弥补超声对深部血管显示不足。 |
| CTPA | 诊断 PE 的常用影像方法，显示肺动脉内充盈缺损。 |
| 下腔静脉造影 | 介入术中明确下腔静脉通畅性、直径、肾静脉开口和滤器释放位置。 |
| DSA/导管造影 | 用于导管溶栓、取栓或滤器置入过程中的实时定位。 |

## 下腔静脉滤器

### 1. 治疗原理

下腔静脉滤器经皮穿刺置入下腔静脉，通过金属支架样结构拦截来自下肢或盆腔静脉系统的脱落血栓，从而降低致命性肺栓塞风险。

滤器不能治疗已经存在的 DVT，也不能防止新血栓形成。它解决的是“血栓向肺动脉迁移”的机械风险。因此只要抗凝可以安全进行，抗凝仍是 VTE 治疗基础。

### 2. 滤器类型

| 类型 | 特点 | 考试意义 |
| --- | --- | --- |
| 永久型滤器 | 设计为长期留置。 | 适用于长期存在 PE 高风险且无法抗凝者。 |
| 可回收滤器 | PE 风险解除后应评估取出。 | 目前临床常用，重点是“能取就评估取”。 |
| 两用型滤器 | 可作为永久或可回收使用。 | 取决于患者后续风险和器械条件。 |

### 3. 适应证及机制

| 适应证 | 为什么需要滤器 |
| --- | --- |
| 急性近端 DVT 或 PE 且存在抗凝绝对禁忌 | 不能用抗凝降低血栓进展和脱落风险，只能用机械拦截降低 PE 风险。 |
| 充分抗凝治疗下仍发生 PE | 说明单纯抗凝未能控制栓塞事件，需考虑机械保护。 |
| 抗凝出现严重并发症必须停止 | 停止抗凝后血栓脱落风险上升。 |
| 下肢 DVT 拟行导管溶栓或取栓且 PE 风险高 | 操作可能扰动血栓，滤器可降低术中血栓脱落进入肺动脉的风险。 |

### 4. 禁忌证和术前评估

| 项目 | 重点 |
| --- | --- |
| 下腔静脉完全闭塞 | 无法安全释放滤器或滤器不能发挥拦截作用。 |
| 下腔静脉直径异常 | 直径超过滤器适用范围可导致移位，过小可导致阻塞。 |
| 穿刺入路闭塞或感染 | 增加置入失败和感染风险。 |
| 严重凝血功能障碍 | 增加穿刺出血和术后并发症。 |
| 肾静脉开口定位 | 滤器通常置于肾静脉开口以下，避免影响肾静脉回流。 |

### 5. 操作逻辑

1. 选择入路，常用股静脉或颈内静脉。
2. 下腔静脉造影，确认通畅性、直径、血栓范围和肾静脉位置。
3. 将输送鞘送至预定位置。
4. 释放滤器，确认位置、展开形态和有无倾斜。
5. 记录滤器类型和回收计划。

### 6. 并发症及机制

| 并发症 | 机制和意义 |
| --- | --- |
| 滤器倾斜或移位 | 下腔静脉直径不合适、释放位置不佳或血流冲击导致，影响拦截效果。 |
| 滤器内血栓形成 | 滤器拦截血栓或本身成为血栓附着点，可造成下腔静脉阻塞。 |
| 下腔静脉闭塞 | 血栓负荷增加或慢性内膜增生导致回流障碍。 |
| 滤器穿透静脉壁 | 滤器支脚长期压迫或穿出静脉壁，可累及邻近结构。 |
| 回收困难 | 留置时间过长、内皮化、倾斜或血栓包埋使回收难度增加。 |

### 7. 随访和回收

可回收滤器不是“放进去就结束”。当 PE 风险降低、抗凝禁忌解除或血栓负荷稳定后，应尽早评估是否回收。具体时间取决于滤器型号、说明书、血栓负荷、患者出血风险和后续抗凝计划。

随访重点包括滤器位置、倾斜、断裂、穿透、滤器内血栓和下腔静脉通畅性。影像方法可根据情况选择腹部平片、超声、CTV 或腔静脉造影。

### 【考试答题版】

下腔静脉滤器用于特定 VTE 患者预防下肢或盆腔静脉血栓脱落导致肺栓塞。其原理是经皮将滤器置入下腔静脉，机械拦截脱落血栓。滤器不能治疗 DVT，也不能替代抗凝；抗凝可行时仍以抗凝为基础。

主要适应证包括急性近端 DVT 或 PE 合并抗凝绝对禁忌、抗凝失败仍发生 PE、抗凝严重并发症需停止，以及高风险 DVT 行导管溶栓或取栓前的机械保护。术前需评估下腔静脉通畅性、直径、血栓范围和肾静脉开口位置。

可回收滤器在肺栓塞风险解除后应评估取出。并发症包括滤器倾斜、移位、滤器内血栓、下腔静脉闭塞、静脉壁穿透和回收困难。

### 【一句话记忆】

**下腔静脉滤器 = 抗凝不能用或抗凝失败时，放在下腔静脉里拦截下肢血栓，防止它跑到肺动脉。**

### 【易错点】

- 滤器不溶栓、不治疗 DVT，只是降低 PE 风险。
- 可回收滤器需要主动随访和评估回收，不能默认永久留置。
- 滤器通常置于肾静脉开口以下，避免影响肾静脉回流。
- 抗凝不是滤器的对立面；抗凝禁忌解除后仍需重新评估抗凝和滤器回收。

## 导管接触溶栓术

### 1. 治疗原理

导管接触溶栓术是将多侧孔溶栓导管置入血栓内部，持续灌注尿激酶、rt-PA 等溶栓药物，使药物直接作用于血栓。与全身给药相比，局部药物浓度更高，理论上可提高溶栓效率并减少全身出血风险。

它的核心目标不是“所有 DVT 都溶掉”，而是在合适患者中快速降低血栓负荷，改善静脉回流或肺动脉灌注，减少严重后果。

### 2. 主要应用

| 应用场景 | 目的 |
| --- | --- |
| 急性髂股静脉 DVT | 清除近端大血栓负荷，改善下肢肿胀，降低血栓后综合征风险。 |
| 急性大面积或高危肺栓塞 | 改善肺动脉阻塞和右心负荷，适用于选择性高风险病例。 |
| 急性动脉血栓形成 | 恢复远端供血，避免组织缺血坏死。 |
| 血管内支架或移植物血栓 | 恢复管腔通畅，必要时联合取栓、球囊扩张或支架处理。 |

### 3. 影像和操作逻辑

1. 术前用超声、CTV、CTPA 或造影确认血栓部位、范围和时间窗。
2. 穿刺进入目标静脉或动脉，导丝通过血栓段。
3. 将多侧孔溶栓导管放置在血栓内。
4. 持续泵入溶栓药物，并监测凝血功能、出血和症状变化。
5. 复查造影评估残余血栓和管腔通畅情况。
6. 必要时联合机械取栓、球囊扩张、支架或滤器保护。

### 4. 禁忌证

| 禁忌证 | 原因 |
| --- | --- |
| 活动性出血或严重出血倾向 | 溶栓会显著增加出血风险。 |
| 近期颅内出血、颅脑手术或严重脑卒中 | 颅内出血后果严重。 |
| 近期大手术、严重创伤或不可压迫部位穿刺 | 局部止血困难。 |
| 未控制的严重高血压 | 增加脑出血风险。 |
| 严重肝肾功能不全或凝血异常 | 药物代谢和凝血调控异常，出血风险高。 |

### 5. 并发症和随访

| 并发症 | 机制 |
| --- | --- |
| 出血 | 溶栓药物激活纤溶系统，既溶血栓也削弱正常止血。 |
| 颅内出血 | 最严重并发症，虽少见但致死致残风险高。 |
| 血栓脱落和 PE | 操作扰动血栓，可导致血栓迁移，高风险时需考虑滤器保护。 |
| 穿刺点血肿 | 穿刺损伤和凝血功能受影响共同导致。 |
| 再狭窄或再血栓 | 原有狭窄、压迫或高凝状态未解除时可复发。 |

### 【考试答题版】

导管接触溶栓术是将多侧孔导管置入血栓内部，持续灌注溶栓药物，使局部药物浓度增高，从而提高溶栓效率并相对减少全身用药量。常用于选择性急性髂股静脉 DVT、高危肺栓塞、急性动脉血栓和血管支架血栓。

术前需影像明确血栓部位、范围、病程和出血风险。操作后需复查造影判断管腔通畅，并监测凝血功能、穿刺点和全身出血。最大风险是出血，尤其颅内出血；操作扰动血栓也可能导致栓子脱落。

### 【一句话记忆】

**导管溶栓 = 把多孔导管放进血栓里局部滴药，目标是快速减轻血栓负荷，但最大风险是出血。**

### 【易错点】

- 导管溶栓不是所有 DVT 的常规治疗，必须选择合适患者并评估出血风险。
- 局部溶栓仍可造成全身出血，不能理解为“没有出血风险”。
- 溶栓后若残余狭窄或压迫未处理，仍可能再血栓。
- 高风险下肢 DVT 操作前是否放置滤器，应结合血栓位置、负荷和 PE 风险评估。
