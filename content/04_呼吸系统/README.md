## 大纲覆盖核对

- **肺部感染：** 大叶性肺炎、小叶性肺炎、肺脓肿、肺结核。
- **气道阻塞：** 阻塞性肺气肿、阻塞性肺不张、阻塞性肺炎、支气管扩张。
- **肺肿瘤：** 中央型肺癌、周围型肺癌、肺结节良恶性鉴别。
- **胸膜疾病：** 胸腔积液、气胸、液气胸。
- **纵隔疾病：** 前、中、后纵隔常见肿瘤及定位诊断。

## 高频征象速查

| 征象 | 首先考虑 | 底层含义 |
|---|---|---|
| 空气支气管征 | 肺炎、肺泡性病变 | 肺泡被液体或细胞填充，但支气管仍含气 |
| 树芽征 | 支气管播散性结核、小气道感染 | 细支气管及其分支被炎性物或干酪物填充 |
| 空洞伴气液平面 | 肺脓肿 | 坏死液化与支气管相通，空气进入脓腔 |
| 印戒征 | 支气管扩张 | 扩张支气管直径大于伴行动脉 |
| 反 S 征 | 右上叶中央型肺癌 | 肺门肿块与右上叶不张边缘共同形成反 S 形 |
| 分叶、毛刺、胸膜凹陷 | 周围型肺癌 | 肿瘤不均衡生长并牵拉间质、胸膜 |
| 脏层胸膜线 | 气胸 | 肺组织回缩，胸膜线外侧无肺纹理 |
| 脊柱旁肿块伴椎间孔扩大 | 神经源性肿瘤 | 后纵隔肿瘤沿神经根向椎管内生长 |
