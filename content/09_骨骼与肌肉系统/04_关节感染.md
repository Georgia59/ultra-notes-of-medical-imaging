## 化脓性关节炎

### 1. 疾病本质

化脓性关节炎是化脓性细菌进入关节腔后引起的急性破坏性关节炎，常见病原体为金黄色葡萄球菌。感染可来自血源播散、邻近骨髓炎蔓延、穿刺或外伤直接进入关节。

好发于儿童和青少年，常累及髋、膝等承重大关节。临床常急性起病，发热、关节红肿热痛、活动受限明显，若不及时治疗，可在短时间内破坏关节软骨。

### 2. 底层病理逻辑

| 病理过程 | 对影像的影响 |
| --- | --- |
| 细菌进入关节腔 | 滑膜充血、水肿和渗出，早期出现关节肿胀和关节积液。 |
| 脓液和炎性酶增多 | 关节软骨被蛋白水解酶快速破坏，承重面软骨受压和摩擦最多，破坏更早。 |
| 软骨变薄和消失 | 关节间隙由增宽转为迅速变窄。 |
| 软骨下骨外露和受侵 | 出现承重面软骨下骨质破坏、骨质疏松和边缘不规则。 |
| 肉芽组织和骨修复 | 晚期两侧骨端可由骨组织连接，形成骨性强直。 |

**总推导：** 化脓性关节炎的关键词是“急、脓、酶”。急性脓液和炎性酶快速破坏软骨，所以关节间隙变化快，晚期可骨性强直。

### 3. 影像学表现及机制

| 检查/阶段 | 表现 | 机制 |
| --- | --- | --- |
| X线早期 | 关节周围软组织肿胀，关节间隙可增宽。 | 滑膜炎和关节积液先于骨质破坏出现。 |
| X线进展期 | 关节间隙迅速变窄，承重面软骨下骨质破坏。 | 蛋白水解酶和脓液快速破坏软骨，承重面压力最大。 |
| X线晚期 | 关节间隙消失，骨小梁贯通，形成骨性强直。 | 软骨完全破坏后两侧骨端通过修复性骨组织连接。 |
| 超声 | 关节积液、滑膜增厚，可用于穿刺抽液。 | 液体和增厚滑膜形成无回声或低回声区。 |
| MRI | 关节积液、滑膜强化、软骨破坏、骨髓水肿和邻近软组织脓肿。 | 炎症渗出、血管通透性增加和软骨下骨受侵。 |
| 增强 MRI | 滑膜明显强化，脓肿中心不强化、壁强化。 | 充血滑膜和肉芽组织有血供，脓液无血供。 |

### 4. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 关节结核 | 关节肿胀、积液、骨质破坏和关节间隙变窄。 | 结核起病慢，边缘性骨质破坏，骨质疏松明显，关节间隙缓慢变窄，晚期多纤维性强直。 | 结核为慢性肉芽肿性炎症，破坏从滑膜反折和关节边缘开始。 |
| 类风湿关节炎 | 滑膜炎、关节间隙变窄和骨侵蚀。 | 类风湿多为对称多关节、小关节受累，病程慢，无急性脓毒表现。 | 自身免疫性滑膜 pannus 侵蚀软骨和骨，而非细菌性脓液快速破坏。 |
| 痛风性关节炎 | 急性红肿热痛，可有骨质破坏。 | 痛风常有高尿酸、痛风石，骨侵蚀呈穿凿样伴悬垂边，关节间隙早期相对保留。 | 尿酸盐结晶沉积引起炎症和骨侵蚀，不以脓液破坏软骨为主。 |
| 创伤性关节积血 | 急性肿痛和关节积液。 | 有明确外伤史，缺乏感染指标升高和滑膜脓性改变。 | 出血导致积液，不产生细菌性软骨破坏。 |

### 【考试答题版】

化脓性关节炎是化脓性细菌感染关节腔引起的急性破坏性关节炎，常累及儿童和青少年的髋、膝等大关节。病理过程为滑膜充血水肿、关节积液、脓液和炎性酶快速破坏关节软骨，软骨下骨继发受侵。

影像上早期 X线可见关节周围软组织肿胀和关节间隙增宽，进展期关节间隙迅速变窄，并出现承重面软骨下骨质破坏，晚期可形成骨性强直。MRI 可显示关节积液、滑膜强化、软骨破坏、骨髓水肿和软组织脓肿，超声可用于发现积液和引导穿刺。

### 【一句话记忆】

**化脓性关节炎 = 急性高热 + 大关节 + 关节积液后迅速间隙变窄 + 承重面破坏 + 晚期骨性强直。**

### 【易错点】

- 早期关节间隙可因积液增宽，不能误认为病变轻。
- 化脓性关节炎的关节间隙变窄进展快；关节结核通常变窄慢。
- 骨性强直是骨小梁贯通连接两侧骨端，不等同于单纯关节活动受限。
- 髋关节化脓性感染可继发股骨头缺血坏死，儿童尤其要警惕。

## 关节结核

### 1. 疾病本质

关节结核是结核分枝杆菌感染滑膜、骨端和关节软骨所致的慢性肉芽肿性关节炎。多继发于肺结核或骨结核，好发于儿童和青年，常为单关节受累，髋、膝、踝等大关节多见。

其核心不是急性脓液破坏，而是结核性肉芽组织从滑膜反折、关节边缘和骨端缓慢侵蚀软骨和骨质。因此关节间隙变窄较慢，晚期多为纤维性强直。

### 2. 底层病理逻辑

| 病理过程 | 对影像的影响 |
| --- | --- |
| 滑膜结核和肉芽组织增生 | 关节肿胀、滑膜增厚、积液，MRI 增强可见滑膜强化。 |
| 肉芽组织从边缘侵犯 | 先出现关节边缘骨质破坏和软骨破坏。 |
| 慢性失用和炎症充血 | 关节周围骨质疏松明显。 |
| 软骨缓慢受侵 | 关节间隙缓慢变窄，不如化脓性关节炎迅速。 |
| 纤维组织修复 | 晚期形成纤维性强直，关节间隙可狭窄但无骨小梁贯通。 |
| 干酪坏死液化 | 可形成冷脓肿和窦道，急性红热反应相对轻。 |

### 3. 影像学表现及机制

| 检查/阶段 | 表现 | 机制 |
| --- | --- | --- |
| X线早期 | 软组织肿胀，关节间隙正常或稍增宽，关节周围骨质疏松。 | 滑膜炎、积液和失用性骨质疏松先出现。 |
| X线进展期 | 关节边缘虫蚀状骨质破坏，关节间隙缓慢变窄。 | 结核肉芽组织从滑膜反折和关节边缘侵蚀。 |
| X线晚期 | 关节间隙明显狭窄，骨质破坏和畸形，可纤维性强直。 | 软骨被慢性破坏，纤维组织替代，但无骨小梁贯通。 |
| CT | 更清楚显示边缘骨质破坏、小死骨、冷脓肿钙化和骨端破坏。 | CT 对骨质和钙化敏感。 |
| MRI | 显示滑膜增厚、积液、骨髓水肿、软骨破坏、冷脓肿和窦道。 | MRI 对滑膜、软骨、骨髓和软组织最敏感。 |
| 增强 MRI | 滑膜和肉芽组织强化，冷脓肿壁强化、中心不强化。 | 肉芽组织有血供，干酪坏死和脓液无血供。 |

### 4. 化脓性关节炎 vs 关节结核

| 项目 | 化脓性关节炎 | 关节结核 |
|---|---|---|
| 起病 | 急，高热，中毒症状 | 缓慢，低热 |
| 软骨/骨破坏起始 | **承重面**软骨破坏明显 | **边缘**骨质破坏明显 |
| 病程 | 进展快 | 进展慢 |
| 骨质疏松 | 不明显 | 明显（失用性） |
| 最终 | **骨性强直** | **纤维性强直** |
| 关节间隙 | 迅速变窄 | 缓慢变窄 |
| 脓肿 | 急性脓肿，红热痛明显 | 冷脓肿，急性炎症表现轻 |

### 5. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 化脓性关节炎 | 关节积液、肿胀、骨质破坏和间隙变窄。 | 化脓性急起，间隙迅速变窄，晚期骨性强直；结核慢性，边缘破坏和骨质疏松明显。 | 化脓性为脓液和炎性酶快速破坏；结核为肉芽组织慢性侵蚀。 |
| 类风湿关节炎 | 滑膜炎、骨质疏松、边缘骨侵蚀。 | 类风湿多为对称多关节，手足小关节常见，血清学可支持；结核多单关节。 | 自身免疫性 pannus 与结核肉芽组织均可侵蚀边缘骨。 |
| 色素沉着绒毛结节性滑膜炎 | 单关节肿胀、滑膜增厚和骨侵蚀。 | MRI 可见含铁血黄素低信号，通常无结核中毒症状和冷脓肿。 | 滑膜增殖伴反复出血，含铁血黄素造成低信号。 |
| 滑膜肿瘤 | 单关节肿胀和软组织肿块。 | 肿瘤呈局灶肿块样生长，缺乏典型冷脓肿、窦道和结核证据。 | 肿瘤细胞增殖不同于感染性肉芽组织。 |

### 【考试答题版】

关节结核是结核分枝杆菌感染滑膜和关节骨端所致的慢性肉芽肿性关节炎，多为儿童和青年单关节受累，髋、膝多见。病理为滑膜结核、肉芽组织增生、干酪样坏死和慢性软骨骨质破坏。

影像上早期表现为关节肿胀、积液和关节周围骨质疏松，进展期出现边缘性虫蚀状骨质破坏，关节间隙缓慢变窄，晚期可形成纤维性强直。MRI 可显示滑膜增厚强化、软骨破坏、骨髓受累、冷脓肿和窦道。

鉴别化脓性关节炎时抓住：化脓性急起、承重面破坏、间隙迅速变窄、晚期骨性强直；结核慢性、边缘破坏、骨质疏松明显、间隙缓慢变窄、晚期纤维性强直。

### 【一句话记忆】

**关节结核 = 慢性单关节 + 边缘破坏 + 骨质疏松 + 间隙慢慢窄 + 纤维性强直。**

### 【易错点】

- 关节结核不是一定先侵犯软骨中央，常从滑膜反折和关节边缘开始。
- 纤维性强直不是骨性强直；前者无骨小梁贯通。
- 关节结核可有冷脓肿和窦道，但急性红热痛不如化脓性明显。
- 类风湿关节炎也可边缘侵蚀，要看是否对称多关节和有无结核证据。
