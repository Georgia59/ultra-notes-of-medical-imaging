# 骨感染

> **资料依据**
>
> - 主教材：《医学影像学》第 9 版，骨骼与肌肉系统骨感染相关内容。
> - 课程重点：急性化脓性骨髓炎、慢性化脓性骨髓炎、骨结核，尤其是死骨、骨包壳、窦道、冷脓肿和脊柱结核。
>
> **口径说明：** 骨感染影像复习不能只背“骨质破坏、骨膜反应、死骨”。考试更常考病程、好发部位、影像出现时间和鉴别诊断。答题时要把“感染类型和病理过程”推到“影像表现”。

## 1. 骨感染总论

骨感染是病原体进入骨髓、骨皮质或骨膜后引起的炎症性骨病。常见类型包括化脓性骨髓炎和骨结核。两者都能造成骨质破坏，但病原体、炎症速度和宿主反应不同，因此影像表现不同。

### 1.1 底层病理逻辑

| 病理因素 | 对影像的影响 |
| --- | --- |
| 骨髓充血、水肿和炎细胞浸润 | 骨髓脂肪被液体和炎性组织替代，MRI 最早出现 T1WI 低信号、T2WI/STIR 高信号；X线早期可无骨质改变。 |
| 脓液形成和骨内压升高 | 髓腔压力升高，静脉回流受阻，炎症可经哈弗管、Volkmann 管向骨皮质和骨膜下扩展。 |
| 骨膜被脓液或炎症掀起 | 形成骨膜反应；儿童骨膜较松，急性化脓性骨髓炎更容易出现骨膜下脓肿和明显骨膜新生骨。 |
| 血管栓塞和骨质缺血 | 受累骨段失去血供形成死骨；死骨不参与代谢，常表现为相对高密度骨片或低信号骨片。 |
| 修复性新骨形成 | 活骨在死骨外形成骨包壳；慢性期表现为骨皮质增厚、骨干增粗、髓腔变窄。 |
| 慢性排脓通道形成 | 脓液经骨皮质破口和软组织向外排出，形成骨瘘孔、窦道和皮肤瘘口。 |
| 肉芽肿和干酪样坏死 | 结核以慢性肉芽肿为主，破坏可明显，但骨膜反应相对轻；液化坏死可形成冷脓肿。 |

**总推导：**

> 急性化脓性感染速度快，脓液多、压力高、骨膜反应明显；慢性化脓性感染以死骨、骨包壳和窦道为核心；骨结核进展慢，以肉芽肿、干酪坏死、冷脓肿和轻骨膜反应为特点。

### 1.2 影像检查选择

| 检查 | 主要价值 | 局限 |
| --- | --- | --- |
| X线 | 初筛和随访；显示骨质破坏、骨膜反应、死骨、骨包壳、骨硬化和畸形。 | 早期不敏感，骨矿物质丢失达到一定程度后才显示异常。 |
| CT | 显示骨皮质破坏、死骨、骨包壳、骨瘘孔、气体和复杂部位骨质细节。 | 早期骨髓水肿和软组织炎症不如 MRI。 |
| MRI | 最敏感地显示骨髓水肿、骨髓炎范围、骨膜下脓肿、软组织脓肿、窦道和关节侵犯。 | 对死骨钙化和细小骨皮质破坏不如 CT；金属伪影可影响观察。 |
| 增强 MRI | 区分蜂窝织炎、脓肿、坏死组织和活动性肉芽组织；显示窦道壁强化。 | 不是所有病例都必须增强；肾功能不全或禁忌证时需慎用。 |
| DWI/ADC | 脓肿和高细胞炎症区可弥散受限，有助于发现脓液和活动灶。 | 骨髓水肿、肿瘤和术后改变也可影响弥散，需结合常规序列。 |
| 核素骨显像 | 对多灶病变、全身筛查和 MRI 受限者有价值，敏感性较高。 | 特异性低，感染、肿瘤、骨折和术后修复均可摄取增高。 |
| 超声 | 可显示浅表软组织脓肿、关节积液和骨膜下液体，便于穿刺引导。 | 不能可靠评价深部骨髓和骨皮质内病变。 |

## 急性化脓性骨髓炎

### 1. 疾病本质

急性化脓性骨髓炎是化脓性细菌感染骨髓、骨皮质和骨膜所致的急性骨感染，最常见病原体为金黄色葡萄球菌。儿童多为血源性感染，成人也可由开放性骨折、手术、糖尿病足或邻近软组织感染直接蔓延。

儿童和青少年好发于四肢长骨干骺端，尤其股骨远端、胫骨近端、肱骨近端。临床常有发热、寒战、局部红肿热痛、拒动或跛行，白细胞、ESR 和 CRP 可升高。

### 2. 为什么儿童长骨干骺端好发

| 解剖/生理特点 | 机制 | 影像和临床意义 |
| --- | --- | --- |
| 干骺端血供丰富 | 儿童长骨生长活跃，干骺端代谢旺盛，血流量大。 | 血源性细菌更容易到达干骺端。 |
| 毛细血管袢呈急转弯 | 终末小动脉靠近骺板后折返，血流变慢。 | 血流淤滞使细菌更容易停留和定植。 |
| 静脉窦样网流速慢 | 干骺端窦样血管腔大、流速慢。 | 细菌和微小栓子不易被快速冲走。 |
| 局部吞噬功能相对弱 | 干骺端毛细血管内吞噬细胞清除能力有限。 | 一过性菌血症后可发展为感染灶。 |
| 骺板阻挡感染扩散 | 年长儿童感染多先局限于干骺端。 | 早期定位常在干骺端，随后可向骨干、骨膜下和软组织扩展。 |

**一句推导：** 儿童干骺端是“血供多但流速慢、转弯多、清除弱”的区域，所以血源性细菌容易停留，形成骨髓炎起点。

### 3. 底层病理逻辑

| 阶段 | 病理过程 | 影像结果 |
| --- | --- | --- |
| 早期充血水肿 | 细菌定植后引起骨髓充血、炎细胞浸润和水肿。 | MRI 最早阳性；X线可仅有软组织肿胀或完全阴性。 |
| 脓肿形成 | 炎症坏死和中性粒细胞聚集形成脓液，骨内压力升高。 | MRI 可见骨髓水肿、脓肿和增强异常；超声可见浅表骨膜下液体。 |
| 骨皮质和骨膜受累 | 脓液经骨小管穿破骨皮质，进入骨膜下。 | X线/CT 可见骨皮质破坏和骨膜反应；MRI 显示骨膜下脓肿。 |
| 血供中断 | 骨内压力升高、血栓性血管炎和骨膜掀起共同阻断骨血供。 | 局部骨质坏死形成死骨，CT 显示较清楚。 |
| 修复和包绕 | 骨膜和周围活骨产生新骨，包绕死骨和脓腔。 | 形成骨包壳，急性后期或转慢性时更明显。 |

### 4. 为什么早期 X线可阴性

早期病变主要是骨髓充血、水肿和炎细胞浸润，骨矿物质含量尚未明显减少。X线主要显示钙化骨基质，对“骨髓成分改变”不敏感，因此起病最初 7-14 天常可阴性。

当骨小梁被破坏、骨质脱钙达到一定程度后，X线才出现骨质疏松、斑片状或虫蚀状破坏。考试遇到“高热、局部痛、X线阴性”，不能排除急性骨髓炎，应想到 MRI。

### 5. X线表现及机制

| 时间/征象 | 典型表现 | 机制 |
| --- | --- | --- |
| 早期软组织肿胀 | 局部软组织肿胀，肌间隙模糊，脂肪线消失；骨质可正常。 | 炎症充血、水肿和渗出先发生在骨髓及邻近软组织。 |
| 干骺端骨质疏松 | 局部骨质密度减低，骨小梁模糊。 | 炎症充血、破骨活动增强和失用共同造成骨量减少。 |
| 斑点状/虫蚀状破坏 | 干骺端出现边界模糊的小片低密度破坏，可融合成片。 | 脓液和炎性肉芽组织破坏骨小梁。 |
| 骨膜反应 | 可见平行、层状、花边状或侵袭性骨膜新生骨。 | 脓液和炎症掀起骨膜，骨膜成骨细胞被刺激形成新骨。 |
| Codman三角 | 骨膜被快速掀起，两端残留三角形骨膜新生骨。 | 病变进展快，骨膜来不及形成连续厚层新骨；可与骨肉瘤、Ewing 肉瘤相似。 |
| 死骨 | 破坏区内出现相对高密度骨片，周围可有透亮带。 | 坏死骨失去血供，不被吸收，周围肉芽和脓液造成相对分离。 |
| 骨包壳 | 死骨周围出现新生骨壳，骨干增粗。 | 骨膜和活骨修复性成骨，包绕坏死骨和感染腔。 |

### 6. CT 表现及机制

CT 对骨皮质、死骨和骨包壳的显示优于 X线。急性期可见干骺端骨质破坏、骨皮质中断、骨膜新生骨、骨膜下或软组织脓肿。若存在气体，提示产气菌感染或开放性污染。

死骨在 CT 上常表现为破坏区内致密骨片，周围可有低密度肉芽或脓液。骨包壳表现为死骨外层不规则新生骨壳。CT 还能显示骨瘘孔，为手术清创和穿刺定位提供依据。

### 7. MRI 表现及机制

| 序列/技术 | 典型表现 | 机制 |
| --- | --- | --- |
| T1WI | 受累骨髓呈低信号，正常高信号脂肪髓被替代。 | 骨髓脂肪被水肿、脓液、炎细胞和肉芽组织替代，T1缩短脂肪信号减少。 |
| T2WI/STIR | 骨髓、骨膜下和软组织水肿呈高信号。 | 炎症渗出增加自由水含量，T2延长。 |
| 增强 T1WI | 活动性炎症和肉芽组织强化；脓肿中心不强化，脓肿壁环形强化。 | 肉芽组织血供丰富而强化，脓液无血供不强化。 |
| DWI/ADC | 脓肿腔可 DWI 高信号、ADC 降低；骨髓炎活动区可有弥散受限。 | 脓液黏稠，含大量炎细胞、细菌和蛋白，限制水分子运动。 |
| 骨膜下脓肿 | 骨皮质外侧、骨膜下液性高信号，可有环形强化。 | 脓液经骨皮质通道进入骨膜下，将骨膜掀起。 |
| 软组织受累 | 肌肉、筋膜和皮下组织水肿、蜂窝织炎或脓肿。 | 感染突破骨膜后向软组织扩散。 |

**骨髓水肿的核心表现：** T1WI 低信号，T2WI/STIR 高信号，增强后可强化。它是急性骨髓炎最早、最重要的 MRI 信号，但不是特异征象，骨挫伤、肿瘤和应力损伤也可有骨髓水肿。

### 8. 骨膜反应、死骨和骨包壳机制

| 征象 | 形成机制 | 提示意义 |
| --- | --- | --- |
| 骨膜反应 | 骨膜被脓液、炎症或局部充血刺激后成骨。儿童骨膜较松，更容易被掀起。 | 提示感染已累及骨皮质外侧或骨膜下。 |
| 死骨 | 骨内压升高、血管炎、血栓和骨膜剥离使骨段缺血坏死。 | 是化脓性骨髓炎的重要征象，尤其提示病变较重或进入慢性过程。 |
| 骨包壳 | 骨膜和周围活骨产生修复性新骨，包绕死骨和感染腔。 | 常提示机体试图限制感染，是慢性化倾向的重要表现。 |
| 骨瘘孔 | 脓液从骨包壳或骨皮质破口排出。 | 是窦道和反复排脓的骨性基础，CT/MRI 有助于定位。 |

### 9. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 骨肉瘤 | 青少年长骨干骺端，骨质破坏、骨膜反应、Codman三角和软组织肿块均可出现。 | 骨肉瘤常有瘤骨，表现为云絮状、象牙质样或放射状致密影；临床多无急性感染中毒表现。 | 骨肉瘤由肿瘤细胞直接成骨，感染则以脓液、炎症破坏、死骨和骨髓水肿为主。 |
| Ewing 肉瘤 | 儿童青少年，发热、疼痛、白细胞升高，长骨骨干或干骺端破坏伴层状骨膜反应。 | Ewing 肉瘤常有较大的软组织肿块，骨干受累更典型，死骨和脓肿证据少。 | 小圆细胞肿瘤可引起全身炎症样反应，但本质是肿瘤细胞浸润而非化脓坏死。 |
| 骨结核 | 可有骨质破坏、冷脓肿或窦道。 | 骨结核起病慢，全身中毒轻，骨膜反应相对少，死骨多细小；脊柱结核常有椎间隙变窄和椎旁脓肿。 | 结核为慢性肉芽肿性炎症，化脓性骨髓炎为急性脓液性炎症。 |
| 应力骨折/骨挫伤 | MRI 可有骨髓水肿，局部疼痛。 | 常有运动或外伤史，线样骨折或骨皮质反应为主，无脓肿、死骨和明显感染指标升高。 | 机械损伤导致骨髓出血水肿，不产生脓液和细菌性破坏。 |
| Langerhans 细胞组织细胞增生症 | 儿童骨质破坏，可有软组织肿块。 | 扁骨或颅骨常见，临床感染表现不典型，可多灶；确诊依赖病理。 | 单核-巨噬细胞系统异常增殖，而非细菌感染。 |

### 【考试答题版】

急性化脓性骨髓炎多由金黄色葡萄球菌血源性感染引起，儿童好发于长骨干骺端。其原因是干骺端血供丰富但血流缓慢，毛细血管袢急转弯，细菌易停留定植。临床表现为急性发热、寒战、局部红肿热痛及白细胞、ESR、CRP 升高。

早期病理改变为骨髓充血、水肿和炎细胞浸润，骨矿物质尚未明显丢失，因此起病 1-2 周内 X线可阴性或仅有软组织肿胀。进展后 X线可见干骺端骨质疏松、斑点状或虫蚀状骨质破坏、骨膜反应，严重者形成死骨和骨包壳。

CT 有利于显示骨皮质破坏、死骨、骨包壳、骨瘘孔和气体。MRI 是早期最敏感检查，表现为骨髓 T1WI 低信号、T2WI/STIR 高信号，增强后活动性炎症强化，脓肿中心不强化而壁强化，DWI 可显示脓肿弥散受限。鉴别诊断重点为骨肉瘤、Ewing 肉瘤和骨结核。

### 【一句话记忆】

**急性高热 + 儿童长骨干骺端 + 早期 X线可阴性 + MRI 骨髓水肿 + 后期骨膜反应、死骨和骨包壳 = 急性化脓性骨髓炎。**

### 【易错点】

- 不能用早期 X线阴性排除急性骨髓炎；早期应看 MRI。
- Codman三角不是骨肉瘤专属，快速进展的感染也可出现。
- 骨髓水肿是早期敏感征象，但不特异，需要结合发热、实验室指标、脓肿、死骨和病程。
- 儿童干骺端好发不是因为“离关节近”，而是因为干骺端血流慢、转弯多、细菌易停留。
- 死骨是坏死骨，骨包壳是周围活骨形成的新生骨，两者不能混用。

## 慢性化脓性骨髓炎

### 1. 疾病本质

慢性化脓性骨髓炎多由急性骨髓炎治疗不及时、不彻底或感染反复发作迁延而来，也可由开放性骨折、手术内固定、慢性溃疡和糖尿病足长期污染引起。

其本质是感染灶内存在难以清除的死骨、慢性肉芽组织和反复排脓通道。死骨无血供，抗生素和免疫细胞难以进入，因此成为细菌长期存留的核心。

### 2. 底层病理逻辑

| 病理过程 | 影像结果 |
| --- | --- |
| 死骨长期存留 | 破坏区内见高密度骨片；MRI 上死骨多为各序列低信号且不强化。 |
| 骨膜和骨内膜反复修复 | 骨皮质增厚、骨干增粗、髓腔变窄或闭塞，形成骨硬化。 |
| 脓液寻找低阻力通道 | 形成骨瘘孔、软组织窦道和皮肤排脓口。 |
| 活动性炎症反复发作 | MRI 出现新的骨髓水肿、软组织水肿、脓肿和窦道强化。 |
| 低毒力或局限性感染 | 可形成 Brodie 脓肿，表现为骨内局限性脓腔，周围硬化。 |

### 3. X线表现及机制

慢性骨髓炎的 X线表现常是“破坏和修复并存”。骨质破坏区提示感染和脓腔，骨质增生硬化提示长期修复，二者叠加可使骨形态粗大、变形。

常见表现包括骨皮质增厚、骨干增粗、髓腔变窄或消失、斑片状骨硬化、死骨、骨包壳、死腔和窦道。长期病变可造成骨畸形、病理骨折或邻近关节功能障碍。

### 4. CT 表现及机制

CT 对慢性骨髓炎最重要的价值是显示死骨、骨包壳、骨瘘孔和复杂骨结构。死骨因仍保留矿化骨基质而呈致密骨片，周围炎性肉芽和脓液吸收骨质，形成相对低密度间隙。

骨包壳表现为死骨或脓腔周围不规则新生骨。骨瘘孔表现为穿透骨皮质或骨包壳的通道，提示脓液可经此排出。CT 还能帮助制定清创范围和穿刺取材路径。

### 5. MRI 表现及机制

MRI 对判断是否仍有活动性感染最有价值。活动灶常表现为骨髓 T1WI 低信号、T2WI/STIR 高信号，增强后肉芽组织强化。脓肿中心不强化，周围壁强化，DWI 可弥散受限。

窦道在 MRI 上可表现为从骨感染灶通向皮肤或软组织表面的线状或管状异常信号，增强后窦道壁强化。MRI 还能显示肌肉、筋膜、皮下软组织受累范围和邻近关节侵犯。

死骨在 MRI 上常呈 T1WI、T2WI 均低信号且不强化，但细小死骨不如 CT 清楚。因此“找死骨看 CT，判活动和窦道看 MRI”是慢性骨髓炎的实用原则。

### 6. 特殊类型

| 类型 | 典型表现 | 机制和意义 |
| --- | --- | --- |
| Brodie 脓肿 | 长骨干骺端或骨干局限性透亮区，周围硬化；MRI 可见脓腔和环形强化。 | 低毒力感染被局限在骨内，病程较慢，容易误认为良性骨肿瘤。 |
| 硬化性骨髓炎（Garre 骨髓炎） | 骨皮质增厚、髓腔变窄，以骨硬化为主，破坏不明显。 | 低毒力慢性感染反复刺激成骨反应。 |
| 慢性复发性多灶性骨髓炎 | 多灶骨髓水肿，可反复发作，培养常阴性。 | 偏炎症免疫机制，需结合临床和排除感染、肿瘤。 |

### 7. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 骨肿瘤 | 慢性疼痛、骨质破坏、硬化和软组织肿块可相似。 | 慢性骨髓炎常有死骨、骨包壳、窦道和反复感染史；肿瘤可见瘤骨、软骨钙化或持续侵袭性肿块。 | 感染是破坏和修复反复交替；肿瘤是异常细胞持续增殖。 |
| 骨肉瘤 | 骨膜反应、骨质破坏和软组织肿块可重叠。 | 骨肉瘤有肿瘤骨和进行性软组织肿块，缺乏排脓窦道和死骨包壳组合。 | 骨肉瘤瘤骨来自肿瘤细胞成骨，慢性骨髓炎硬化来自修复性成骨。 |
| 骨结核 | 慢性病程、骨质破坏、窦道和冷脓肿可相似。 | 骨结核骨膜反应相对轻，死骨多细小，常有冷脓肿和脊柱椎间隙受累；化脓性慢性骨髓炎骨硬化和骨包壳更明显。 | 结核是肉芽肿性干酪坏死，化脓性炎症是脓液、死骨和修复性新骨反复循环。 |
| 骨纤维异常增殖症 | 骨膨胀、硬化或磨玻璃样改变。 | 通常无感染症状、无死骨窦道，病变呈较均匀磨玻璃基质。 | 发育性骨纤维组织替代正常骨，不产生脓液。 |

### 【考试答题版】

慢性化脓性骨髓炎多由急性骨髓炎迁延或开放性损伤、手术、慢性溃疡感染而来。病理基础是死骨存留、肉芽组织包绕、骨包壳形成和窦道反复排脓。死骨无血供，抗生素难以进入，是感染反复不愈的关键。

X线表现为骨质破坏和骨质增生硬化并存，可见骨皮质增厚、骨干增粗、髓腔变窄或消失、死骨、死腔、骨包壳和窦道。CT 对显示死骨、骨包壳、骨瘘孔和手术清创范围最有价值。

MRI 主要用于判断活动性炎症和软组织范围。活动灶表现为骨髓 T1WI 低信号、T2WI/STIR 高信号，增强后肉芽组织强化；脓肿中心不强化，窦道壁强化。鉴别诊断重点为骨肿瘤和骨结核。

### 【一句话记忆】

**慢性骨髓炎 = 死骨长期存留 + 骨包壳包绕 + 骨硬化增粗 + 窦道反复排脓；找死骨看 CT，判活动看 MRI。**

### 【易错点】

- 慢性骨髓炎不是“没有活动炎症”，而是活动和修复反复交替。
- 死骨是感染复发的核心，单靠影像看到硬化不能说明已经痊愈。
- CT 比 MRI 更适合找细小死骨；MRI 比 CT 更适合看活动性炎症、软组织脓肿和窦道。
- Brodie 脓肿可表现为边界较清的骨内透亮区，容易误诊为良性骨肿瘤。
- 有软组织肿块不一定是肿瘤，慢性骨髓炎也可形成炎性肿块和窦道。

## 骨结核

### 1. 疾病本质

骨结核是结核分枝杆菌感染骨和关节所致的慢性肉芽肿性炎症，多继发于肺结核或其他部位结核，经血行播散到骨骼。好发于儿童和青少年，也可见于成人。

最常见部位为脊柱，其次可累及长骨干骺端、骨骺、短管状骨和关节邻近骨端。骨结核病程慢，局部疼痛、低热、盗汗、消瘦等全身症状可较轻，影像上常表现为“破坏明显，反应较轻”。

### 2. 慢性肉芽肿性炎症机制

| 病理过程 | 影像结果 |
| --- | --- |
| 结核菌进入松质骨 | 松质骨内形成结核结节和肉芽组织。 |
| 干酪样坏死 | 局部骨小梁被破坏，形成虫蚀状、囊状或地图样骨质破坏。 |
| 炎症进展慢 | 骨膜刺激弱，骨膜反应和骨硬化常不如化脓性感染明显。 |
| 液化坏死向软组织流注 | 形成冷脓肿，疼痛和急性红热可不明显。 |
| 邻近关节或椎间盘受累 | 可出现关节破坏、椎间隙变窄和脊柱畸形。 |

### 3. 为什么骨质破坏明显而骨膜反应较轻

骨结核以肉芽肿和干酪样坏死为主，病程缓慢，炎症压力和急性脓液刺激不如化脓性骨髓炎强。骨膜没有被大量脓液快速掀起，因此骨膜反应相对少或轻。

同时，结核肉芽组织可长期侵蚀松质骨，导致骨质破坏范围逐渐扩大。于是影像上可出现“骨破坏不少，但骨膜新生骨不多”的不匹配表现，这是骨结核区别于急性化脓性骨髓炎的重要线索。

### 4. 长骨和短骨结核表现

| 部位/征象 | 表现 | 机制 |
| --- | --- | --- |
| 长骨干骺端或骨骺 | 囊状、虫蚀状或地图样骨质破坏，可跨骺板或侵犯关节。 | 结核肉芽组织沿松质骨和关节滑膜缓慢蔓延。 |
| 骨膜反应 | 无或较轻，儿童可有轻度骨膜新生骨。 | 慢性低度炎症对骨膜刺激弱。 |
| 死骨 | 可有细小、泥沙样死骨。 | 干酪样坏死和小片骨坏死，不如化脓性大片死骨典型。 |
| 冷脓肿 | 骨旁或软组织内低密度/液性病灶，壁较薄，可钙化。 | 干酪样坏死液化后沿筋膜间隙流注，急性炎症反应轻。 |
| 窦道 | 晚期可形成皮肤窦道。 | 冷脓肿或坏死组织向体表破溃。 |
| 短骨结核 | 指骨、掌骨或跖骨膨胀性骨质破坏，皮质变薄，呈“骨气臌”。 | 儿童短管状骨松质骨受累后膨胀，骨皮质被撑薄。 |

### 5. 脊柱结核

脊柱结核又称 Pott 病，是骨结核最常见、考试最重要的类型。病变常起于椎体前部或椎体终板附近，随后累及相邻椎体和椎间盘，可形成椎旁或腰大肌冷脓肿。

| 征象 | 表现 | 机制 |
| --- | --- | --- |
| 椎体破坏 | 相邻两个椎体前缘或终板附近骨质破坏，椎体可楔形塌陷。 | 结核经椎体松质骨和终板下区域蔓延，破坏承重结构。 |
| 椎间隙变窄 | 椎间盘受累后椎间隙逐渐变窄或消失。 | 成人椎间盘本身血供差，多由相邻椎体终板破坏后继发受累。 |
| 椎旁脓肿 | 椎旁梭形软组织影，CT/MRI 可见液性脓肿，壁较薄。 | 干酪样坏死液化后沿前纵韧带和椎旁筋膜扩散。 |
| 腰大肌脓肿 | 腰椎结核可沿腰大肌向下流注至髂窝或腹股沟。 | 冷脓肿沿肌筋膜间隙下行，局部急性红热不明显。 |
| 后凸畸形 | 多椎体破坏和塌陷后形成脊柱后凸。 | 椎体前柱承重结构被破坏，前方高度丢失。 |
| 椎管受压 | 硬膜外脓肿、肉芽组织、塌陷骨块压迫脊髓。 | 感染向椎管内扩展或脊柱畸形造成神经压迫。 |

**脊柱结核答题线索：** 相邻椎体破坏 + 椎间隙变窄 + 椎旁/腰大肌冷脓肿 + 后凸畸形。

### 6. 影像模态表现

| 检查 | 表现 | 价值 |
| --- | --- | --- |
| X线 | 骨质疏松、虫蚀状或囊状骨质破坏，骨膜反应少；脊柱可见椎体破坏、椎间隙变窄和椎旁软组织影。 | 初筛和随访畸形，但早期不敏感。 |
| CT | 清楚显示骨质破坏、细小死骨、椎体塌陷、椎旁脓肿壁钙化。 | 评估骨破坏范围、死骨和穿刺活检路径。 |
| MRI | 显示骨髓受累、椎间盘侵犯、椎旁/硬膜外脓肿、脊髓受压和软组织扩散。 | 脊柱结核和早期骨结核最重要检查。 |
| 增强 MRI | 肉芽组织和脓肿壁强化，脓肿中心不强化。 | 区分脓肿、肉芽组织和坏死范围。 |
| DWI/ADC | 冷脓肿或活动性炎症可弥散受限。 | 帮助发现脓液和活动病灶，但需结合常规 MRI。 |
| 核素显像 | 可提示多灶骨病变。 | 敏感但不特异，不能单独区分结核、化脓性感染和肿瘤。 |

### 7. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
| --- | --- | --- | --- |
| 急性化脓性骨髓炎 | 骨质破坏、脓肿和软组织受累。 | 化脓性起病急、高热明显，骨膜反应和大片死骨更常见；结核起病慢，骨膜反应轻，冷脓肿和泥沙样死骨更典型。 | 化脓性感染脓液压力高、破坏快；结核为慢性肉芽肿和干酪坏死。 |
| 慢性化脓性骨髓炎 | 慢性病程、窦道、死骨、硬化可重叠。 | 慢性化脓性骨髓炎常有明显骨硬化、骨包壳和大片死骨；结核骨硬化较轻，冷脓肿和关节/椎间隙受累更常见。 | 化脓性修复性成骨强，结核反应性成骨弱。 |
| 骨肿瘤 | 骨质破坏、软组织肿块、椎体塌陷可相似。 | 肿瘤常缺乏冷脓肿、窦道和感染证据；转移瘤椎间隙常保留，骨肉瘤可有瘤骨。 | 肿瘤以细胞增殖和肿瘤基质为主，结核以肉芽肿、干酪坏死和脓肿为主。 |
| 脊柱转移瘤 | 椎体破坏、压缩骨折、椎旁软组织肿块。 | 转移瘤多见中老年，有原发瘤史，椎间盘常保持正常，椎弓根受累常见；结核常跨椎间盘累及相邻椎体并形成冷脓肿。 | 转移瘤经血行种植椎体骨髓，通常不先破坏椎间盘。 |
| 布鲁菌性脊柱炎 | 腰椎受累、终板破坏和椎旁炎症。 | 布鲁菌病常有畜牧、未消毒乳制品接触史，骨破坏较轻，椎旁脓肿通常小。 | 低毒力细菌感染可慢性进展，但冷脓肿和广泛椎体塌陷不如结核典型。 |

### 【考试答题版】

骨结核是结核分枝杆菌经血行播散至骨骼引起的慢性肉芽肿性炎症，多继发于肺结核。病理基础为结核肉芽肿、干酪样坏死和冷脓肿形成。因病程慢、脓液压力低、骨膜刺激轻，所以骨质破坏可明显，而骨膜反应相对少。

长骨骨结核表现为囊状、虫蚀状或地图样骨质破坏，骨膜反应少，可见细小泥沙样死骨和冷脓肿。短管状骨结核可出现膨胀性破坏、皮质变薄，称骨气臌。CT 有助于显示细小死骨和脓肿钙化，MRI 可显示骨髓、软组织和关节侵犯。

脊柱结核最常见，常表现为相邻椎体破坏、椎间隙变窄、椎旁或腰大肌冷脓肿，晚期可有椎体塌陷、后凸畸形和椎管受压。鉴别时，化脓性骨髓炎起病急、骨膜反应明显、大片死骨多；转移瘤椎间隙多保留，常有原发肿瘤史。

### 【一句话记忆】

**骨结核 = 慢性起病 + 肉芽肿干酪坏死 + 骨破坏明显但骨膜反应轻 + 冷脓肿；脊柱结核记住相邻椎体破坏、椎间隙窄、椎旁脓肿。**

### 【易错点】

- 骨结核不是没有脓肿，而是“冷脓肿”：红、热、急痛不明显。
- 骨质破坏明显但骨膜反应轻，是结核的重要提示。
- 脊柱结核常累及相邻椎体和椎间隙；脊柱转移瘤椎间隙多保留。
- 结核也可形成窦道和死骨，但死骨多较细小，不如化脓性大片死骨典型。
- 影像怀疑骨结核时，最终仍需结合结核接触史、实验室检查、病原学或病理证据。
