## 骨折总论

### 1. 疾病本质

骨折（fracture）是骨和/或软骨结构发生断裂，骨的连续性中断。以长骨和脊椎骨较多见。骨折后在断端之间及其周围形成血肿→血肿机化→纤维性骨痂→骨性骨痂→塑形改建。

### 2. X线表现（首选检查）

| 特征 | 表现 |
|---|---|
| **骨折线** | 不规则透明线（骨皮质断裂清楚整齐，骨松质断裂表现为骨小梁中断、扭曲、错位） |
| 嵌入性骨折 | 不显示骨折线，而表现为边缘不规则的高密度条带影，骨皮质与骨小梁连续性中断（易漏诊，以股骨颈多见） |

### 3. 骨折类型

| 分类依据 | 类型 |
|---|---|
| 程度 | 完全性、不完全性 |
| 形状 | 横行、斜行、螺旋形、"T"形、"Y"形 |
| 骨片 | 撕脱性、嵌入性、压缩性、粉碎性 |

### 4. 儿童骨折特点

| 类型 | 表现 | 机制 |
|---|---|---|
| **青枝骨折** | 局部骨皮质和骨小梁扭曲，看不见骨折线，或骨皮质褶皱、凹陷或隆突 | 儿童骨骼柔韧性大，外力不完全断裂 |
| **骺离骨折** | 骨骺与干骺端距离增加（骨骺软骨不显影故只见间隙增宽） | 骨骺骨折导致骨骺移位 |

### 5. 骨折移位

- 对位不良：内外、前后、上下移位
- 对线不良：成角移位
- 旋转移位：骨折远段围绕骨纵轴旋转

### 6. 骨折愈合过程

| 阶段 | X线表现 |
|---|---|
| 血肿期（骨折后即刻） | 骨折线清晰 |
| 纤维性骨痂（2～3天后） | 骨折线开始模糊 |
| 骨性骨痂 | 骨折线模糊不清，骨膜增生骨化形成外骨痂 |
| 骨性愈合 | 骨折线消失 |
| 改建 | 恢复骨骼正常形态 |

### 7. 常见并发症

| 并发症 | 表现 |
|---|---|
| 延迟愈合/不愈合 | 骨痂延迟/不出现，骨折线长期存在/断端硬化吸收变尖 |
| 畸形愈合 | 成角、旋转、缩短、延长 |
| 骨缺血性坏死 | 股骨颈骨折后股骨头坏死最常见 |
| 骨感染 | 同骨髓炎表现 |
| 关节强直/退行性变 | 关节内骨折后 |
| 骨化性肌炎 | 骨折后肌纤维间广泛骨化 |

### 8. 骨折诊断注意事项

- 骨干骨折线需与骨滋养动脉影（仅斜穿一侧骨皮质、边界光整）区别
- 干骺端骨折线需与骺线（解剖部位固定、两旁有硬化线）区别
- 怀疑骨折但X线阴性→CT/MRI（发现骨挫伤/隐性骨折）或2周后复查

<a id="colles-fracture"></a>

## Colles骨折（伸直型桡骨远端骨折）

- 桡骨远端3cm以内的横行或粉碎性骨折
- 骨折远段向**背侧**移位，断端向**掌侧**成角
- 可伴尺骨茎突骨折

---

<a id="supracondylar-fracture"></a>

## 肱骨髁上骨折

- 多见于**儿童**，骨折线横过喙突窝和鹰嘴窝
- 远侧端多向**背侧**移位

---

<a id="femoral-neck-fracture"></a>

## 股骨颈骨折

- 多见于**老年妇女**
- 分头下型、颈中型、基底型
- **头下型**影响股骨头血供→愈合缓慢→易**股骨头缺血性坏死**
- 约10%为嵌入性骨折X线难以检出→CT/MRI

### 考试答题版

骨折首选X线平片（正侧位）。典型表现：骨折线（不规则透明线，骨皮质断裂、骨小梁中断扭曲）。儿童特殊骨折：青枝骨折（骨皮质扭曲无骨折线）、骺离骨折（骨骺与干骺端距离增宽）。Colles骨折：桡骨远端3cm内，远段向背侧移位、向掌侧成角。股骨颈骨折：老年女性，头下型易股骨头坏死。并发症：延迟愈合、不愈合、缺血坏死、感染、关节强直。

### 一句话记忆

**外伤史 + X线不规则透明骨折线 = 骨折，儿童青枝骨折无骨折线，股骨颈头下型→股骨头坏死。**
