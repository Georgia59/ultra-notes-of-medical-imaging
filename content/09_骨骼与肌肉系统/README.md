## 高频征象速查

| 征象 | 疾病 | 含义 |
|---|---|---|
| Codman三角（骨膜三角） | 骨肉瘤 | 骨膜被肿瘤掀起→残留骨膜与皮质间三角形改变 |
| 日光/放射状骨膜反应 | 骨肉瘤 | 肿瘤沿血管向外生长 |
| 皂泡样改变 | 骨巨细胞瘤 | 骨端膨胀性溶骨破坏呈分房状 |
| 骨性强直 | 化脓性关节炎 | 关节间隙消失+骨小梁贯通 |
| 纤维性强直 | 关节结核 | 关节间隙缩窄但存在+无骨小梁贯通 |
| 竹节椎 | 强直性脊柱炎 | 脊柱韧带广泛骨化致竹节样外观 |
| 关节鼠 | 退行性骨关节病 | 关节内游离体 |
| 分层状骨化 | 骨化性肌炎 | 周缘成熟骨化较明显，反映从外向内成熟 |
