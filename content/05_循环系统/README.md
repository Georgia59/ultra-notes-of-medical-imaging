## 大纲覆盖核对

- **先天性心脏病：** 房间隔缺损、法洛四联症。
- **缺血性心脏病：** 冠状动脉粥样硬化性心脏病、急性与陈旧性心肌梗死。
- **心肌病：** 扩张型心肌病。
- **心包疾病：** 心包积液、缩窄性心包炎。
- **主动脉疾病：** 主动脉夹层、主动脉壁内血肿、主动脉瘤。
- **瓣膜病：** 风湿性心脏病、二尖瓣及主动脉瓣病变。
- **血管疾病：** 肺动脉栓塞、下肢动脉粥样硬化。

## 高频征象速查

| 征象 | 首先考虑 | 底层含义 |
|---|---|---|
| 肺血增多、右房右室增大 | 房间隔缺损 | 心房水平左向右分流增加右心容量负荷 |
| 靴形心、肺血减少 | 法洛四联症 | 右室肥厚使心尖上翘，肺动脉狭窄减少肺血流 |
| 心内膜下或透壁 LGE | 心肌梗死 | 坏死后细胞外间隙扩大，钆对比剂延迟清除 |
| 室间隔中层条形 LGE | 扩张型心肌病 | 非缺血性心肌间质纤维化 |
| 烧瓶形心影 | 大量心包积液 | 液体均匀包绕心脏并扩大心影 |
| 心包增厚钙化 | 缩窄性心包炎 | 慢性炎症纤维化限制心室舒张 |
| 内膜片及主动脉双腔 | 主动脉夹层 | 血流进入中层形成真腔和假腔 |
| 肺动脉腔内充盈缺损 | 肺动脉栓塞 | 血栓不被对比剂充填 |

## 血流动力学总线

| 病理变化 | 首要后果 | 影像表现 |
|---|---|---|
| 左向右分流 | 肺血流和右心容量负荷增加 | 肺血增多、肺动脉扩张、右心增大 |
| 右向左分流 | 未氧合血进入体循环 | 发绀，肺血量取决于肺动脉狭窄程度 |
| 瓣膜狭窄 | 瓣前压力负荷增加 | 瓣前心腔扩大、心肌肥厚、跨瓣高速血流 |
| 瓣膜关闭不全 | 反流造成容量负荷 | 反流涉及的前后心腔扩大 |
| 心肌收缩减弱 | 射血分数降低、残余血增多 | 心腔扩大、室壁运动减弱、肺淤血 |
| 心包缩窄 | 心室舒张充盈受限 | 心房扩大、腔静脉扩张、体静脉淤血 |
