# 扩张型心肌病

<a id="dilated-cardiomyopathy"></a>

## 1. 疾病本质

在没有足以解释整体收缩功能障碍的异常负荷或冠状动脉疾病时，出现左心室扩张和收缩功能下降，可伴右室受累。病因可为遗传、感染、毒性或特发性，也可代表多种心肌病的终末期。

## 2. 底层病理逻辑

- **心肌收缩无力：** 每搏排血减少，舒张末期残余血增加，心腔逐渐扩大。
- **容量重构：** 左室最明显，严重时四个心腔均扩大，心脏呈球形。
- **室壁相对变薄：** 心腔扩张超过心肌肥厚代偿能力。
- **瓣环扩大和乳头肌移位：** 形成继发性二尖瓣、三尖瓣反流。
- **血流淤滞：** 扩大心腔内可形成附壁血栓。
- **间质纤维化：** MRI 可出现室间隔中层条形延迟强化。

## 3. 临床表现及机制

| 临床表现 | 产生机制 |
|---|---|
| 劳力性气短、端坐呼吸 | 左室收缩功能下降导致肺静脉淤血 |
| 心悸 | 心腔扩张和纤维化引起心律失常 |
| 下肢水肿、肝大 | 右心衰竭和体静脉淤血 |
| 动脉栓塞 | 心腔内血栓脱落 |
| 活动耐量下降 | 心排量不足 |

## 4. X 线与 CT 表现及机制

### X 线平片

- 心影呈普大型或主动脉瓣型，反映全心尤其左室扩大。
- 心功能不全时可见肺淤血、肺水肿和胸腔积液。

### CT

- 舒张末期左、右心室腔扩大，以左室最显著，可伴双房扩大。
- 心室壁厚度正常或变薄，区别于以肥厚为主的压力负荷。
- Cine CT 可见整体室壁运动普遍减弱、射血分数降低。
- 冠脉 CTA 的重要用途是排除缺血性心肌病。

## 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
|---|---|---|
| Cine | 左室显著扩大，整体收缩减弱，EF 降低 | 弥漫性心肌收缩功能障碍 |
| T1/T2 | 多无特异信号；活动性炎症时可异常 | 病因和病程决定水肿程度 |
| LGE | 室间隔中层条形强化，可无强化 | 非缺血性间质纤维化，不从心内膜下开始 |
| T1 mapping/ECV | 可升高 | 弥漫性心肌纤维化扩大细胞外间隙 |
| 血栓序列 | 心腔内无强化低信号团块 | 血栓缺乏血供 |

> PPT 固定表述：**左心腔扩张、室壁均匀性变薄、收缩功能显著下降；室间隔中层条形 LGE 具有提示意义。**

## 6. 超声表现及机制

- 全心扩大，以左室扩大最明显。
- 室壁运动呈普遍性减低，而非单一冠脉供血区节段性异常。
- 二尖瓣活动幅度降低，可见多瓣膜反流。
- 心腔内“云雾状”回声提示低流速血液，严重时形成血栓。

## 7. 关键征象机制

- **征象：左室扩大伴整体收缩减弱**  
  **机制：** 弥漫性心肌功能障碍导致容量重构。  
  **提示意义：** DCM 的核心形态与功能组合。
- **征象：室间隔中层条形 LGE**  
  **机制：** 非缺血性间质纤维化主要位于心肌中层。  
  **提示意义：** 有助于与心肌梗死型强化鉴别，并与预后相关。
- **征象：多瓣膜反流**  
  **机制：** 心腔和瓣环扩大、乳头肌移位使瓣叶不能正常对合。  
  **提示意义：** 多为继发性功能性反流。

## 8. 扩张型与缺血性心肌病对比

| 项目 | 扩张型心肌病 | 缺血性心肌病 |
|---|---|---|
| 冠脉 | 无足以解释病变的严重狭窄 | 存在责任冠脉狭窄或闭塞 |
| 室壁运动 | 普遍性减弱 | 按冠脉供血区节段性异常 |
| LGE | 中层条形或斑片状 | 心内膜下或透壁 |
| 室壁瘤 | 少见 | 陈旧透壁梗死后常见 |
| 底层逻辑 | 弥漫性心肌病变 | 局部供血中断后坏死瘢痕 |

## 9. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
|---|---|---|---|
| 风湿性瓣膜病 | 心腔扩大、心衰、反流 | 存在原发瓣叶增厚、粘连、钙化和开放受限 | 瓣膜病先发生，心肌改变继发 |
| 心肌炎后心肌病 | 扩张、收缩减弱、中层 LGE | 需结合感染史、水肿及心肌炎分布 | 两者可连续，病因证据决定分类 |
| 心包积液 | 心影普遍增大 | 超声见心包液性暗区，心腔本身未必扩大 | 心影扩大来自心外液体 |

## 10. 考试答题版

扩张型心肌病表现为左心室扩张和收缩功能下降，可累及右心。X 线可见普大型心影及肺淤血。超声、CT 和 MRI 显示全心扩大，以左室明显，室壁正常或变薄，整体室壁运动普遍减弱，射血分数降低，可伴多瓣膜反流及心腔内血栓。MRI 延迟强化可见室间隔中层条形强化，反映非缺血性纤维化；冠脉 CTA 用于排除缺血性心肌病。

## 11. 一句话记忆

**扩心病是“大、薄、弱”：左室大、壁相对薄、整体收缩弱，中层 LGE 不按冠脉走。**
