## 四、高频征象速查

| 征象名称 | 英文 | 疾病 | 含义 |
|---|---|---|---|
| 半月综合征 | meniscus sign | 溃疡型胃癌 | 不规则半月形龛影+环堤+指压痕+裂隙征 |
| 蚯蚓状充盈缺损 | — | 食管静脉曲张 | 黏膜下静脉迂曲扩张呈蚯蚓状透亮影 |
| 卵石征 | cobblestone sign | 克罗恩病 | 纵横交错溃疡与水肿隆起黏膜形成鹅卵石路面状 |
| 梳样征 | comb sign | 克罗恩病活动期 | 系膜直小血管增多呈梳齿状 |
| 快进快出 | wash-in/wash-out | HCC | 动脉期明显强化→门静脉期/平衡期迅速廓清 |
| 快进慢出 | early enhancement + delayed filling | 肝海绵状血管瘤 | 动脉期周边结节状强化→延迟期向心填充 |
| 灯泡征 | light-bulb sign | 肝海绵状血管瘤 | T2WI上肿瘤呈均匀明显高信号 |
| 牛眼征 | bull's eye sign | 肝转移瘤 | 边缘环形强化+中央坏死无强化 |
| 环征（双环征） | double target sign | 肝脓肿 | 无强化脓腔+明显强化脓肿壁+外周水肿带 |
| 中心点征 | central dot sign | Caroli病 | 囊状扩张胆管包绕门静脉分支→中心点状强化 |
| 双管征 | double duct sign | 胰头癌/下段胆管癌/壶腹癌 | 胆总管+胰管同时扩张 |
| 软藤征 | — | 胆管梗阻 | 肝内扩张胆管迂曲如藤条 |
| 靶征 | target sign | 胆总管结石/UC | 结石周围胆汁环绕；或肠壁分层强化 |
| 瓷化胆囊 | porcelain gallbladder | 慢性胆囊炎 | 胆囊壁广泛钙化 |
| 皮革胃 | linitis plastica | 弥漫浸润型胃癌（Borrmann Ⅳ型） | 全胃壁僵硬增厚无蠕动 |
| 三明治征 | sandwich sign | 胃肠道淋巴瘤 | 肿大淋巴结融合包绕强化血管 |

| 铅管征 | lead pipe sign | 溃疡性结肠炎慢性期 | 肠管短缩+结肠袋消失 |

## 五、本章总体诊断逻辑

### 食管与胃肠道

```
吞咽困难 → 食管钡剂造影
  ├── 充盈缺损/龛影/管壁僵硬 → 食管癌（分四型）
  └── 蚯蚓状充盈缺损/管壁柔软 → 食管静脉曲张（门静脉高压）

上腹痛 → 上消化道钡剂造影
  ├── 龛影突出于轮廓外+黏膜线/项圈征 → 良性胃溃疡
  ├── 龛影位于轮廓内+环堤/指压痕 → 恶性溃疡（胃癌）
  ├── 球部变形+激惹征 → 十二指肠球部溃疡
  ├── 胃壁增厚柔软+多部位+淋巴结 → 淋巴瘤
  └── 边界清黏膜完整明显强化 → GIST

便血/肠梗阻 → 结肠造影/CT
  ├── 局限性狭窄/肿块/结肠袋消失 → 结直肠癌
  ├── 跳跃性+系膜侧+梳样征 → 克罗恩病
  ├── 回盲部+环形对称性狭窄+淋巴结环形强化 → 肠结核
  └── 连续性+左半结肠+铅管征 → 溃疡性结肠炎
```

### 肝脏

```
肝内占位 → CT多期增强/MRI
  ├── 快进快出+肝硬化+AFP → HCC
  ├── 快进慢出+灯泡征+T2WI极高信号 → 海绵状血管瘤
  ├── 多发+牛眼征+原发瘤史 → 转移瘤
  ├── 水样密度+壁菲薄无强化 → 肝囊肿
  └── 环征+发热+DWI显著高信号 → 肝脓肿
```

### 胆道

```
右上腹痛/黄疸 → 超声→CT/MRCP
  ├── 强回声+后方声影+移动 → 胆结石
  ├── 胆囊壁不均匀增厚+腔内肿块 → 胆囊癌
  ├── 扩张胆管远端突然截断+壁厚强化 → 胆管癌
  ├── 胆总管全程囊性扩张+肝内胆管正常 → 胆总管囊肿（Ⅰ型）
  └── 肝内多发囊+中心点征+与胆管相通 → Caroli病（Ⅴ型）
```

### 胰腺

```
上腹痛+淀粉酶升高 → CT
  ├── 胰腺肿大+均匀强化+胰周渗出 → 水肿型胰腺炎
  └── 不均匀+坏死无强化 → 坏死型胰腺炎

梗阻性黄疸+CA19-9 → CT/MRI
  ├── 乏血供肿块+双管征+血管侵犯 → 胰腺癌
  └── 富血供+明显强化 → pNEN
```

### 急腹症

```
急性腹痛 → 立位腹平片→CT
  ├── 阶梯状气液平面 → 小肠梗阻
  │   ├── 无绞窄征象 → 单纯性
  │   └── 假肿瘤征/咖啡豆征/空回肠换位 → 绞窄性
  ├── 膈下游离气体 → 消化道穿孔
  └── 高密度血肿+腹腔积血 → 腹部实质脏器外伤
```
