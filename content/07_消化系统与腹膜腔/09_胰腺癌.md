## 胰腺癌（胰腺导管腺癌）

### 1. 疾病本质

胰腺癌（pancreatic carcinoma）通常指胰腺导管腺癌，约占胰腺原发恶性肿瘤的90%。胰头癌占60%～70%，体尾部次之。好发年龄45～65岁，男女比约2∶1。病理：肿瘤富有黏蛋白和致密胶原纤维性基质（间质丰富），易局部侵犯周围血管和神经，易发生淋巴结及肝转移。预后极差，是所有恶性肿瘤中生存率最低者之一。临床表现：早期无特异症状；进展期出现进行性无痛性梗阻性黄疸；体尾部肿瘤晚期出现持续性剧烈左腰背部痛。血清CA19-9常显著增高。

### 2. 底层病理逻辑

- **病理因素：** 胰腺导管上皮恶变→大量致密纤维间质（促结缔组织增生反应）→肿瘤质地硬、乏血供。胰头癌→阻塞胆总管→梗阻性黄疸→肝内外胆管扩张+胆囊增大+胰管扩张→"双管征"。肿瘤侵犯胰周血管（肠系膜上血管、腹腔干、门静脉等）和神经丛→疼痛。易肝转移和淋巴结转移。
- **对影像的影响：** 大量纤维间质+乏血供→增强扫描强化不明显（相对于正常胰腺呈低密度/信号），可出现延迟强化。肿块阻塞胰管→上游胰管扩张、胰腺实质萎缩。胰头癌阻塞胆总管+胰管→双管征。血管侵犯→血管被包绕、狭窄、闭塞。

### 3. 临床表现及机制

| 临床表现 | 产生机制 |
|---|---|
| 进行性无痛性梗阻性黄疸 | 胰头癌阻塞胆总管下端→胆汁排出受阻 |
| 持续性剧烈左腰背部痛（体尾部） | 肿瘤侵犯腹膜后腹腔神经丛 |
| 反复发作性急性胰腺炎 | 肿瘤阻塞胰管致胰液引流障碍 |
| 消瘦、恶病质 | 肿瘤消耗 + 消化吸收障碍 |
| CA19-9显著增高 | 肿瘤细胞分泌糖类抗原19-9 |

### 4. CT 表现及机制（最主要影像检查方法）

#### 直接征象

| 特征 | 表现 | 机制 |
|---|---|---|
| CT平扫 | 肿块密度常与邻近胰腺组织相似，较小者不易发现；较大者胰腺局部增大，肿块内可有坏死性低密度灶 | 肿瘤以纤维间质为主，密度接近胰腺 |
| CT增强（关键） | 胰腺癌为**乏血供肿瘤**，强化不明显，呈**相对低密度**（与周围明显强化的正常胰腺形成对比），可出现一定程度的**延迟强化** | 间质丰富+血供少→对比剂进入少且慢；延迟强化因对比剂在扩大细胞外间隙内滞留 |

#### 间接征象

| 特征 | 表现 | 机制 |
|---|---|---|
| **双管征**（double duct sign） | 胰头癌合并胰管和胆总管扩张 | 肿瘤同时阻塞胆总管下端和胰管 |
| 胰腺体尾部萎缩 | 胰管阻塞→上游胰腺实质萎缩 | 胰液引流障碍 |
| 潴留性囊肿 | 阻塞上游的胰管囊状扩张 | 胰管压力增高 |
| 胰周侵犯 | 胰周低密度脂肪层消失 | 肿瘤向外侵犯 |
| 血管侵犯 | 血管被包绕、狭窄甚至中断 | 肿瘤直接侵犯包埋 |
| 淋巴结转移 | 胰周、肝门、腹膜后多发软组织结节 | 淋巴道转移 |
| 肝转移 | 多发低密度灶 | 血行转移至肝 |

### 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
|---|---|---|
| T1WI | 胰腺肿块信号稍低于正常胰腺 | 肿瘤组织T1值延长 |
| 脂肪抑制T1WI | 低信号病灶与高信号正常胰腺对比更显著 | 肿瘤信号更低，正常胰腺被强调 |
| T2WI | 多呈等或稍高信号 | 肿瘤含水量与胰腺相近或略高 |
| 增强 | 同CT增强（乏血供+延迟强化） | 同CT机制 |
| DWI | 原发灶、淋巴结转移和肝转移均呈高信号 | 肿瘤细胞致密弥散受限 |

MRCP可清晰显示"双管征"及胆胰管扩张范围。

### 6. 超声表现及机制

- 胰腺局限性增大，低回声肿块，较大者为混合回声，边界不清呈"**蟹足样**"（浸润性生长）。
- CDFI：肿块内无明显血流信号（乏血供）。
- 间接征象：胰管扩张、肝内外胆管扩张、胆囊增大。

### 7. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
|---|---|---|---|
| 慢性肿块性胰腺炎 | 胰头局限增大、胆管扩张 | 慢性胰腺炎T2WI信号较低、动脉期轻度强化+持续渐进强化、胰管不规则扩张可见结石钙化、无血管侵犯和远处转移 | 纤维化 vs 肿瘤 |
| 自体免疫性胰腺炎 | 胰头局限增大 | 临床症状轻、无腹痛、边界清楚、激素治疗有效、血清IgG4升高、无血管侵犯 | 免疫介导炎性 vs 肿瘤 |
| 胰腺神经内分泌肿瘤（pNEN） | 胰腺肿块 | pNEN富血供动脉期明显强化；胰腺癌乏血供强化不明显 | 血供差异显著 |

### 8. 考试答题版

胰腺癌（导管腺癌）占胰腺恶性肿瘤90%，胰头癌占60%～70%。临床：进行性无痛性梗阻性黄疸 + CA19-9显著增高。CT为主影像检查：平扫肿块与胰腺等密度；增强为乏血供→强化不明显呈相对低密度（与明显强化正常胰腺对比），延迟可轻度强化。胰头癌间接征象：双管征（胆总管+胰管扩张）、胰腺体尾部萎缩、血管侵犯。MRI脂肪抑制T1WI肿块显示更清晰。鉴别慢性肿块性胰腺炎（渐进持续强化+钙化+血管无侵犯）和pNEN（富血供明显强化）。

### 9. 一句话记忆

**进行性无痛性黄疸 + CA19-9升高 + 乏血供肿块（增强低于胰腺）+ 双管征 = 胰腺癌。**

## 胰腺神经内分泌肿瘤

### 1. 疾病本质

胰腺神经内分泌肿瘤（pNEN）起源于胰腺内分泌细胞或全能干细胞，占胰腺肿瘤1%～3%。分为功能性和无功能性。功能性中胰岛细胞瘤最常见（Whipple三联征）。所有pNEN均具有不同程度恶性潜能。影像学核心特征：** 绝大多数血供丰富，动脉期明显强化**。

### 2. CT/MRI 表现及机制

| 特征 | 表现 | 机制 |
|---|---|---|
| CT平扫 | 境界清晰、圆形/卵圆形、等或略低密度 | 肿瘤局限生长 |
| CT/MRI增强（典型） | 动脉期**明显强化**（高于周围胰腺实质），可**持续强化** | 富血供特征 |
| MRI T1WI | 稍低信号（脂肪抑制T1WI显示更清楚） | 与正常胰腺信号差异 |
| MRI T2WI | 稍高信号 | 含水量增加 |
| DWI | 高信号 | 肿瘤细胞致密 |

功能性pNEN常较小（因激素分泌症状早期就诊）；无功能性pNEN常较大，以实性/囊实性多见，可伴胰胆管扩张。

### 3. 鉴别诊断（与胰腺癌的关键区别）

| 项目 | pNEN | 胰腺癌 |
|---|---|---|
| 血供 | 富血供→动脉期明显强化 | 乏血供→强化不明显 |
| 边界 | 多境界清晰 | 多边界不清呈蟹足样 |
| 胰胆管扩张 | 多无或轻度 | 常见且明显（双管征） |
| 血管侵犯 | 少见 | 常见 |
| 临床表现 | 功能性者有激素症状 | 进行性黄疸+疼痛 |

### 4. 一句话记忆

**胰腺富血供肿块 + 动脉期明显强化 + 可有激素症状 = 胰腺神经内分泌肿瘤（vs 胰腺癌乏血供）。**

## 胰腺囊性肿瘤

### 1. 疾病本质

胰腺囊性肿瘤占胰腺肿瘤10%～15%，多为良性或低度恶性。常见类型：
- **浆液性囊腺瘤（SCN）：** 好发60～70岁老年女性，胰尾部多见，无恶变倾向。微囊型由多发小囊构成呈蜂窝状，可有中央纤维瘢痕伴星芒状钙化。
- **黏液性囊腺瘤（MCN）：** 好发40～50岁中年女性，胰尾部多见。单囊或几个大囊，囊内含黏液，有分隔。为潜在恶性肿瘤→囊壁厚薄不均/壁结节→提示黏液性囊腺癌。
- **导管内乳头状黏液性肿瘤（IPMN）：** 好发老年男性。分主胰管型、分支胰管型和混合型。特征性表现是**囊性病变与胰管相通**。

### 2. 关键鉴别特征

| 类型 | SCN | MCN/囊腺癌 | IPMN |
|---|---|---|---|
| **与胰管相通** | **否** | **否** | **是**（关键特征） |
| 囊腔特征 | 多发小囊蜂窝状 | 单囊/大囊/分隔 | 分叶状/葡萄串样（分支型）或主胰管扩张（主胰管型） |
| 中央瘢痕/钙化 | 星芒状钙化 | 无 | 无 |
| 恶变倾向 | 无 | 有（囊壁厚+壁结节→提示恶性） | 有（分为良性/交界性/恶性） |
| MRI优势 | T2WI显示蜂窝状特征优于CT | MRI/MRCP | MRCP确定与胰管关系 |

### 3. 一句话记忆

**胰腺囊性病变：SCN=蜂窝状+星芒状钙化+与胰管不通；MCN=大囊+分隔+与胰管不通；IPMN=与胰管相通（MRCP关键）。**
