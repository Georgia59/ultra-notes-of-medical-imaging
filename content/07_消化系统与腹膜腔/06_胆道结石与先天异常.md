## 胆石症与胆囊炎

### 1. 疾病本质

在胆汁淤滞和胆道感染等因素影响下，胆汁中胆色素、胆固醇、黏液物质和钙盐等析出、凝集而形成胆结石。分为胆固醇性、胆色素性和混合性结石。胆结石与胆囊炎往往互为因果——结石导致胆汁淤滞继发感染，感染又促进结石形成。胆石症以中年女性多见。临床表现为反复性右上腹绞痛（放射至背部和右肩胛下区），急性胆囊炎呈持续性疼痛阵发性绞痛，墨菲征阳性。

### 2. 底层病理逻辑

- **病理因素：** 胆汁成分失衡（胆固醇过饱和/胆红素钙析出）→结石形成→胆囊管/胆总管梗阻→胆汁淤滞→细菌感染→胆囊壁充血水肿→急性胆囊炎。反复发作→胆囊壁纤维化增厚→慢性胆囊炎→胆囊萎缩或瓷化胆囊。
- **对影像的影响：** 结石的超声波特性→强回声 + 后方声影（声波在结石界面全反射）。结石密度差异→高密度（含钙多）、等密度、低密度（胆固醇性）。胆囊壁水肿→增厚分层状强化。慢性胆囊炎→胆囊缩小、壁增厚钙化（瓷化胆囊）。

### 3. 临床表现及机制

| 临床表现 | 产生机制 |
|---|---|
| 右上腹绞痛放射至背部和右肩胛下区 | 胆石嵌顿于胆囊管/胆总管引起痉挛 + 膈神经牵涉 |
| 墨菲征阳性 | 检查者触压右上腹时患者因疼痛突然屏气 |
| 畏寒、高热、呕吐 | 急性感染中毒症状 |
| 黄疸 | 结石阻塞胆总管致胆汁排出受阻 |

### 4. 影像学表现及机制

#### 超声（首选检查）

| 征象 | 表现 | 机制 |
|---|---|---|
| **胆结石** | 胆囊/胆管内强回声团/光斑/弧形强光带，后方伴**声影**，随体位改变**移动** | 结石声阻抗高致全反射→声影；重力作用→移动 |
| 泥沙型结石 | 胆囊后壁细小强回声光点带，后方伴较宽声影 | 小结石沉积于后壁 |
| 结石填满胆囊 | "**胆囊壁弱回声-结石强回声-声影**"三联征 | 胆囊腔被结石占据 |
| 急性胆囊炎 | 胆囊增大、壁增厚（多层弱回声带） | 炎性水肿致壁增厚、分层 |
| 慢性胆囊炎 | 胆囊缩小、壁增厚钙化、边缘毛糙 | 纤维化、钙盐沉积 |

#### CT

| 特征 | 表现 | 机制 |
|---|---|---|
| 阳性结石 | 胆囊/胆管内高密度影，密度均一/不均/分层 | 含钙结石高密度 |
| 阴性结石（胆固醇性） | 等或低密度 | 胆固醇性接近胆汁密度 |
| 胆总管结石 | "**靶征**"（圆形结石 + 周围环状胆汁），"**新月征**"（部分围绕） | 结石在扩张胆管中心，胆汁环绕 |
| 急性胆囊炎 | 胆囊增大（＞5cm），壁弥漫增厚（＞3mm）分层状强化，周围脂肪密度增高 | 水肿和渗出 |
| 慢性胆囊炎 | 胆囊缩小，壁增厚可钙化→"**瓷化胆囊**" | 纤维化、钙化 |

#### MRI/MRCP

- 结石：T1WI多为低信号（胆固醇/胆色素结石可呈混杂/高信号），T2WI为低信号。
- MRCP：高信号胆汁中圆形/类圆形/多边形低信号充盈缺损，可整体直观显示结石数目、位置和胆管扩张范围。

### 5. 关键征象机制

- **征象：** 胆囊内强回声 + 后方声影 + 随体位改变移动。
  **机制：** 结石声阻抗远高于胆汁→声波在结石表面强反射→后方无声波穿透→声影。体位改变→结石在重力作用下沉降。
  **提示意义：** 胆囊结石的超声诊断标准三要素。

- **征象：** "靶征"——CT上胆总管结石呈中心高密度、周围环状低密度胆汁。
  **机制：** 结石嵌顿于扩张的胆总管中心位置，胆汁环绕。
  **提示意义：** 胆总管结石的CT特征性表现。

- **征象：** "瓷化胆囊"——胆囊壁广泛钙化。
  **机制：** 慢性胆囊炎反复发作→胆囊壁纤维化→钙盐沉积。
  **提示意义：** 慢性胆囊炎的终末表现。

### 6. 考试答题版

胆石症好发于中年女性，与胆囊炎互为因果。超声为首选检查：典型胆结石表现为胆囊内强回声 + 后方声影 + 随体位改变移动。急性胆囊炎：胆囊增大、壁增厚分层强化、墨菲征阳性。慢性胆囊炎：胆囊缩小、壁增厚钙化（瓷化胆囊）。CT对肝外胆管结石显示优于超声（靶征、新月征），但对阴性结石可能漏诊。MRCP可整体直观显示结石和胆管扩张。需与胆管癌（扩张胆管远端软组织肿块不规则狭窄）和胆管炎（鼠尾状逐渐狭窄）鉴别。

### 7. 一句话记忆

**右上腹痛 + 强回声后方声影 + 随体位移动 = 胆囊结石；胆囊增大/壁厚分层 = 急性胆囊炎。**

## 胆系先天性发育异常

### 1. 疾病本质

胆系先天性发育异常中，先天性胆管扩张最常见，系先天性胆管壁发育不全所致。根据扩张的形态、部位、范围分为五型：
- **Ⅰ型（最常见，80%～90%）：** 胆总管囊状或梭形扩张（先天性胆总管囊肿）。临床表现为黄疸、腹痛和右上腹包块。
- **Ⅱ型：** 胆总管憩室。
- **Ⅲ型：** 胆总管十二指肠壁内段囊状扩张。
- **Ⅳ型：** 肝内、外胆管或仅肝外胆管多发囊状扩张。
- **Ⅴ型（Caroli病）：** 肝内胆管多发囊状扩张，常合并胆管炎、肝脓肿和肝纤维化。

### 2. 底层病理逻辑

- **病理因素：** 先天性胆管壁发育不全→胆管壁薄弱→胆管内压力增高→囊状或梭形扩张。Caroli病为肝内胆管节段性扩张，扩张囊腔与胆管相通（关键鉴别点）。
- **对影像的影响：** 胆总管囊状扩张→肝门区囊性病变与胆管相连。Caroli病→肝内多发囊性灶与胆管相通 + "中心点征"（门静脉分支被扩张胆管包绕）。

### 3. 影像学表现及机制

#### Ⅰ型（胆总管囊肿）

| 检查 | 表现 |
|---|---|
| 超声 | 胆总管全程液性无回声区，与肝门部胆管相连 |
| CT | 胆总管全程明显扩张，壁薄均匀，肝内胆管无扩张（关键！与梗阻性扩张不同），可合并结石或癌变 |
| MRI/MRCP | 同CT，MRCP直观显示囊肿与胆管的关系 |

#### Ⅴ型（Caroli病）

| 检查 | 表现 |
|---|---|
| CT | 与门静脉走行一致、以肝周围部为主的多发囊状低密度灶；增强检查"**中心点征**"（扩张胆管将并行血管包绕其内→中心点状强化） |
| MRI/MRCP | MRCP可显示扩张囊腔与肝内胆管间的**交通**→Caroli病特征性表现 |

### 4. 关键征象机制

- **征象：** "中心点征"——增强CT示囊状扩张胆管内中心点状强化。
  **机制：** 扩张的胆管将并行的门静脉分支包绕其中，增强后门静脉分支强化表现为囊内点状高密度。
  **提示意义：** Caroli病的特征性表现。

### 5. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
|---|---|---|---|
| 梗阻性胆总管扩张 | 胆总管扩张 | 梗阻性肝内、外胆管成比例扩张，可发现梗阻病因；胆总管囊肿壁薄、肝内胆管不扩张 | 囊肿为先天性壁薄弱，梗阻为后天性阻塞 |
| 多发性肝囊肿 | 肝内多发囊性灶 | Caroli病与胆管相通、中心点征；肝囊肿与胆管不相通 | Caroli为胆管扩张，囊肿为独立囊腔 |

### 6. 考试答题版

先天性胆管扩张以Ⅰ型（胆总管囊肿）和Ⅴ型（Caroli病）最常见。Ⅰ型：胆总管全程囊状扩张，壁薄，肝内胆管不扩张，MRCP可明确诊断。Ⅴ型（Caroli病）：肝内（以周围部分布为主）多发囊状扩张，"中心点征"（增强CT囊内点状强化），MRCP显示囊腔与胆管相通是特征性表现。鉴别关键：胆总管囊肿肝内胆管不扩张（vs梗阻性扩张全胆系成比例扩张）；Caroli病与胆管相通且中心点征（vs肝囊肿不相通）。

### 7. 一句话记忆

**Ⅰ型胆总管囊肿=胆总管全程囊状扩张+肝内胆管正常；Ⅴ型Caroli病=肝内多发囊+中心点征+与胆管相通。**
