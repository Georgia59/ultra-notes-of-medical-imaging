## 肝硬化

### 1. 疾病本质

肝硬化（cirrhosis of liver）常见病因为病毒性肝炎、自身免疫性肝炎和酗酒。早期肝细胞弥漫性变性、坏死；中晚期大量纤维组织增生并形成再生结节（RN），致肝脏变形、变硬、肝叶萎缩，进一步继发门静脉高压。部分RN演变成不典型增生结节（DN），最后可导致HCC。临床表现为食欲缺乏、腹胀、黄疸、腹腔积液、呕血和肝性脑病。

### 2. 底层病理逻辑

- **病理因素：** 肝细胞反复损伤→纤维组织增生（肝星状细胞活化）→肝小叶结构破坏→纤维间隔包绕再生肝细胞形成再生结节（RN）→门静脉阻力增加→门静脉高压→侧支循环开放（食管胃底静脉曲张、脾肾分流等）。RN→DN→早期HCC→进展期HCC（多步癌变）。
- **对影像的影响：** 纤维化→肝表面凹凸不平、肝裂增宽。RN→T1WI中高信号、T2WI低信号结节（铁沉积）。DN→T2WI低信号结节但可动脉期轻度强化。门静脉高压→脾大、腹水、侧支静脉迂曲扩张。一旦T2WI低信号结节内出现稍高信号灶（"结中结"）且动脉期强化呈快进快出→提示早期HCC。

### 3. 临床表现及机制

| 临床表现 | 产生机制 |
|---|---|
| 食欲缺乏、腹胀 | 肝功能损害、门静脉高压性肠病 |
| 黄疸 | 肝细胞处理胆红素能力下降 |
| 腹腔积液 | 门静脉高压 + 低蛋白血症 + 淋巴回流受阻 |
| 呕血、黑便 | 食管胃底静脉曲张破裂 |
| 脾大、脾功能亢进 | 门静脉高压致脾淤血 |
| 肝性脑病 | 肝功能衰竭致血氨升高 |
| 蜘蛛痣、肝掌 | 雌激素灭活减少 |

### 4. 影像学表现及机制

#### CT

| 表现 | 机制 |
|---|---|
| **直接征象** | |
| 全肝萎缩或部分肝叶萎缩、部分代偿增大（各肝叶大小比例失常）| 不同区域纤维化和再生程度不同 |
| 肝轮廓凹凸不平，肝门、肝裂增宽 | 纤维组织收缩牵拉 |
| 肝密度不均匀（脂肪变性、纤维化、RN导致）| 多种病理成分混杂 |
| 增强动脉期RN轻度强化，门静脉期与肝实质强化一致 | RN以门静脉供血为主 |
| **间接征象** | |
| 脾大（＞5个肋单位）| 门静脉高压致脾淤血 |
| 腹腔积液 | 门静脉高压 + 低蛋白血症 |
| 胃底食管静脉曲张（CTA显示增粗迂曲侧支血管）| 门静脉高压开放侧支循环 |
| 胆囊结石、胆囊周围积液 | 肝功能异常 |

#### MRI（对RN-DN-早期HCC监测更有价值）

| 序列 | RN表现 | DN表现 | 早期HCC |
|---|---|---|---|
| T1WI | 略高、等或低信号 | 略高、等或低信号 | 低信号为主 |
| T2WI | **低信号**（铁沉积） | **低信号**为主 | 低信号结节内出现**稍高信号灶**（"结中结"） |
| 增强 | 各期与肝实质一致 | 可动脉期轻度强化，门静脉期/平衡期与肝实质一致 | 动脉期明显强化，门静脉期/平衡期减退（快进快出） |
| 肝胆期（肝特异性对比剂） | 等信号（仍有功能肝细胞） | 等/稍低信号 | **低信号**（丧失摄取功能） |

### 5. 考试答题版

肝硬化常见病因为病毒性肝炎和酗酒。影像表现分为直接征象和间接征象。直接征象：肝脏大小形态改变（部分肝叶萎缩、部分代偿增大）、轮廓凹凸不平、肝裂增宽、密度/信号不均。间接征象（门静脉高压）：脾大、腹水、食管胃底静脉曲张。MRI对监测RN→DN→早期HCC演变有重要价值：T2WI低信号结节内出现稍高信号灶（"结中结"）且呈快进快出强化→提示早期HCC。肝细胞特异性对比剂肝胆期低信号更敏感检出小HCC。

### 6. 一句话记忆

**肝炎/酗酒→肝萎缩变形凹凸不平 + 脾大腹水侧支静脉 = 肝硬化，T2WI低信号结节中见快进快出灶 = 早期HCC。**

## 肝细胞癌

### 1. 疾病本质

原发性肝癌80%～90%为肝细胞癌（HCC），好发于30～60岁男性。HCC发病与肝硬化密切相关，经历RN→DN→早期HCC→中晚期HCC的演变过程。病理分三型：①巨块型（≥5cm）；②结节型（每个＜5cm）；③弥漫型（＜1cm且数目众多，弥漫分布）。直径不超过3cm的单发结节或2个结节直径之和不超过3cm者为小肝癌。HCC主要由肝动脉供血（90%以上血供丰富），易形成假包膜，易侵犯门静脉和肝静脉形成癌栓，可发生肝内外转移。多数患者血AFP明显升高。

### 2. 底层病理逻辑

- **病理因素：** 肝硬化→DN→HCC（多步癌变）。90%以上HCC由肝动脉供血（正常肝75%由门静脉供血）→增强呈"快进快出"。肿瘤压迫周围肝组织→假包膜形成（含纤维组织和受压血管）。侵犯门静脉→门静脉癌栓→可致门静脉高压加重；侵犯肝静脉→可至右心房。肝内血行转移、淋巴转移（肝门、腹膜后）、远处转移（肺、骨、肾上腺）。
- **对影像的影响：** 动脉期明显强化（肝动脉供血）→门静脉期/平衡期相对低密度/信号（缺乏门静脉供血）→"快进快出"。假包膜延迟强化。癌栓→门静脉/肝静脉内充盈缺损。肝细胞特异性对比剂肝胆期→HCC为低信号（不摄取对比剂）。

### 3. 临床表现及机制

| 临床表现 | 产生机制 |
|---|---|
| 肝区疼痛 | 肿瘤增大牵拉肝包膜 |
| 消瘦乏力、恶病质 | 肿瘤消耗和肝功能损害 |
| 黄疸 | 肿瘤压迫或侵犯胆管，或肝功能衰竭 |
| 腹部包块 | 巨块型HCC可触及 |
| 血AFP明显升高 | HCC细胞分泌甲胎蛋白 |
| 发热 | 肿瘤坏死吸收热 |
| 肿瘤破裂出血 | 近肝表面HCC破裂致腹腔出血 |

### 4. CT 表现及机制

#### CT 平扫

- **巨块型/结节型：** 肝实质内低密度肿块，中央可坏死呈更低密度区。**机制：** 肿瘤密度低于正常肝实质，坏死液化密度更低。
- **弥漫型：** 全肝或局部增大，境界不清多发低密度小结节。**机制：** 弥漫分布的小癌结节。
- **假包膜：** 瘤周低密度带。**机制：** 肿瘤压迫周围肝组织形成的纤维包膜。
- **肿瘤破裂出血：** 瘤内斑片状高密度。**机制：** 肿瘤内急性出血。

#### CT 多期增强（诊断关键——"快进快出"）

| 时相 | 表现 | 机制 |
|---|---|---|
| 动脉期 | 明显斑片状、结节状强化，CT值迅速达峰，可见肿瘤血管 | 肿瘤主要由肝动脉供血，血供丰富 |
| 门静脉期 | 肿瘤呈相对低密度（正常肝实质明显强化） | 肿瘤缺乏门静脉供血，正常肝由门静脉供血显著强化 |
| 平衡期 | 肿瘤密度持续减低，与正常肝对比更明显 | 对比剂从肿瘤较快廓清 |

假包膜一般在门静脉期或平衡期出现强化（纤维组织延迟强化）。

#### 间接征象

- **静脉癌栓：** 门静脉/肝静脉内低密度充盈缺损，门静脉期最清楚。**机制：** HCC侵犯血管形成癌栓。
- **淋巴结转移：** 肝门、腹主动脉旁、腔静脉旁淋巴结肿大。
- **胆管侵犯：** 上方胆管扩张。
- **合并肝硬化表现。**

### 5. MRI 表现及机制

| 序列 | 典型表现 | 机制 |
|---|---|---|
| T1WI | 低信号（出血/脂肪变可呈高信号） | 肿瘤组织T1弛豫时间延长 |
| T2WI/脂肪抑制T2WI | 稍高信号 | 肿瘤含水量增多 |
| DWI | 高信号，ADC值减低 | 肿瘤细胞致密，水分子弥散受限 |
| 假包膜（T1WI） | 瘤周环状低信号影 | 纤维包膜 |

**Gd-DTPA多期增强：** 同CT多期增强表现（快进快出）。

**肝细胞特异性对比剂（钆塞酸二钠/钆贝葡胺）增强：** 肝胆期（延迟约20分钟）正常肝实质持续强化呈高信号，HCC因不具备摄取功能呈**低信号**，对小HCC检出更敏感。

### 6. 超声表现及机制

- **直接征象：** 肝实质内单发或多发肿块，以低回声和混合回声多见；肿瘤周围有完整或不完整的环形低回声带（假包膜）。
- **间接征象：** 多数有肝硬化声像图；癌栓→扩张门静脉/胆管内低回声灶；肝内管道受压；淋巴结转移。
- **超声造影：** 动脉期早于肝实质呈整体均匀高增强，门静脉期和延迟期呈低增强（"快进快出"）。

### 7. 关键征象机制

- **征象：** "快进快出"强化——动脉期明显强化，门静脉期/平衡期迅速减退呈相对低密度/信号。
  **机制：** HCC主要由肝动脉供血（动脉期早于肝实质强化），而正常肝组织75%由门静脉供血（门静脉期显著强化），HCC缺乏门静脉供血故呈相对低密度/信号。
  **提示意义：** HCC的特征性强化方式，是诊断HCC的核心依据。

- **征象：** "结中结"——T2WI低信号结节内出现稍高信号灶。
  **机制：** RN在T2WI为低信号（铁沉积），当其中发生癌变（早期HCC）时，癌变区含水量增加致T2WI信号增高。
  **提示意义：** 肝硬化结节癌变的早期MRI征象。

### 8. 鉴别诊断

| 鉴别疾病 | 相似点 | 区分点 | 底层逻辑 |
|---|---|---|---|
| 肝海绵状血管瘤 | 动脉期明显强化 | 血管瘤"快进慢出"（延迟期持续填充），T2WI明显高信号（"灯泡征"）；HCC"快进快出" | 血管瘤为血窦结构血流缓慢；HCC为高血供肿瘤廓清快 |
| 肝转移瘤 | 肝内多发结节 | 转移瘤常多发、环形强化（"牛眼征"）、有原发瘤史；HCC常合并肝硬化、AFP升高 | 转移瘤来自肝外原发瘤 |
| 肝腺瘤 | 富血供、动脉期明显强化 | 腺瘤好发青年女性、口服避孕药史、无肝硬化背景、出血脂肪变常见 | 不同病理基础 |
| 局灶性结节性增生（FNH） | 动脉期明显强化 | FNH无"快出"、有延迟强化的中央瘢痕、肝胆期为高信号 | FNH为良性增生，含正常肝细胞 |

### 9. 考试答题版

HCC是原发性肝癌最常见类型（80%～90%），好发于30～60岁男性，与肝硬化密切相关，多数AFP升高。CT/MRI多期增强检查是确诊关键：典型表现为"快进快出"——动脉期明显强化（肝动脉供血），门静脉期和平衡期迅速减退呈相对低密度/信号（缺乏门静脉供血）。其他特征：假包膜（延迟强化）、静脉癌栓、合并肝硬化。肝细胞特异性对比剂MRI肝胆期HCC呈低信号，更敏感检出小肝癌。鉴别诊断：血管瘤（快进慢出 + T2WI灯泡征）、转移瘤（多发 + 牛眼征 + 原发瘤史）、FNH（无快出 + 中央瘢痕 + 肝胆期高信号）。

### 10. 一句话记忆

**肝硬化 + AFP升高 + 快进快出（动脉期高强化→门静脉期廓清）+ 假包膜/癌栓 = HCC，T2WI灯泡征 + 快进慢出 = 海绵状血管瘤。**

## 脂肪肝

### 1. 疾病本质

正常肝脏脂肪含量低于5%，超过5%则为肝脏脂肪浸润（脂肪肝）。分为弥漫性和局灶性脂肪肝。CT和超声均可作为首选检查方法。

### 2. 影像学表现及机制

| 检查 | 表现 | 机制 |
|---|---|---|
| 超声 | 弥漫性："明亮肝"，肝肾对比征阳性，近场回声高、远场衰减 | 脂肪滴使声波散射衰减增加 |
| CT平扫 | 肝密度弥漫性减低，肝/脾CT值比值＜0.85 | 脂肪取代正常肝细胞使密度降低 |
| 血管湮没征 | 低密度肝内血管显示不清 | 肝密度降至与血管相近 |
| 血管反转征 | 肝血管密度相对高于肝密度 | 严重脂肪浸润时肝密度低于血管 |
| CT增强 | 肝实质强化程度减低，血管显示更清晰 | 脂肪替代使肝血供相对减少 |
| MRI化学位移 | 反相位（out phase）较同相位（in phase）信号明显减低 | 水脂信号相位差异致信号抵消 |

### 3. 鉴别诊断

局灶性脂肪肝需与肝血管瘤、HCC、转移瘤鉴别。局灶性脂肪肝**无占位效应**，增强扫描病灶内可见正常血管走行，无受压、侵及表现。

### 4. 一句话记忆

**肝密度弥漫性减低 + 肝/脾CT值＜0.85 + 无占位效应 + 反相位信号明显降低 = 脂肪肝。**
